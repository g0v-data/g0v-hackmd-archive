# 05-15 房東面談紀錄

雖然租了好幾次房子，但這是第一次認真準備，以及認真面談。面談過程大約一小時，也比想像中的還有收穫。

## 開始前的準備

前一天先傳了[無法復原裝修的細部規劃 + 預算](https://g0v.hackmd.io/5zM5SeoFTNqZeCH4joXxIw)給房東，並附上主要想要確定的合約內容，讓雙方到時候有討論的基礎資訊，減少因為現場釐清資訊而造成的不確定因素，到時候就可以直接聚焦談租金以及合約細節。

因為面談當天早上，宣佈雙北三級防疫警戒，所以提早了面談的時間。原本雙方各自都有考慮過是否線上聊就好，但因為都覺得是第一次討論，還是見面，可以得到比較多的資訊，減少日後誤差。

## 面談時的發現

和面試一樣，先建立信任後，就能找到彼此的交集。

1. 房東想要確認的：
   1. 房客是否是個會善待房子的人，越有能力、意願善待，對房客組成的容忍度，會越高
      - 除了基本的拜拜、營業登記不行外，有談到既然是會照顧房子的人，就算有小孩其實也可以
   2. 對房客的信任度
      - 藉由了解房客對提案的掌握度、專業能力，也能增加房東的信任程度
      - 面談前傳的文件的確有助於證明房客的相關能力與誠意
      - 面談期間簡單帶過各種邏輯、強調今天只談帶不走的部份（站在房東立場），也有助於傳達同理的能力
   3. 房東的權力義務
      - 房東清楚表達自己的財務需求，但也同意帶不走的基礎裝修，理應由房東承擔
      - 若有鄰里問題，例如樓上漏水，會出面處理
   4. 了解裝修細節
      - 要求裝修不可動結構
      - 裝修期間如造成鄰居損失，須由房客賠償（合理，房客也同意）
3. 房客想要確認的：
   1. 房東對裝修的看法，以及房客的權力義務要求
      - 想確認房東是否認同基本居住舒適的重要性，以及了解例如結構、漏水等，須由屋主處理
   2. 是否能用承擔裝修的時間與財務成本、報稅成本，換取租金減免
      - 直接提了裝修期間免租，以及租金減免 1,500
      - 忘了強調換算方式，例如以上方案，等於是房東出 xx 元裝修，剩下大部分的 oo 元是房客承擔，到很後面才提到
   3. 物件的歷史脈絡，包含硬體以及鄰里，以判斷裝修風險，以及如何與鄰里互動
      - 確認是否為自己的老房子，以判斷對裝修、財務槓桿、物件保存的要求度
      - 了解看房過程中看到的硬體特性，像是水路均為明管，可能是因為大樓原本管線無法修繕
      - 整片基地其實是家族蓋的，亦有許多親戚住在這裡
      - 尚無人來談都更，因為不在大馬路旁，區位不夠好（所以可以好好裝修）
   4. 提早解約的處理方式
      - 大致同意房東若提早解約時，須按時間比例賠償大型裝修的成本

## 面談小結

1. 雙方均滿意對方的方案，以及租金方向，可以往下談合約細節
2. 合約延長為五年，房東也會在財務金流許可的狀況下，總共負擔 8.1 萬元的修繕成本
3. 基於對房客提案，以及其相關知識的信任，由房客以內政部租賃契約為範本，提出五年契約草稿，進行後續討論
4. 有提到同社區的另一間公寓，明年也有計畫出租，若這間做得好，就有機會多一個空間！

## 一點心得

1. 如果是尚待裝修的空間，當房客建立自己的專業能力後，只要房東對自身權力義務有務實的理解，應該就可以認真談出，適合彼此的合約，是個不錯的組合
2. 雖然 ddio 的歷任房東，多少都有這樣的特質，但依然不確定是不是特例，需要思考如何協助有興趣的房客，就算沒有同樣的社經條件，也能建立類似的專業形象
