## 飲料 (預算 $15,000)
[108創藝廚房空間資訊](https://docs.google.com/presentation/d/15pZFA_psD-2roYTDwX9e2OQ6CLo0ayRo/edit#slide=id.p1)

- 酒精飲料 (12-13 R108 室)
    - Sandra 總籌酒精飲料
    - ronnywang 可提供一箱酒、vodka 和德恩奈苦艾酒





```
* 沒有人喝的調酒 *

可供應約 30-40 杯

1. 製作桂花青茶伏特加：
混合一兩的青茶茶葉，20g乾燥桂花，1400ml伏特加，用真空袋封好，60度舒肥4小時。

2. 混合所有調酒材料：
柑橘苦精 20 dash
青蘋果果泥 300 ml
檸檬汁 300 ml
蜂蜜甘露酒 200 ml
西洋梨利口酒 300 ml
桂花青茶伏特加 1400 ml

3. 進行奶洗：
加入 125ml 牛奶後，等待牛奶與酸會反應，與調酒中的雜質一併凝結成塊，過濾之後便可以得到清澈的酒體。

4. 加冰攪拌，喝。
```



另外買兩手啤酒，提供不喝調酒的人另外的酒精選項。
