# 當代地方志 / 方志好好玩


鄉鎮區誌出版品，不只是文件掃描成檔案提供下載，那會是什麼？

### 架構討論

- 傳統方志架構
- 「需要什麼樣的架構」發想
    - 已經有的內容架構，延伸至近期資料、即時現況
    - 新增近代的內容項目：
        - 建築設計細節
        - 基礎建設篇章，應增加網路通訊設施
        - 非在籍人口
        - 國際姐妹社區
        - 社區報、地區型粉絲頁
        - 土地開發案件、環境訴訟判決
        - ...
    - 文獻目錄彙編，可以持續更新、共筆
    - Chorography
        - 行事曆：[https://g0v.hackpad.com/cjdeZ8orXk4](https://g0v.hackpad.com/cjdeZ8orXk4)
    - Dashboard
