# Web設計規範
- 主流螢幕尺寸大小(px)
1024x768(4:3)
1366x768
1280x800
1280x1024(4:3)
1440x900
1600x900
1920x1080

- 網站/網頁內容
國內:996~1000(px)

- 網頁字體(可辨識性、易讀性)
盡量以系統預設自體為主
明體、宋體、微軟正黑、黑體(Apple)
Arial
**最小大小: 12px**
12px:用於非突出性的資料 ex:日期、版權宣告
14px:用於主文、正文
16px 18px 20px:用於標題

- 字體間距
1. 文字間距:
預設為主，視情況調整
2. 行間距:
字體大小1.5~2倍
3. 段間距:
字體大小2~2.5倍
4. 字體色彩
主文:易讀->深灰->#333~#666
說明:淺色->#999
以公司、單位主色系為主
