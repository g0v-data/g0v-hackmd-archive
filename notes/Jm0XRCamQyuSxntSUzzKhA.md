# con-太多了吧怎麼記?
con-有"共同"之意，有時候可以試著拆開看:
1. con**struction** (結構) -> (建造)
2. con**test** (測驗) -> (競賽)
3. con**tent** (帳篷) -> (內容)
4. con**text** (文本) -> (上下文)
5. con**tract** (傳單) -> (契約)
6. con**tribute** (貢物) -> (貢獻)
