# Taiwan Little Green Data Book

> [點此觀看原始內容](https://g0v.hackpad.tw/uutXqCkF16A)

台灣綠色資料手冊

台灣由於國際地位特殊，目前仍沒有加入世界銀行。世界銀行每年會整理出版一本，各國環境資訊的手冊：[_Little Green Data Book 2015 (LGDB)_](http://data.worldbank.org/products/data-books/little-data-book/little-green-data-book/)_。_
其實我們的政府各部會也有相關的資料，希望將資料挖出來，透過LGDB的框架來和世界比較，一方面是認識自己，也是想將台灣介紹出去。
