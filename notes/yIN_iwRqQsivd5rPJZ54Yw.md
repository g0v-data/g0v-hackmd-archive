### 飞行前检查

#### 遥控器 

注 APP設備: iphone、ipad、android裝置

- [ ] 检查"APP設備"電池电量
- [ ] 将"APP設備"安装到遥控器中
- [ ] 在"APP設備"關閉任何打開的APP
- [ ] 将"APP設備"设置为飞行模式及使用離線地圖功能
- [ ] 展開RC天線
- [ ] 检查飛行模式:设置为P模式

#### 飛機

- [ ] 檢查Air Traffic App操作區域是否有航班
- [ ] 檢查飛機的電池電量
- [ ] 展開雙臂- for mavic 系列
- [ ] 確認WiFi / RC開關設置為RC - for mavic 系列
- [ ] 確認上列開關保護蓋已關閉 
- [ ] 確認SD卡安裝正確
- [ ] 取下雲台保護蓋
- [ ] 取下雲台後方(mavic系列為-保護塑膠;精靈系列為-海綿) 
- [ ] 檢查雲台是否有損壞/阻塞
- [ ] 安裝螺旋槳-白/黑標記到適當的電機(確認保裝完成) 

---

### 準備起飛

- [ ] 飛機放置起飛/降落 適當地點
- [ ] 打開遙控器電源
- [ ] 開啟智能飛行電池
- [ ] 啟動APP

---
   
### 起飛前最後檢查

- [ ] 檢查操作區域是否有障礙物
- [ ] 確認帳號是否登入完成
- [ ] 檢查飛機基本資訊和校準 
- [ ] 遙控器控制及圖傳信號強度是否正常及檢查有無干擾 
- [ ] 飛機電池電壓及容量是否異常
- [ ] 檢查避障設定
- [ ] 檢查/設定正確的返航高度 
- [ ] 確認姿態球方向是否正常/Map位置是否正常
- [ ] 檢查SD卡狀態/容量
- [ ] 檢查相機設置/焦點 
- [ ] 確認GPS抓星完成 (特殊場地:可空中GPS抓星-專業級飛手)
- [ ] 啟動/停止電機測試 
- [ ] 手動/自動起飛 
- [ ] 懸停於 3米處確認飛機有無異常
- [ ] 確認高度/距離數據是否正常
- [ ] 檢查拍照/錄影功能正常後開始執行任務
