# 📣 語法筆記 📣
-  ## 標題(H1~H6) (#~######) 
-  分隔線 (\-\-\-)
	-  ---
- 內文 
	- **粗體** \**abc**
	- _斜體_ \_abc_
	- 
	- ~~刪除線~~ \~~abc~~
	- `標注` \`abc`
	- > 引用 \>abc
- 清單
	- 項目 ( - abc )
		- A
		- B
		- C
	-  數字項目 ( 1. abc )
		1. A
		2. B
		3. C
	-  勾選欄 ( - [  ]  )
		-  [ ] A
		-  [ ] B
		-  [ ] C
- 程式碼 \```language \```
