# 心理服務污染
- 定義：造成被服務對象的干擾、不舒服、服務無效、甚至是創傷，以及造成認知、價值觀的不當影響。
- 如何避免：
    - 服務前
        - 調查社區與服務對象的需求，服務內容要圍繞在社區的需求上，並做好充足的準備，避免產生無效服務、干擾等汙染。
        - 辦理專業訓練，了解被服務對象的現況、特質、價值觀、文化...等，讓自己帶著正確、欣賞與尊重的觀點進入服務場域，降低造成被服務對象不舒服、歧視、受創...等汙染。
        - 制定志願服務倫理或服務行為準則，提高對服務汙染的警覺心。
    - 服務中
        - 嚴守志願服務倫理或服務行為準則，志工夥伴相互提醒，也要虛心接受夥伴的提醒。
        - 尊重對方文化，避免價值觀的壓迫或歧視。志願服務不是用上對下的壓迫方式來傳遞「優勢文化」，而是對等尊重的相處，讓被服務對象保留選擇的權利。
        - 發揮同理心與欣賞的眼光，看到被服務對象的優勢，協助他們整合自身優勢與新的觀點。
        - 自身碰到或觀察到有價值觀的衝突時，應該要在團體會議中提出討論，透過團體智慧來集思廣益。
        - 用協助探索來取代給予答案。
    - 服務後
        - 進行服務反思與團體分享討論，並將反思與討論歸納出的經驗與知識，帶到下次服務的場域實踐，這樣不但可以對服務汙染有更高的敏感度，也能有效降低服務污染的可能性。

- 醫師誓詞
    - 希波克拉底誓詞：仰賴醫神阿波羅、埃斯克雷波斯及天地諾神為證，鄙人敬謹直誓：願以自身能力及判斷力所及，遵守此約。凡授我藝者，敬之如父母，作為終身同業伴侶，彼有急需，我接濟之。視彼兒女，猶我兄弟，如欲受業，當免費並無條件傳授之。凡我所知，無論口授書傳，俱傳之吾與吾師之子及發誓遵守此約之生徒，此外不傳與他人。我願盡余之能力與判斷力所及，遵守為病家謀利益之信條，並檢柬一切墮落和害人行為，我不得將危害藥品給予他人，並不作該項之指導，雖有人請求亦必不與之。尤不為婦人施墮胎手術。我願以此純潔與神聖之精神，終身執行我職務。凡患結石者，我不施手術，此則有待於專家為之。無論至於何處，遇男或女，貴人及奴婢，我之唯一目的，為病家謀幸福，並檢點吾身，不作各種害人及惡劣行為，尤不作誘姦之事。凡我所見所聞，無論有無業務關係，我認為應守秘密者，我願保守秘密。尚使我嚴守上述誓言時，請求神祗讓我生命與醫術能得無上光榮，我苟違誓，天地鬼神實共亟之。
    - 誓詞衍生出的醫學倫理（四大原則）
        1. 尊重自主原則:強調尊重病人的尊嚴與自主性，例如不隱瞞病人之病情及診斷、保護病人的隱私及強調知情同意的重要性。
        2. 不傷害原則:強調醫療人員謹慎地執行以達到適當的照顧標準。此原則的精神在於不要傷害、防止傷害以及除去傷害。避免讓病人承擔任何不當的、受傷害的風險。
        3. 行善原則:指在不傷害他人之外，進一步關心並致力提升病人的福祉。所有醫療行為一切應以病人的利益為前提。
        4. 正義原則:公平地分配醫療資源、尊重病人的權利及尊重道德允許的法律。

- 醫師的社會責任
    - 社會責任是指：出於自發性的動機與理念，來提高生活品質，創造更好的社會，並希望藉此行為對更廣泛的人群與社會帶來正面的影響
    - 醫學專業與社會有關，可視作一種社會契約（社會給予醫療人員特權，提供特定服務並承擔責任，醫療人員則同意將此特權用在提升社會福祉上，其次才是個人利益）
        1. 維護患者健康與安全： 醫師的首要責任是確保患者的健康與安全。他們需要透過準確的診斷、有效的治療和良好的醫療管理，幫助患者恢復健康或者緩解症狀。
        2. 尊重與保護患者權利： 醫師應該尊重患者的自主權，並確保他們的意願和權利得到充分尊重。這包括尊重患者的隱私、保密權，以及讓他們參與醫療決策的權利。
        3. 持續專業發展： 醫師需要不斷更新自己的知識和技能，以跟上醫學科學和技術的發展。這有助於確保他們提供給患者的是最新、最有效的治療方法。
        4. 公平對待所有患者： 醫師應該公平地對待所有患者，不論其社會地位、種族、性別、宗教信仰或其他身份。他們應該避免歧視，並致力於提供平等的醫療服務。
        5. 參與社區健康促進： 醫師可以通過參與社區健康教育活動、預防性醫療措施以及公共衛生計劃等方式，促進整個社區的健康。
        6. 與其他醫療專業人員合作： 醫師應該與其他醫療專業人員合作，共同為患者提供全面的、協調一致的醫療服務。
        7. 關心社會議題 : 他們應該致力於改善公共衛生，並參與預防疾病的努力。這可能包括教育公眾關於健康生活方式的重要性，或者參與疫苗接種和其他公共衛生活動。
        8. 關心健康知識的傳播 : 教育公眾關於健康生活方式的重要性，或者參與疫苗接種和其他公共衛生活動。
        9. 致力於提高醫療服務的品質和可及性 : 倡導更公平的醫療政策，或者努力確保所有人都能獲得他們需要的護理。
        10. 參與政策研究與製定：從政策面改善醫療體系、提升全民健康。例如，參與研究分析、建議反饋、監督評估、推動並落實。
        11. 對環境的關懷 : 醫療行業是一個資源密集型的行業，醫生可以在他們的實踐中推動更環保的選擇，例如使用可再生的醫療產品，或者推動減少醫療廢棄物。
        12. 分配醫療資源並防止病患擅用他們無權取得之服務：避免浪費及缺乏效率的醫療行為，分辨有效與無效的治療手段
        13. 對抗不公正的社會結構 : 努力確保所有人都能獲得他們需要的護理。
        14. 致力於終身學習：可以繼續提供最新和最有效的治療方法。這可能包括參加研討會，閱讀醫學文獻，或者參與專業組織。
        15. 跨學科合作交流：和其他領域專家合作，提供醫療衛生或其他領域多元化的視角與思維。
        16. 尊重他們的病人 : 應該尊重病人的自主權，並且在所有決策中都考慮到病人的需求和願望。
        17. 探討及推廣生死學:目前亞洲社會對於生死的議題仍較保守，探討相關議題的族群多為哲學家和宗教家；因此，有許多相關生死經驗的醫生應該投入探討和推廣生死學。目前社會上已有部分例子，如善終概念的推行、器官捐贈（取代保留全屍的傳統觀念）和推廣大體老師的概念。
        
    - 醫師的社會責任可粗分為「法律責任、倫理責任與自發責任」三種：
        1. 法律責任：法律是社會對與錯的法典，而醫師的法律責任明定在相關的醫療法典當中， 以規定相關醫護人員的責任、權利與義務。所以，醫療法典中關於醫師的規定可算是醫師社會責任的最低要求。
        2. 倫理責任：身為醫師應該「奉獻自己、尊重師長、以良心來從事醫業、重視人類的生命與價值、維護醫事專業的傳統、不做違反人道的事、不因他種因素給予病人不同的對待、與其他醫療團隊充分合作」等，說明身為醫師倫理責任的所在。
        3. 自發責任：指身為社會公民所應盡到的責任與義務。每個人基本角色本都包含專業面與社會面，關切社會事務本是每個人擁有的權利，只是在參與社會事務時應如何謹守專業範疇的規範與要求。（如在提及醫事倫理時，都有強調「醫生除了精通醫術之外，尚須修身立德，要發濟世的宏願，不要做出違背道德的事情。」而現今歐美國家地區的醫師自發性地「組織醫療人權組織、醫療服務組織、 醫療反戰組織、提出相關宣言、參與政治事務」等，都為此原則做了最好的注釋。）

# 病毒＆細菌
- 結構
    - 病毒：病毒的結構相對簡單。它們主要由核酸（DNA 或 RNA）和外層的蛋白質組成。病毒需要寄生在宿主細胞內才能繁殖。
    - 細菌：細菌的結構較為複雜。除了核酸外，細菌還有細胞壁、細胞膜和細胞質等組成。細菌是單細胞生物，可以在其他細胞內外存活。
- 大小：病毒比細菌小得多。病毒的大小約是細菌的10到100倍。
- 生存方式
    - 病毒：病毒必須依賴宿主細胞才能繁殖。它們入侵宿主細胞並利用細胞的機制來複製自己。
    - 細菌：細菌是獨立的生物體，可以在宿主內外存活。它們無需寄生在其他細胞內。
- 致病性
    - 病毒：大多數病毒對人體通常具有致病性，如流感病毒、感冒病毒等。
    - 細菌：細菌則是有益菌與壞菌。人體內有許多與細菌共存的菌種，其中大部分是有益的好菌。但某些細菌，如沙門氏菌、金黃色葡萄球菌等，對人體不利。
- 傳播方式
    - 病毒：病毒通常通過空氣飛沫、接觸或食物傳播。例如，感冒病毒和流感病毒可以通過咳嗽和打噴嚏傳播。
    - 細菌：細菌可以通過接觸、食物、水和昆蟲傳播。例如，沙門氏菌可以通過食物中的污染傳播。
- 生命週期
    - 病毒：病毒無法獨立生存，必須寄生在宿主細胞內。它們的生命週期包括感染宿主、複製自己的核酸，然後釋放到宿主細胞外。
    - 細菌：細菌是獨立的生物體，具有自主生存能力。它們可以在不同環境中生存，並以分裂方式繁殖。
- 藥物治療：抗生素可用來殺細菌，通常一種抗生素可對付數十種細菌，但要吃到一定療程否則細菌會產生抗藥性；但是抗生素並不能殺病毒，而所謂的抗病毒藥物通常只能針對特定一種病毒，還經常會失敗，最常見的例子就是愛滋病毒HIV，用以治療的雞尾酒療法，就是將不同種藥物結合來執行。

- 如果消失會對人類產生什麼影響？
    - 病毒在世界的作用：
        - 支撐生態系統
            - 噬菌體是海洋中細菌種群的主要調節器，控制細菌菌種不過度增生，壓縮其他細菌菌種生存空間。
            - 通過淘汰微生物，病毒確保產氧浮游生物有足夠的營養進行高速率的光合作用
            - 物種控制：當某一物種過剩時，病毒會傾向快速複製並消滅物種，為其他物種創造生存空間（如人類大流行病）
        - 維持有機體健康
            - 有些物種會依賴病毒生存，在競爭中取得優勢
            - 黃石公園例子：一種感染真菌的病毒使草變得對地熱土壤溫度具有耐受性，單靠真菌無法做到
            - 某些病毒能幫助抵禦病原體
            - GB病毒C型與延遲人類愛滋病病症進展有關
            - 皰疹使老鼠不易受到某些細菌感染，包括黑死病和李斯特菌，而對於人類而言雖然可能造成免疫疾病，但某種程度上亦賦予免疫益處，可視作一種共生關係
        - 未來應用：發展於藥物開發與疾病治療
    - 細菌在世界的作用：
        - 分解死亡生物體，使其轉化成生物生長所需營養物質
        - 分解垃圾與有毒物質，延遲人類對於環境的衝擊
        - 協助生物消化作用、製造所需維生素及礦物質
        - 牽制其他微生物生長（藉由競爭或毒殺），可進一步避免生物體生病

# 食物中毒
- 症狀
    - 腹瀉、噁心、嘔吐、腹痛、發燒、頭痛、虛弱、血便、膿便
    - 敗血症：敗血症是微生物(細菌、黴菌、病毒)侵入人體在血液內繁殖並且產生毒素引起的全身性的發炎反應，病人會有發燒、心跳加速、呼吸急促、白血球上升、尿少、發紺、斑駁樣皮膚、意識變化等生命不穩定徵象，病情可能在數小時至數日內快速惡化。
    - 急性肝衰竭：又名暴發性肝炎，病人大量的肝細胞在短時間內嚴重發炎及死亡，導致肝臟沒辦法維持排毒功能，引起嚴重的併發症，包括感染、出血、腎功能衰竭及腦水腫

- 常見原因
    - 化學性食物中毒
        - 製造、加工時滲入化學物質：多氯聯苯、多溴聯苯
        - 工業的廢棄物汙染：重金屬
        - 農業及畜牧業的殘留化學物：DDT、生長促進素
        - 對人體有害的食品添加物：硼砂、吊白塊
        - 包裝材料的溶出物：甲醛、甲醇
    - 細菌性食物中毒
        |細菌類別|污染食物|臨床表現|
        |-------|-------|------|
        |沙門菌屬|肉類、禽蛋類|先有腹痛嘔吐，繼而腹瀉，多伴有發熱|
        |副溶血弧菌|海產品、醃漬品|先有腹痛發熱，繼而有腹瀉，部分患者伴有嘔吐|
        |變形桿菌屬|隔夜剩飯菜、肉類、魚類|胃腸型：腹痛、嘔吐、腹瀉；過敏型：皮膚潮紅、頭痛、頭暈、蕁麻疹|
        |蠟樣芽孢桿菌|隔夜剩飯菜、肉類、乳類|嘔吐、腹痛及腹瀉|
        |大腸埃希菌|隔夜剩飯菜、肉類、乳類|先有食慾缺乏，繼而腹痛及腹瀉|
        |金黃色葡萄球菌|澱粉食物、肉類、乳類|先有噁心頭痛，繼而腹痛及嘔吐|
        |肉毒桿菌|罐頭食品、密封食品、豆製品、蜂蜜|突然發病，主要為中樞神經系統症狀，如乏力、頭痛、眼瞼下垂、吞咽困難、尿瀦留等|
- 處理方式
    - 停止食用受污染食物：如果您意識到食物可能已經被污染，請立即停止食用。
    - 保持身體水分：儘量多喝水，以防脫水。
    - 就醫：如果症狀嚴重（如持續嘔吐、嚴重腹瀉、高燒等），請及時就醫。醫生可能會推薦適當的藥物來幫助治療。
    - 觀察症狀：注意可能出現的症狀，如噁心、嘔吐、腹瀉、發燒等。
    - 催吐：如果吃下食物的時間不長，在1～2小時之內，毒物還停留在胃內，可以試著把毒物吐出，減少毒素吸收。可以用手指、匙柄、筷子等刺激舌根部；或快速飲用冷鹽水（食鹽20克，加溫開水200ml冷卻後使用）、薑汁來催吐。
    - 勿服用非處方止瀉藥：因為會延長腸道內容物留滯時間，可能會助長人體吸收毒素。
- 預防
    - 遵守個人衛生原則，確保自身飲食健康
    - 遵守食品處理之原則 
        - 區分生食、熟食，避免交叉污染，食物需徹底煮沸
        - 注意保存溫度，區分冷凍、冷藏
        - 注意水、食材、廚具、人員清潔（傷口要包紮）
        - 廚師檢查 e.g. 聞、異狀
    - 注意外出用餐習慣 e.g. 避免冷食、生食、來路不明的食品
    - 食物來源提供、上游廠商 e.g. 注意衛生、食品來源監管、食品保存、清潔、定期審核
- 米酵菌酸
    1. 在印尼椰子發酵餅發現
    2. 一種強烈呼吸毒素→影響呼吸作用路徑
        - 抑制ADP/ATP 轉運酶(ACC)，使ATP無法離開粒線體
    3. 由發酵穀物、椰子等受到唐菖蒲伯克氏菌污染產生
        - 唐菖蒲伯克氏菌可分泌毒黃素、米酵酸菌
        - 在特定種類濃度之下才會分泌米酵酸菌
            - 受脂質的種類、濃度影響，決定分泌何種毒素 -> 脂肪含量低於20%的食材引發的中毒是由毒黃素 -> 產生米酵菌酸需一定的濃度
    4. 毒性極高，1毫克即可致命
    5. 其好發於室溫、酸鹼值中性下、發酵1-2天及有一定脂肪含量食物之中，故預防和抑制可藉由:15°C及以下低溫、短期保存及酸性環境。
    6. 症狀
        - 腸胃道症狀，例如噁心、嘔吐、上腹部不適、腹瀉等，還常伴有頭暈、全身無力等症狀
        - 嚴重中毒者會出現肝腫大、黃疸、皮下出血、少尿、血尿、煩躁不安、意識不清、抽搐、休克等症狀
        - 重症中毒病人多因呼吸衰竭而死亡
# 如何分辨真假訊息
1. 對標題保持懷疑態度：假新聞/假消息常會使用聳動的標題，如果標題內容令人難以置信，很有可能就是不實報導。
2. 調查消息來源：如果報導來自不熟悉的機構或組織，請調查它們的詳細資料，了解更多背景資料再對訊息的真實性進行判斷。
3. 查詢相關報導：如果有多個具公信力的來源都報導相同的內容，則新聞內容較可能屬實。
4. 只分享可信的消息：還沒確認新聞/消息的真實性之前，不要在網路上分享。
5. 善用各種假訊息查證管道（如：LINE官方提供之「LINE訊息查證」帳號、趨勢科技「防詐達人」等)，彙整各方提供的資訊後再對新聞/消息的真實性進行判斷。
# 中暑
- 定義：人長時間處於水分易蒸發、流失的空間或狀態下（如高溫或悶熱環境中），又未適時補充水分及鹽分，造成身體熱量無法順利排出，核心體溫升高，出現各種不適症狀。
- 症狀：身體燥熱、皮膚乾燥發紅、低血壓 、心跳加快、呼吸加快、神經系統異常（行為異常、幻覺、意識模糊不清、精神混亂等）、口渴、疲倦、全身無力、頭暈、頭痛、說話喘、激動、焦慮、昏迷、抽搐、大量出汗但也可能會無汗、臉色蒼白，嚴重時，患者還可能意識不清、昏迷或休克。
- 成因：中樞神經系統失調，包括體溫調節功能失常，加劇體溫升高，使細胞產生急性反應。通常是熱衰竭未及時發現所致，體溫可能超過40度。
    - Q：跟熱衰竭有什麼不同？
      熱衰竭成因：大量出汗嚴重脫水，導致水份與鹽份缺乏所引起的血液循環衰竭。多因熱痙攣、熱暈厥演變而來。患者可能流失過多水分，或有補充水分但卻缺乏鈉離子，體溫通常介於38度~40度，可以稱為中暑的前兆。
- 分類：根據衛福部，中暑可分為傳統型中暑跟勞動型中暑
    - 傳統型中暑：身體無法應應環境溫度和濕度
        - 高危險人群：年長、肥胖、慢性病、嬰幼兒、孕婦
    - 勞動型中暑：長時間在高溫高濕度的環境下工作所導致
		- 高危險人群：軍人、工人、在密閉無空調空間工作的人、農夫、運動員
- 預防：在外盡量在陰涼處行走、適時補充水分及防曬產品、穿著淺色衣物並保持身體涼爽
- 急救五步驟：
  1. 陰涼：將患者從高熱的環境中移至蔭涼通風處。
  2. 脫衣：鬆脫衣物、足部稍微抬高、平躺休息，若有嘔吐現象則側躺保持呼吸道暢通。
  3. 散熱：全身皮膚噴灑水，通風（扇風）促進排汗，將冰袋（毛巾包覆冰塊或冰涼飲料罐）放置於頸部、腋窩和鼠蹊部（因為這些部位血管較淺薄） 。
  4. 喝水：意識清醒者可給予稀釋電解質飲料（如運動飲料）或加少許鹽的冷開水。患者意識不清時，不可給予飲水。
  5. 送醫：可撥打 119 或自行送醫。
- 常見迷思
    - 吃退燒藥是沒有幫助的：熱傷害主要是因為體溫調節中樞異常，使身體產熱與散熱失調，因此對退燒藥劑並不會有反應
    - 利用擦酒精來散熱：若於體表擦拭酒精，會使體表的水分蒸散(暫時會有涼感)，但它卻會引起表層血管急速收縮，反而有礙身體的散熱

# 韓國醫師罷工
- 背景：韓國實習醫生和住院醫生輪班時間為36小時，美國則不到24小時，有半數美國年輕醫生每週工作60小時甚至更少，而韓國醫生工時往往超過100小時。因此造成高工時的工作條件，加上惡劣的勞動環境、各科別醫師的分佈不均、高訴訟風險，這些都不是靠擴增醫學生人數可以解決，不過韓國政府認為是醫療人力缺乏導致這些問題，所以選擇以擴編方式處理這項問題，因此造成這次罷工。
- 影響：
    - 院方：原先就因資深員工多、人事費用高，幾乎沒有盈餘。加上最近的罷工，每天幾乎呈虧損狀態。大型醫院每日虧損數十億韓元、中型醫院每日虧損約七億韓元（1680萬新台幣）。因此，院方開始呼籲政府擴大醫院低利融資規模。
    - 醫師：		
        - 教授：集體辭職以示抗議
        - 一般醫師：以離職方式表達抗議。因此導致手術、診療量減少，病床使用率大幅下降。
    - 醫師以外的其他醫療人員：被醫院要求配合醫師數的減少而扮強制性的休無薪假
    - 醫學生：以休學的方式來抗爭。此外，增加醫學生招生名額（預計最少招收2000名）可能導致人氣的專科更飽和，而人氣低的專科將更少人去，醫療品質可能進一步下降。
    - 民眾：延後就診，有些甚至因無法及時做醫療處置而死亡
    - 政府：否認醫學院新生的增加會導致醫學院教育品質下降、吊銷不回崗位的醫師
- 延伸討論：為什麼台灣沒有類似的醫療抗爭？
    - 台灣政府多以接受輿論、回應民意改變政策方向作收：意味政府的反應其實不慢，且醫界不太需要透過極端的方式便能夠影響公共政策。
    - 台灣的政府對於醫學生名額控管非常嚴格：名額正好是影響醫師長期收入與工作條件最大的關鍵，醫師名額已經穩定很長一段時間（每年國內醫學生約1000，國外醫學生約 200）。即使曾有過小增加也是增加公費生的名額（+100），但這些人專科訓練完後，就要被分配到偏鄉的特定科別服務，不會搶佔大多數醫師的市場，所以這個政策雖然有引起小爭議但還算是過關。

- 公益社團:醫療相關服務社團，例如紅鼻子醫生、路竹會
- 宗教信仰：許多安寧療護病房設有宗教室，若醫生能對於各宗教有一定了解程度，宗教信仰可以在精神健康治療中發揮作用。一些醫生可能會將宗教信仰納入心理治療過程中，以幫助患者應對情緒困擾或心理壓力，並增強病患受同理之感受。此外宗教信仰通常帶來一套明確的價值觀和道德準則，鼓勵醫生堅守正直、慈悲和同情心，這有助於他們在醫療實踐中做出正確的道德和倫理決策，並提供更溫暖、人性化之醫療服務。
- 運動
    - 身體健康：運動有助於增強身體素質和免疫力，降低患上慢性疾病的風險，包括心血管疾病、糖尿病和高血壓。
    - 另外醫學學習和醫師工作壓力大，運動是釋放壓力的有效途徑，可以減輕焦慮，提升心理健康。運動亦能促進大腦血液循環，提高專注力和記憶力，有助於學習和工作效率的提升，並有助於調節生物鐘，改善睡眠質量，提高睡眠效率，使人精神狀態更佳。
    - 而長期坐姿學習和工作容易導致姿勢不良和肌肉僵硬，運動可以改善身體姿勢，增強身體靈活性，減少患上骨骼肌肉問題的風險。
    - 參加團隊運動或運動活動也可以培養團隊合作和溝通能力，增進與同儕和同事的關係。
    - 總的來說，醫學系學生和醫師培養運動習慣有助於身心健康的提升，增強工作效率和生活品質。
- 旅行
    - 提升人文關懷：旅遊可以培養醫師對於患者的人文關懷，通過了解不同文化背景下的人們和生活方式，更好地理解和尊重患者的需求和價值觀。
    - 跨文化交流：透過旅遊，可以接觸到不同文化背景和醫療系統，了解不同地區的醫療實踐方式和醫療文化，有助於培養跨文化溝通和理解能力。
    - 學習新技術和方法：在旅遊過程中，有機會參觀不同國家的醫療機構或參與相關活動，學習到當地先進的醫療技術和方法，豐富自己的專業知識。
    - 處理不同環境下的挑戰：旅遊經驗讓人習慣於處理不同環境下的挑戰，訓練臨機應變的能力，在面對醫療行為的緊急時刻派上用場。
    - 促進團隊合作和溝通：與同行一起旅行可以加強團隊合作和溝通能力，這對於在醫療團隊中的協作至關重要。
# 工作、家庭和生活如何兼顧
- 制定優先事項：醫生需要確定工作和家庭生活中的優先事項，並根據這些優先事項來安排時間。這樣可以確保將時間和精力集中在最重要的事情上。
- 建立良好的溝通：與家人和同事之間建立良好的溝通至關重要。分享工作和家庭的計劃，並確保每個人都明白彼此的需求和期望。
- 合理分配時間：嘗試在工作和家庭之間建立平衡，合理分配時間。這可能意味著需要學會說「不」，以確保有足夠的時間和精力處理工作和家庭事務。
- 設定界限：確定工作和家庭之間的界限是重要的。努力在工作時間專注於工作，在家庭時間專注於家庭。這樣可以避免工作和家庭之間的混淆和衝突。
- 實施自我照顧：醫生需要照顧好自己的身心健康，這包括足夠的睡眠、健康飲食、適量的運動和時間與家人和朋友一起放鬆。
- 彈性安排：儘管醫生的工作時間可能不太靈活，但盡量尋找一些彈性安排的方式，以應對突如其來的家庭需求或個人需求。
- 共同承擔責任：家庭成員之間應該共同承擔家庭責任，例如家務、照顧孩子等。這樣可以減輕醫生的負擔，讓他們有更多的時間專注於工作和自我照顧。
- 總的來說，兼顧工作、家庭和生活需要計劃、組織和努力。重要的是要意識到平衡是一個持續的過程，需要不斷調整和改進。
# 醫療糾紛
- 常見原因
    - 醫病溝通不良
    - 服務不周或態度不佳
    - 收費過高或家屬不滿收費
    - 診斷過程未詳細告知
    - 誤診或漏診
    - 不滿意醫療程序
    - 醫療或手術有疏失不當
    - 醫病關係信賴不當
    - 因醫療知識進步而對醫療處置不滿
- 如何避免
    - 以醫師角度
        - 病人及家屬反映問題後，應最好親自探視：避免大問題以小病症表現時誤判嚴重程度、拖延治療，使病人病情因疏忽而加重
        - 增加護理法律知識：了解您的權益和醫療法規，以便在需要時能夠採取適當的行動。
        - 緊急事件應付的步驟：如果您遇到醫療緊急情況，請儘快尋求專業幫助，並保留相關證據。
        - 加強在職教育，提昇品質注意安全：醫護人員應不斷學習和提升專業知識，以確保高品質的醫療服務。
        - 增進人際關係：建立良好的醫患關係，尊重彼此，並在溝通中保持開放和誠實，對於治療的疑慮及病情盡量多加說明。
        - 了解醫療法律：熟悉醫療相關法規，如醫療法、醫師法等，確保行為符合法律規定，避免法律風險。
        - 告知醫學的不可控性：明確告知患者醫療行為無法保證100%治癒疾病，理解並尊重醫學的不確定性。
        - 建立明確的醫療契約：確定治療關係是基於委任契約或承攬契約，明確雙方的責任和權利，避免混淆和誤解。
        - 履行告知責任：對患者進行全面的告知，包括治療選項、預期效果、可能的併發症和風險，確保患者理解並同意治療計劃。
        - 實踐病主法：尊重患者的意願和權利，充分尊重患者的自主權，讓患者參與醫療決策。
        - 詳實記載病歷（病歷完整化）：紀鍵性資訊的詳細紀錄，包括患者病史、診斷、治療過程和結果，以及患者與醫師之間的溝通內容，確保醫療紀錄的完整性和準確性。
        - 醫療行為時有第三人在場
    - 以病人角度：
        - 就醫前：加強預防措施，選擇正規且有良好聲譽的醫療機構，並與醫生建立良好的溝通和信任。
        - 條列清楚不適的症狀和想問的問題，有禮貌地請教醫師，有疑問處應當面問清楚。
        - 診斷後：當醫師建議要開刀或告知為重大疾病後，可帶著病歷和檢查報告，找另一位同樣專科領域的醫師再次確認。
        - 治療中：如果遵從醫師指示治療一段時間後，病情仍然沒有改善，不妨試著請教醫師：「為什麼會這樣？」若無法得到答覆，應帶著病歷和檢查報告，並且條列不適情形與治療歷程，尋求另一位醫師協助。
- 如何處理
    - 院內申訴：先詢問主治醫師，再向醫院內的醫療爭議處理小組、關懷小組申訴，尋求協助和解決。

    - 醫糾調處服務：尋求醫師公會、當地衛生局的醫療糾紛調解委員會等協助。
    - 法律訴訟：如果其他方法無法解決，您可以考慮提起法律訴訟

# 癌症
- 十大癌症：肺癌、大腸癌、女性乳癌、肝癌、口腔癌(含口咽、下咽)、攝護腺癌、甲狀腺癌、胃癌、皮膚癌、胰臟癌
- 常見的治療方法
    - 外科療法：利用外科手術切除遭癌細胞侵襲之組織
    - 化學療法：注射或口服藥物，殺死癌細胞或抑制癌細胞擴張
    - 放射療法：以放射線照射癌症患部，殺死癌細胞組織
    - 免疫療法
        - 免疫單株抗體藥物：將製造此種抗體的免疫細胞與癌細胞融合，使其同時具備製造抗體與不斷分裂的能力，具有抑制病變組織訊息傳遞、誘導凋亡等功能
        - 自體免疫細胞療法：從病患血液分離出富含淋巴球的單核球細胞，再將殺手細胞、細胞激素、腫瘤抗原共同培養，藉此活化增殖免疫細胞注回患者體內
          ![](https://elearning.canceraway.org.tw/uploads/FCF_201912165350.png)
    - 標靶治療：針對癌細胞特殊表現或轉移惡化機轉為標的之治療，有口服、注射兩種
    - 賀爾蒙治療：乳癌、攝護腺癌等癌細胞與賀爾蒙激素有關之抗賀爾蒙治療
    - 幹細胞治療：著重於醫療後的復原與再生，以其餘方式消滅癌細胞後，利用幹細胞將被癌細胞侵蝕的組織與血球修復
    - 心理療法：加強患者及家屬之心理建設，如腫瘤心理師諮商
    - 緩和醫療：針對癌症末期患者之不適症狀，運用各種藥物或輔助性療法減輕病人症狀
- 癌症減藥
    - 起因：以「最高耐受量」為出發點的概念可能不適用癌症藥物治療
    - 目的＆優點
        - 減少病人副作用以提升病人生活及護理品質
            - 例：Sotorasib (洛滿舒) 的使用頻率降低可減少其對患者所造成的腹瀉以及肝臟問題
            - 例：低劑量放療對於女性乳癌的復發率僅比高劑量高出1.6個百分比（4.4% 對 2.8%)，但對於患部纖維化的機率則是明顯低於接受高劑量放療的患者
        - 減少病人開銷
            - 例：最近的一項分析確定了56種口腔癌藥物，透過減少劑量或改變療程，其成本可以至少削減33%，這種方法可以為患者平均節省94,000美元
        - 減少醫療資源浪費
    - 隱憂
        - 藥物銷售額大幅下降引起經濟問題
    - 推動：FDA Project Optimus
      重新制定腫瘤治療藥物的用藥效用最佳化、藥物選擇的方針，希望藉由這些指引讓病患接受較低劑量但長期的治療，提升病患存活率並提升生活品質
        - 具體作法：
            - 鼓勵學者研究並透過公開會議、工作坊等方式交換藥物開發、效益最佳化的資訊
            - 鼓勵藥廠在開發藥物初期（為了通過核准而展開試驗前）和FDA討論相關議題
            - 制定衡量臨床與非臨床對於藥物劑量等研究的標準，希望藉此來加速藥物減藥的進程
# 大體老師不足
- 遺體捐贈：大體捐贈，指在個體死亡之後，基於教育或研究用途捐獻遺體的行為。除了人類捐贈者，也有飼主捐贈動物遺體。醫學界會尊稱捐贈者為「大體老師」或者「無言老師」
- 流程：
    1. - 至各大醫學中心簽署大體捐贈同意書，除本人意願外，大體捐贈同意書還需要「受任人」簽署同意：配偶、子女、父母、兄弟姊妹
       - 生前未簽署捐贈志願書者，往生後家屬必須護送大體到受贈之醫學院並且簽署捐贈同意書
    2. 往生後，由家屬通知醫院等受贈單位出面評估大體是否堪用
    3. 受贈贈單位接回大體，妥善保存，等待進行醫學解剖研究
        - 防腐至少需要一年以上，才可做教學解剖，大體從防腐措施到教學結束、火化、安奉骨灰需要二年以上時間
        - 學生訪談家屬、啟用、入殮等追思儀式
    4. 大體完成解剖研究後，由家屬接回安葬或由受贈單位妥善安葬
- 限制
    - 需要親屬同意，才可以捐贈。親屬順序如下：配偶、子女、父母、兄弟姊妹
    - 必須是16歲以上自然死亡或病故者。若16歲以下適宜捐贈者，需有法定代理人簽名同意
    - 評估是否適合捐贈，24小時內都可接受
        - 無法定傳染病（如愛滋病、肺結核鈣化等）
        - 不能太胖、太瘦 → 超過或低於依身高計算的標準體重百分之五十者不適合捐贈：長期臥床病人容易肌肉萎縮，造成身體消瘦、肌肉組織變形；體重過重、脂肪含量過高也不行，因脂肪無法防腐，可能孳生細菌，而且也不易搬運及下刀
        - 器官移植後傷口無法癒合，不能作防腐，故不能再做教學解剖
            - 不能同時器官、大體捐贈 → 以器官捐贈優先
            - 器捐評估後只捐眼角膜，則可由醫學院再評估是否能捐大體
        - 手術傷口需痊癒
        - 自戕、溺斃及嚴重水腫、車禍、手術中往生、嚴重創傷、褥瘡、四肢變形或萎縮、冰存過久致遺體明顯破壞者，不可以捐贈
- 捐贈不足：理想的學習品質為6至8人共用一副大體，現況為10-15人一副（北部更缺乏；慈濟：宗教因素所以比較沒有缺乏的問題）
    - 慈濟醫學系：
        1. 扎實解剖課程：大三解剖課5人一組（每間學校不同，通常約10-15人），實作機會充足。
        2. 解剖課程合併許多人文典禮舉辦（如:家訪、啟用典禮、送靈典禮、追思音樂會）；大五進醫院後需要回學校上模擬手術課程，也會再次接觸無語良師與相關人文典禮。
    - 阻礙
        - 家屬不願配合
        - 人們忌諱死亡，生前沒有考慮如何處理自己的屍體
        - 解剖學對大體要求嚴格
        - 文化宗教 e.g. 傳統葬禮觀念（保留全屍、入土為安）
    - 解決方式
        - 加強教育宣導（包含家人）
        - 校際調度：目前北部較不足
        - 改善法律、制度
            - 1984 解剖屍體條例：無親屬認領屍體，可由通知、由醫學院認領 → 各縣市很少執行（法條太冷門，被長期忽略）
            - 台北市處理無名屍體條例：依解剖屍體條例規定辦理或由殯葬管理處依規定處理，新北市可跟進（目前僅由殯葬業處理）
            - 科技 → AR、VR、人體影像解剖桌（中國醫、義大；長庚：3D）、虛擬實際手術模擬
            - 放寬對大體的要求
    - 補充
        - 簽下意願書之後，也要為自己的後事做安排，包括去世後要將身體送往哪裡？火化之後想要如何處理？如樹葬、海葬、花園葬等等。將該交代的事情都交代清楚，將自己的決定也一併通知重要他人，以防在自己去世之後，轉由家人承受不必要的壓力
        - 漫長告別：大體一直在醫學院等待著給醫學生上課，家屬就像懸掛著一件未完成事件：往生者未出殯。異於常態和傳統的死後處理，讓留下來的人也必須承受異於常人的壓力，及延長性的悲傷
            - 在還沒進行教學前，家屬都可以前來探望
            - 大體老師要做一年的防腐，家屬在第二年之後才能探
        - 大體老師啟用儀式：成大 → 分成佛教、基督教
