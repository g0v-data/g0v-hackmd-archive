duckduckgo 有許多 goodie 或 keyword trigger 能去查詢或轉接第三方搜尋引擎。目前還有個 "meaning"，能直接顯示 wikipedia 條目。

字典類的 keyword trigger 目前有個 Urban dictionary，關鍵字是 ud:

    [https://duckduckgo.com/?q=camper+ud&kp=-1](https://duckduckgo.com/?q=camper+ud&kp=-1)

萌典應該可以做個特殊的 API endpoint 接受來自 duckduckgo 的 query。
