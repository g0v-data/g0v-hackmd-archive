# 訂定急診擁塞指標

一、  訂定急診擁塞指標
目前國內各大醫院急診壅塞之情況極為嚴重，但並無統一之標準可供界定何謂急診壅塞，亦無良好之監測系統可以顯示急診室壅塞之即時情況與壅塞成因，作為醫院或主管機關因應之參考。急診的服務狀態會隨著時間而變動，目前衛生福利部要求各個急救責任醫院要在醫院網站公佈急診即時資訊，其中包括1.是否已通報119滿床2.等待看診人數3.等待推床人數4.等待住院人數5.等待加護病房人數等五項急診擁塞指標。本論文將收集各家急救責任醫院的急診即時資訊作為資料庫，並轉化為大眾可以理解的紅黃綠燈號。本系統預設之急診狀態收集與回報機制以五分鐘為單位，抓取其急診服務相關參數至本服務平台的伺服器。
|  | 紅燈 | 黃燈 | 綠燈 |
| --- | --- | --- | --- |
| 通報 | 是 | 否 | 否 |
| ２等待看診人數 | >10 | 5~10 | <5 |
| ３等待推床人數 | >3 | 1~3 | 0 |
| 等待住院人數 | >20 | 10~20 | <10 |
| 等待加護病房人數 | >5 | 1~5 | <1 |
