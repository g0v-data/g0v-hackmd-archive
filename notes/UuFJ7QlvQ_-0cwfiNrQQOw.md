### 合作方式

- 風傳媒希望可以在選舉議題上能有視覺化新聞的合作
- 可以多引用 g0v 的成果，協助 promotion
- 在新聞解讀的部份可以提供意見跟 feedback
- 上述兩點是希望風傳媒多當當 user
- 建議風傳媒可是試行部分內容創用 CC 授權出來
- 像全民記者會這樣的東西可能也能合作，不過合作形式可以再討論。也許是 g0v 的某些成員和風傳媒合作，或是用 api 的方式連結，或是 g0v 做網站，風傳媒可以幫忙有記者去問候選人....
- 立場方面
    - 零時政府保持立場中立
        - 由風傳媒提需求，g0v 專案實做，開放成果，任何人都能用，但不保證產出
        - 或由個人參與，口頭、簽約、聘雇形式貢獻
    - Kirby 針對資料新聞這部分提到的合作模式
        - 零時政府取得資料, 公開
        - 風傳媒透過資料了解並提出議題
        - 零傳媒以 CC 方式開放實作成果，提供成果引用
