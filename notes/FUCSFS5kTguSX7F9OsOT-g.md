### 教案使用
- 多常使用教案輔助教學？是否因人/科目而異？
> 個人備課時不會寫，但研發課程時用教案溝通討論，尤其ＯＯ很常會有創新課程，還是常會寫教案
- 和課程內容相比，教案在教學中扮演怎樣的角色？
>- 老師個人經驗，寫教案通常是較有目的性（比如要參加競賽），校內有很多社群。因為有目的性，所以寫完都會實際執行看看。
>- 例行課程不太會寫教案（比如考試一定會考的那些），書商會給很多教案，較常使用的時機與對象是新手教師與新課綱下無所適從的老師。
>- ＯＯ的老師相對年輕，平均四十歲，國中部導師都是四十歲以下，所以學校較有動力積極推動新課程。
- 如何決定是否要在課堂中使用某個教案？
- 教案使用後如何評估教案對學生的學習是否有幫助？
>- 正規教案內會有學習評量與目標相對應。課程結束後會檢核學生是否達撰寫教案時設定的目標。
>- 公民科情意導向，難以在課堂中直接檢核（可能需要透過公民行動方案、其他活動等）


### 教案內容
- 一個好的教案應該具備哪些元素？
>- 教案用意：提醒其他要使用這份教案的人，該準備的內容。教案的反思很重要，可以避免重蹈覆徹。
>- 學生背景、執行後狀況（上完課及議課，並更改教案）
>- ＯＯ有校內共用雲端，可供傳承與代理老師銜接，包含各科與多元選修。也留有歷年的檔案變化（以ABCD命名），舊版的保存是重要的，因為改掉的版本並非不重要，因每年學生的特質不同，今年刪掉的部分明年可能又能適用。
>- 公民科常見主題：媒體素養、人權教案、國際議題，這些常見的主題就可以拿別人的教案來修改挪用
- 嘗試過哪些長度的教案？怎樣的教案長度教學效果最好？
>簡案：
>中案：
>詳案：包含預期的學生反應、時間
>共享比較偏向簡案與中案，因為只是要抓大方向

>一份教案頂多五六小時（兩到三堂課），一週以內可執行完的份量，若太長難以聚焦
- 目前使用的教案會使用哪些多媒體素材？例如：影片/圖片/音源/podcast/電影
>- 書商現在多會放在雲端，熟悉課程內容或有教學經驗的老師會有自己的資料庫（例如簡報，但現在導入「學習共同體」較少使用簡報）
>- 影片來源：書商光碟、YouTube
>- 老師個人：每課每年段都有建立資料夾，每當看到一個適合的新聞，就會放入對應章節的資料夾。
>- 若是例行課程，對於課程流程都已熟悉，故教案的使用程度會少於素材。剛任職的前一兩年會花較多時間找適合自己風格的素材。

>- 資深老師不一定會寫教案（所以也難以分享），年輕老師的共享風氣較盛，但不一定是可調整的格式（如pdf）。參考的人也會轉換成自己的風格。
>- （老師的經驗）教學現場自私的人比較少，願意共享的人多。

### 教案製作前準備
- 通常是什麼原因會想製作新的教案？
- 如何構思教案idea?
>- 每個人經驗不同，邱老師喜歡使用數位媒材，個人習慣用簡報整理。
- 如何構思教案大綱及內容？
>- 先有學生學習目標（學生本位，並非蓋好房子才塞人進去），再有大綱（骨頭），最後產生內容（肉）
- 開始製作前會花多少時間蒐集相關素材？
>- 難以估計，資料是由經驗累積而成。新手教師可能針對知識概念也不太了解
>- 新課綱實施下資深老師也需要動起來了XD 為了搶時數避免被抓去當行政XDD 所以會有非常多參考新式課程的需求（老師覺得是長期、動態性的影響）新課綱實施下學生可以選擇老師的課，所以課程也會有開不成的壓力。
- 會使用哪些網站做事前研究/蒐集素材?
>- 議題性課程（keywords: 12年國教「重大議題」），政府單位就有建制素材網站（不一定是教案），例如「媒體素養網」就放很多經認證的得獎教案、環境教育、「性別教育網」性別議題、國際交流（ＯＯ高中推動的）、「海洋教育教育資源中心」、運算思維
>- 臉書粉專「廣告裁判」的內容蠻有教育意義，老師很常拿來參考
>- 政府篩選過的內容，會相對無聊（例如極度正向，過於政治正確），但也較安全。自己找的網路資源需自己看過確保不會對學生有不良影響。(如:臺北市海洋教育資源中心)
- 政府已經有整理好的資源，什麼時候會又需要自己搜集？
> - 學生有興趣的
> - 符合自己風格的
- 是否在製作前找人討論交換想法？
>- 學校管道之外，參加研習完也會跟分享的老師索取教案，有些老師也會公開放在網站上（或FB or Line）
>- 一直埋頭自己研發容易疲乏，老師平常就會有習慣搜集素材，對老師而言不會是額外負擔
>- 願意參加研習的老師通常是比較積極向上的，所以在這個場合認識的老師也會比較有討論或分享的機會
>- 老師認為素材包很有需求（也比較不會侷限於個人風格與習慣）

### 教案製作
- 一份教案通常需要花多少時間製作？
> 約兩週
- 製作是花上班時間還是自己私人的時間？
> 常態使用私人時間，因老師有行政職，假日備課已經成為生活常態所以也能接受
- 是否會和其他老師合力製作一個教案？
- 製作時通常會使用哪些軟體？為什麼選擇這些軟體當製作工具？
> 校內 GSuite
- 一學期大概需要製作幾個新教案？
> 少的話兩三個

花最多時間準備：老師們聚焦課堂帶領方式。
參加比賽的教案：著重「條目的挑選」。
每年都會調整教案，依據每年學生特質與狀態的不同（教案上會有一個欄位是針對什麼特質的學生）。

### 教案共享
- 學校內是否提供老師教案交流的方式？
> - 雲端之外，主要都是以Line溝通
- 學校外會透過哪些管道/社群和其他老師交流教案？
- 是否常使用其他老師製作的教案？
- 是否會主動和相關科目老師分享自己製作的教案？
>- 教案完成後，會優先和同科老師討論。同科老師優先（因學生性質相仿），若是議題性的可找過去同系同學、同科同領域的朋友討論
- 共同備課一份教案的方式？
>- 看教師群體的習慣，比如老師現在常用Google doc，由熟悉資訊的老師開端
>- 同步編輯的需求不高，通常會線下討論完再由一位負責線上編輯
>- Google doc 蠻符合現在的使用情境，缺點是格式容易跑掉

嘉豪展示了 hackmd，老師覺得如果教學校老師，可能會願意嘗試

現有教案共享散佈於各個社群（例如：議題性教案共享網站、愛學網），缺點：普遍無審查機制品質參差、陳舊資料未更新

老師認為教案共享網站的重要之處：格式統一、不用下載才能瀏覽、簡案即可、教案下面可陳列推薦素材
