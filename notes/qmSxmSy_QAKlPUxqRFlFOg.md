# Public Money Public Code, and Open API in Taiwan

### 為什麼用人民納稅錢所建立的軟體與資訊系統，卻沒有以自由軟體授權公開釋出呢？並推廣OpenAPI

### 提案人 Proposor: Noah @ g0v summit 

### 摘要 Abstract (max 100 words):

為什麼用人民納稅錢所建立的軟體與資訊系統，卻沒有以自由軟體授權公開釋出呢？歐洲自由軟體基金會 (FSFE) 發起的 Public Money Public Code 議題，其中表現了 FSFE 希望人民可以在各個國家透過立法單位去要求政府產生的軟體服務或架構，其授權改用自由軟體授權方式釋出。另外也想再次推廣 AU 提倡的 OpenAPI（共通性資料存取應用程式介面）。
