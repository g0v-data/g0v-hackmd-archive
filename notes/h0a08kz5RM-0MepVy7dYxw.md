#  公民科技專案分享茶會 Civic Tech Sharing Meetup 
## 活動資訊 Event Info 
- 緣起：2024年11月新加坡科技設計大學[李光耀創新城市中心](https://lkycic.sutd.edu.sg) 會派團來台灣大學 D-School 參訪。該團希望了解台灣在公民科技與公民參與上的發展，預定在 11/16(週六)希望舉辦交流活動。經過討論後，認為可以邀請台灣的公民科技專案進行分享。
    - 單位介紹：
        - 李光耀創新城市中心是以都市治理為研究重點研究中心。
        - D-School則是台大的功能型學院，作為台大測試各種教學創新的實驗沙盒。
    - 參訪源起：創新城市中心的研究院Harvey Neo幾年前訪問D-School時，對其所串聯的社會議題，以及教學創新留下深刻印象。因此本次在規劃訪台行程時，特別聯繫黃書緯助教教授就都市環境、高齡社會、地方創生、青年參與等新加坡也正在面對的都市議題安排行程。期望能藉著參訪、交流、工作坊，的活動設計，了解彼此正在嘗試的解方、過程中的兩難，以作為日後持續合作的起點。
