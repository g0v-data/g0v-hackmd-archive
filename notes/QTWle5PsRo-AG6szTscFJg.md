# KKTIX 行前通知

## 10/18 活動前通知

All you need to know about The 10th g0v Birthday Party! (Please scroll down for English version below) 

「零宇宙大爆炸 - g0v 十週年生日趴」即將在本週日 (10/23) 登場囉！ 

本活動報名踴躍，目前已無票卷。如果您因故不克參加，請務必到 KKTIX 系統取消您的報名，把機會讓給其他想參加的人。（到 KKTIX 系統 > 我的票券 > 報名資料 > 取消訂單。特別注意：因購票系統設定緣故，若您報名兩場以上活動，取消單場次活動報名，將會同步取消其他場次票券。建議轉讓給其他朋友參與。） 
　 
➤ 本次活動官網與議程 https://10th.g0v.tw/

➤ 特別提醒事項 
1. 本次活動提供 Buffet 餐點（午餐及下午茶），請自備環保餐具領用食物。
2. 因應防疫工作，進入會場前將量測體溫，場地將提供酒精消毒。除用餐飲水外，請參與者全程配戴口罩。
3. 因應疫情變化，請考量自身狀況以及接觸史，若身體不適請在家休息。主舞台活動將在 g0v YouTube 上直播：https://www.youtube.com/c/g0vTW 
4. 參與者可領取十週年紀念品，數量有限，領完為止！
5. 15歲(含)以下不需報名，可自由入場。活動設計有親子同樂的#超新星遊樂園，歡迎大小朋友一起來參與！

【場地時間】 
　- 時間：2022/10/23（日） 11:00 - 17:00（11:00 開始報到，11:30 開幕式） 
　- 地點：社會創新實驗中心
