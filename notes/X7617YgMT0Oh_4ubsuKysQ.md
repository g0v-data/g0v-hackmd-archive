# 【社課一   三色螺絲】 

**名稱：** 三色螺絲 

**效果：** 觀眾選到的卡片對應顏色的螺絲轉開 

**關鍵秘密：** 
1. 記號牌 牌背亮星：一顆->紅、兩顆->黃、三顆->綠 
2. 物理原理 

**流程：** 

1.  展示三張不同顏色的卡片 

2.  請觀眾洗亂並抽一張，記好顏色之後拿回卡片，三張疊在一起放桌上，觀眾選的放最上面 

3.  請觀眾檢查螺絲及罐子，並轉緊螺絲，趁觀眾在檢查及轉緊螺絲時，看第一張記號牌 

4.  二次確認觀眾有無把螺絲轉緊(螺帽朝向右邊，右手一前一後)，已知觀眾記的顏色，那個顏色的螺絲就要第二個放進去罐子裡，把觀眾顏色的螺絲偷偷弄鬆(一前一後檢查偷偷弄送)，三個顏色檢查的動作看起來要一致 

5. 將三個螺絲螺帽朝下放入瓶子(觀眾所選顏色的螺絲第二個放) 

6. 請觀眾盯著瓶子，並專心想著選到的顏色，此時魔術師一邊將瓶子順時針水平的轉 

7. 中途可停下來詢問觀眾是否有專心在想，同時偷偷確認是否螺帽已經轉出來了(若已轉出則倒出罐子內螺絲，若尚未轉出則開玩笑的調侃觀眾沒專心想) 

8. 倒出罐子內螺絲後，接著打開觀眾所選的那張卡片，魔術完成 

# 注意事項：

1.  三張卡片在觀眾記好顏色後疊在一起，避免觀眾對比 卡片背面發現記號 

2.  觀眾顏色的螺絲第二個放，比較不容易被發現 

3.  螺帽朝下放入罐子 

4.  問觀眾是否專心並停下來偷偷確認，直到轉開為止，沒轉開請他繼續想
