# #42【島島阿學】以自主學習資源平台推廣民主教育

:::warning
**g0v 零時小學校數位公民課程**
[🏫 首頁 Homepage](https://g0v.hackmd.io/@jothon/B1NIQnmXs)
:::

### ▪️ 課程簡介

身處民主社會，教育的民主化會是什麼樣貌？公民課？學生自治？除了校園民主，學習又該如何民主化？此堂課程將會藉由各自的實際經驗一同覺察台灣教育現況，並反思民主教育的意義與實踐的各種可能性。課堂也將分享島島阿學學習社群中跨齡跨域的夥伴，如何覺察問題並運用數位協作的方式，設計多元學習資源平台，進而解決民主教育的問題。
