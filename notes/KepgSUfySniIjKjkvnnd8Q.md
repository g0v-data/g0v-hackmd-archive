# 政府OID歷史記錄重整

> [點此觀看原始內容](https://g0v.hackpad.tw/kznV059A6XV)


政府常不定期會有組織改組，從改名（2012/2/6 行政院主計處 => 行政院主計總處）、升格（2013/7/23 衛生署 => 衛生福利部）、降級（2013/1/1 行政育體育委員會 => 教育部體育署）。往往會造成在處理跨年的資料時需要人工額外處理，這個 pad 想記錄的是哪邊有相關資源，以便未來可以將這些資訊自動化處理
