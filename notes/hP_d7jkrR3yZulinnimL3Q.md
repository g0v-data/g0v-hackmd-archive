# g0v 社群活動＆專案宣傳指南

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_a07767e1ec94866780b00a8ba4ec66e8.png)

## 總之先用 g0v SNS 宣傳看看吧！

> 詳細內容可點擊 [g0v SNS 社群治理機制](https://g0v.hackmd.io/@chihao/Hkm9tPaxr)

#### g0v 官方臉書
* 除了發布貼文之外，也可以建立活動
* 發完貼文之後，可以分享貼文／活動到後勤中心社團
#### g0v 官方 IG
* 發布限時動態
#### g0v 官方 twitter
* 有權限的 editor 比較少一點

:::warning
**發文流程**
1. 將欲發文章依照格式貼入[編輯台](https://docs.google.com/document/d/1e16R3Issef1qk0eyb3NyMN_4CLYt2kmq_Mt-ElWlVf4/edit#)
2. 於 g0v slack #sns 發文 ，貼上文章連結，並 tag reviewer 2-3 位（24 小時內文章上線需要 3 位）
> Reviewer 請用[抽籤器](https://ronny.tw/g0v-sns/)抽出後，前 2-3 位即是該篇文章的 reviewer，請欲發文者 tag reviewer 
3. 確認被抽中的 reviewer 看過無誤後，可以 tag 有發文權限的夥伴幫忙貼上 FB 設定排程
4. 到[編輯紀錄](https://docs.google.com/spreadsheets/d/1xyxYGKU7iia3xSu6rF9F6YCcXOcEqrFsI2RD493bAxE/edit#gid=1776966921)更新本次的 review 狀況
:::

#### g0v Slack
* 發布到 #general 或各個頻道

#### g0v Podcast
如果想要錄製長一點的內容介紹自己的專案，也可以考慮 g0v Podcast ：
- [零時電台 book](https://g0v.hackmd.io/@podcast/book)

## 也有一些由揪松團管理的 SNS 帳號

#### 零時小學校官方 LINE
* 主動通知：一個月只有 200 則群發訊息扣打，一次可發三則訊息
* 被動通知：近期活動自動回覆訊息

#### LINE 社群
* g0v 揪松團 LINE 社群（[加入連結](http://bit.ly/jothon-LINEg0v)）
    * 可以在加入後直接發文
* g0v 之友／定期定額 LINE 社群
    * 專為定期定額捐款者設立的 LINE 社群，不對外開放


## 發布到 g0v 日曆，讓所有追蹤者看見你的活動

#### g0v 日曆（刊登需走 [g0v 首頁社群治理機制](https://g0v.hackmd.io/@ddio-io/rkeaLlOxJn#%E8%A1%8C%E4%BA%8B%E6%9B%86)）
https://g0v.tw/intl/zh-TW/event/ 

:::success
到 #g0v-landing-page 刊登申請，然後 tag Reviewer

【活動內容格式舉例】
．活動時段：聚會想要改成從 Feb 8 開始，每個月（每個月的第二個週三）一次
．活動標題：OH!SHOWN - 黑熊通報系統定期小聚
．內容欄位：專案共筆 Meeting note https://lihi1.cc/VDemi （聚會線上連結在共筆內）
:::

## 其他宣傳管道

#### 各大學的院辦、系辦或資訊/倡議社團信箱
* [揪松活動宣傳聯絡清單](https://docs.google.com/spreadsheets/d/1ahH_MGqev6_VGHhhpxDwIcPJUeX_KVeD/edit#gid=968181522) >目前鎖權限中，歡迎詢問揪松團


#### 資訊社群 Discord
* SCINT｜北臺灣學生資訊社群
  * 直接在「活動分享」發文即可
* 中學資訊討論群｜CISC
  * 需到「#ticket」點 create ticket，系統會自動開設一個私密頻道，直接在該私頻貼宣傳文案＆圖＆預計發文的時間，會有人審核貼文並協助發文（該頻道類似 g0v 的 #sns。審核的人有 Ted XD）
