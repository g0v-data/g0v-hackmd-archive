# 沒有人零時資訊行動看板

> [點此觀看原始內容](https://g0v.hackpad.tw/NEbUwH8tg4N)

本文件為離線資訊傳播線上化行動方案，歡迎補完

### 源起

2016/6/24 pipi 於華航罷工現場，以qrcode傳播 billy3321 準備之懶人包

![](https://g0vhackmd.blob.core.windows.net/g0v-hackmd-images/upload_6a31b8faaf2e2afa652542d9c9dc2da2)

### 行動步驟

1.  發現目標議題懶人包，想到運動現場衝鋒，解決資訊落差問題。
2.  使用 [http://bit.ly](http://bit.ly) 建立縮址自訂成 [http://j.mp/罷](http://j.mp/罷) 或 [http://bit.ly/罷](http://bit.ly/罷)
3.  將網址貼到 [http://qr.ioi.tw/zh-TW/](http://qr.ioi.tw/zh-TW/) 即可取得"最簡化"二維條碼
4.  貼在身上，到運動現場傳播!
