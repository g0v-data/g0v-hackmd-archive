## [COVID-19] Stop coronavirus hackathon, Japan
> [name=發表者 Hal Seki]

因為面海松而結識的社群夥伴 Code for Japan，在日本同步舉辦一天的黑客松，並來大松登記提案。肺炎疫情擴散全球的時刻，日本的科技公民也不落人後，建立出「東京都武漢肺炎資訊網站」，有大量的源碼、資料庫，在 Github 上開源使用，一周內不停加入的貢獻者解決了 400 個 isuue，目前也有日本各地遍地開花的網站版本，希望有更多人可以使用這些資源，歡迎大家隨時加入解任務！

提案影片：http://bit.ly/38thjpcovid
共筆：https://hackmd.io/@codeforjapan/rk9KbKerI

## [COVID-19] 各國疫情邊境控制可視化 react widget
> [name=發表者 Vincent]

Code for HK 的 Vincent 建立了一個網站，有各國肺炎疫情的資料及數據彙整。本次的提案是從這個網站繼續延伸，目前各國開始管制非本國人入境的情形，然而，這些資訊比較複雜且混亂，因此想將邊境資料做整理，並且改善前端 UI 的呈現，讓一般使用者能一目瞭然目前各國出入境的管制狀況。此外，也期待可以翻譯成多國語言，供更多人使用。

提案影片：http://bit.ly/38thhkcovid
共筆：https://hackmd.io/@JNc71p-sSAmkvOYsyZlLbg/HyX1UgrrI

## [COVID-19] 0archive 零時檔案局 - 武漢肺炎資料分析
> [name=發表者 Chihao]

0archive 專案正在將網路上不同的資料來源（新聞網站、內容農場），從源頭蒐集進專案的資料庫，再將這些資料開源。這些資料可以進行來源分析，例如 A 內容農場的資料來源 30 % 來自特定媒體；或是某篇文章的資訊傳播途徑。黑客松當日的提案希望可以針對資料含有「武漢」的內容來分析、觀察，若對於專案本身有興趣，可以從專案共筆找到更多的相關資訊。

## Cofacts 真的假的
> [name=發表者 bil]

通訊軟體 LINE 總是被轉傳一些無法辨別真假卻聳動的文章或訊息，吃水果會致癌、加好友拿免費貼圖等等族繁不及備載，Cofacts 真的假的闢謠機器人 LINE bot，歡迎不會寫程式的朋友，一起加入事實查核編輯的團隊，一起針對這些文章進行事實查核（fact check）。也募集工程師加入一起解 issue，改善網站和機器人的操作功能！

## 臺灣好植地 Patch by Planting 
> [name=發表者 chewei]


這個專案是用 GIS 圖層來找出台灣甚至世界各地適合／可以種樹的地方，增加土地上植被的面積，也希望可以結合不同關心種樹議題的 NGO 一起倡議，並於今年總統盃黑客松提案，希望實際與政府合作，與土地管理的主管機關深入討論。目前已經整理出一批分析資料，從分析的資料就可以想像、規劃，增加的植物該如何生長和布置，期待有更多關注此議題的朋友成為專案的一份子。

## 歷史的刻痕--轉型正義資料庫 wikidata化計畫
> [name=發表者 Allen]

Wikidata（維基百科）將資料用結構化的方式，收存在資料庫中，而最近轉型正義資料庫上線了，內容涵蓋所有當時的相關人，這些資料可能只有針對人物簡單敘述，但其實民間有很多散落且更豐富的資料，希望可以透過兩相彙整，將同一人物的背景，整理在維基百科上，未來除了資料查詢的功能，也可供研究者分析使用。
