## 7. 技術需求與功能調整
- **Mentimeter** 必須支援「意見編輯+審核」功能（admin 可微調文字、語序）。  
- **逐字稿自動化**：利用 AI API 把語音即時轉文字，並自動生成摘要與關鍵詞。  
- **多人麥克風**：為避免單一麥克風無法對應說話者，考慮在現場設置 2‑3 個收音點，或允許參與者自行上傳語音檔。  
- **權限管理**：使用 email 方式快速授權編輯者（避免 Zero‑Auth 複雜流程），兼顧資料安全。  
- **工具說明頁**：需加入合作團隊（VT1、vTaiwan）與貢獻者的 Credit，並同步至 GitHub README。
