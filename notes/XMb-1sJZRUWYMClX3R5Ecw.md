# 2017 開放政治獻金再啟動

> [點此觀看原始內容](https://g0v.hackpad.tw/8oBrDIGk5he)


鏡傳媒最近發起了專題報導 [【數讀政治獻金】政治獻金開放了嗎？](https://www.mirrormedia.mg/story/political-contribution004/ ) 並且開放了一批去監察院印出來並數位化的資料。之後又再印了一批資料出來，希望透過與開放政治獻金專案合作，讓大家再一起把更多政治獻金帳戶數位化
