# Python Math
## Built-in Math Functions
* min( ) and max( ) 最小值及最大值
```
x = min(5, 10, 25)
y = max(5, 10, 25)
print(x)
print(y)
5
25
```
* abs( ) 絕對值
```
x = abs(-7.25)
print(x)
7.25
```
* pow(x, y) 次方
```
x=pow(5,2)
print(x)
25
```
