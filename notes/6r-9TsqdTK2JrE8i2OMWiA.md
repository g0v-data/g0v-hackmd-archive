# 公有地大行動－參與啥米松筆記

> [點此觀看原始內容](https://g0v.hackpad.tw/YOEzWb7eFqs)


11/08 擺攤紀錄 / Lighting talk
逛攤者意見回饋：
1.  有一位任職於地籍資料系統服務的朋友，相互聊的結語，大致上是應回到對於公有土地地號與全體國民之間的關係的看法來作討論，透過這個層面的確立，來促使相關局處的資料適當開放釋出。
2.  也有朋友詢問到，開發、規劃，不同權責之間的關係。
3.  也有的朋友，在瀏覽天龍特公地網站的時候，可以說出畫面中某一塊公有地曾是他們家族的土地，經過徵收調整等等過程的身世。
4.  歸納今日介紹的方法
    1.  先介紹「天龍特公地」，說明這批不會再更新的研究資料，共有一萬四千筆，稍具有普遍分布，所以我們先來運作看看、試試看當手上有這樣的大批公有土地資料時，我們期待如何應用、串聯、使用，例如將地號畫在地圖上、比對都更地號、讓民眾上傳照片。
    2.  另外，持續詢問相關公部門的資料釋出、開放，包含段小段地號與地籍資料，蒐集癥結課題。
