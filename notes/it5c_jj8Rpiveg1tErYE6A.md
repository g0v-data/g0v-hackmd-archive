# shoutout-關於在da0官網放上貢獻者名單的討論

## 2023/8/8 Slack頻道#shoutout
* [name=n0ah] 晚上 6:40
有個點子跟大家分享，也許能推進一下 SO 也能稍微配合最近社群碰到的問題 @Lucky@Matt@tofus@昶惟@Peixing

有幾個元素
* 在 da0 網站上公開多放置一頁叫 List of prominent contributors
* 只要有被 SO 過（contributed in a way that’s meaningful to someone）就會出現在名單上
* 若是違反 CoC，則會根據某種程序（未定），被從名單中移除

幾個注意的點是
* 核心是希望創造社群對 da0 的 ownership，也給成員一定程度但不影響他人的代表性，this is different from g0v
* 這不是 da0 members 名單，只是列出有貢獻的社群成員，這不是邊界，只是對外界社群正向參與度的參考
* 網頁同時會寫出被列進名單的原因，及社群設定的 CoC

優勢有幾個
* SO 有了確切的使用情境，望能增加使用率
* 被認為是「Prominent contributor」門檻不高，讓參與容易，是這個階段增加參與的設計
* CoC 與 list offboard process (to be designed) 可守住社群底線

名單的使用情境可以對應
* 對外演講，SO 越多的代表參與越多，分享也更有說服力
* 對外談合作，SO 越多的代表跟社群合作關係越緊密，也更有說服力

* [name=維人]  晚上 9:31
我先寫排序公式跟解說。目前提到的排序方法有兩個*
方法1：
@Peixing 的假設
妞妞認為只要鼓勵大家不斷使用SO，根據活躍度來排名即可
我根據這個概念列出公式。
將SO視為有半衰期且會不斷增加的物質。
每一段時間(每天？每兩天？)統計一次收發SO的次數
即可套用半衰期公式計算活躍性：
N=N0*e^(-l)+N1
其中
N=用來排名的總合值
N0=統計前的總合值
N1=這一段時間的SO數量
l=衰變常數，就是"經過幾個週期之後，過去累積的SO意義大幅降低"。視情況調整
e=自然對數
根據此公式，同步統計發送與收到的SO，可得活躍度總公式為：
R=R0*e^(-l)+R1
S=S0*e^(-l)+S1
N=R+kS
k=鼓勵使用SO感謝人的參數。k<1
R表示收到
S表示送出 (已編輯) 
:deployparrot:3


* [name=維人]  晚上 9:37
方法2：我的無腦膝反射。平方投票法
將每一則SO視為給對方的認可。根據平方投票法，對於每個使用者，與n位使用者有SO互動後，總得分為
sigma(1->n) Ri^0.5 + k*Si^0.5
其中
Ri = 你從第i位使用者收到的SO數量
Si = 你發給第i位使用者的SO數量
k = 鼓勵使用SO感謝人的參數，k<1。
兩種方法的k意義相同，都視使用狀況來迭代調整 (已編輯)
