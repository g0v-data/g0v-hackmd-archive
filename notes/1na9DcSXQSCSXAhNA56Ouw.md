# iHelp-android

> [點此觀看原始內容](https://g0v.hackpad.tw/ulv07iXiTkb)

    專案介紹 | Project Readme / Meta
    現今瘖啞人士礙於無法發聲，因此無法使用現有『１１０報案』專線，僅能夠透過簡訊、傳真來通知友人協助報案，但並不是每次都能順利連絡到人，許多人因此錯失了黃金救援時間造成不幸。

    本專案為瘖啞人士這類弱勢族群解決報案方面的困擾，提供GPS報案定位並能夠透過簡訊及網路來進行報案，在報案過成功能夠選擇案發現場狀況來派出相對應的救援
