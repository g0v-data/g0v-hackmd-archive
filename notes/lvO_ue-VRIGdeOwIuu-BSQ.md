## 重點整理（正體中文）

### 一、活動概況
| 項目 | 內容 |
|------|------|
| **活動名稱** | 面海松（Hackathon） |
| **時間** | 10 月上週末（預計） |
| **地點** | 高雄面海松活動場地 |
| **目標** | 讓參與者實作 **SenseMaker** 的 Web UI，體驗 AI‑輔助的意見審議流程；同時收集「AI 在社群媒體言論管制」的政策建議。 |
| **預期人數** | 300‑500 人（最終可望達 1 000 人） |
| **受眾** | 台日韓公民科技工作者、政府官員、國際組織、NGO、一般民眾、障礙者等多元族群。 |

---

### 二、核心技術與工具

| 工具 / 平台 | 用途 | 重要說明 |
|------------|------|----------|
| **SenseMaker Web UI** | 1️⃣ 匯入意見資料（JSON/CSV）<br>2️⃣ 呼叫 OpenRouter（GPT‑OSS‑120b）產出 Markdown 報告<br>3️⃣ 直接在前端完成 AI 分類、主題/子題抽取 | - 需事先取得 **API Key**、點數（約 $5 美金可跑 30 次）。<br>- 前端不會儲存金鑰，安全性依隱私政策說明。 |
| **Pol.is / police.tw** | 既有意見資料來源（CSV / JSON） | 例：先前針對生成式 AI 的意見調查，可直接匯入 SenseMaker。 |
| **Agora Citizen** | 另類意見收集平台（實名認證、投票） | - 可匯出原始資料供 SenseMaker 再分析。<br>- 目前缺少防止匿名/機器人濫用的機制。 |
| **JSONLint** | 檢查匯入的 JSON 是否符合格式 | 匯入失敗時，可先用此工具驗證。 |
| **Cloudflare (sensemakerfrontendpages.dev)** | 部署前端服務，提供即時測試環境 | 參與者只要開瀏覽器即可使用。 |
| **Slack / Line** | 活動即時支援與通報 | Bestian 會在 Slack 中提供即時協助。 |

---

### 三、資料準備與流程

1. **API 金鑰及點數購買**  
   - 取得 OpenRouter 的 GPT‑OSS‑120b 金鑰，預留足夠點數供 30‑50 次測試。  

2. **資料來源**  
   - 兩套「議題小區」的主制稿（去年、今年）  
   - 先前「Police」平台匯出的意見 JSON（PoliceReport.json）  
   - 可能的 Agora Citizen 匯出 CSV（待開發完成）  

3. **資料前處理**  
   - 用瀏覽器「開發者模式」複製 JSON，貼到 `PoliceReport.json`；  
   - 用 **JSONLint** 確認結構正確；  
   - 若資料量太大，可分批上傳。  

4. **使用 SenseMaker**  
   - 輸入 API Key → 選擇模型 → 上傳 JSON/CSV → 設定語系（中文） → 點擊「產生報告」  
   - 系統自動完成：「主題」&「子題」分類、共同點/分歧點分析、Markdown 報告產出。  

5. **結果呈現**  
   - 生成的 Markdown 可直接在 Slack、GitHub、或 Agora Citizen 上分享。  
   - 若需要深入分析，可把 SenseMaker 的結果再匯入其他 AI 分析工具（例：自建的文字探勘模型）。

### 五、策略與決策要點

| 主題 | 觀點/決策 |
|------|-----------|
| **使用哪個平台收集意見** | - **Agora Citizen**：實名制較安全，適合正式政策意見；但需等待資料匯出功能完成。<br>- **Pol.is**：即時匯出 CSV，已能直接接入 SenseMaker；但防止匿名濫用較弱。 |
| **門檻與參與度** | - 允許「直接投票」的低門檻可提升參與度（尤其是年輕族群），但需做好濫用偵測。<br>- 若以「認證」為前提，參與者會較少但資料品質較高。 |
| **AI 工具的角色** | - AI 主要用於 **快速分類與報告生成**，不取代人類討論；避免過度依賴模型造成意見扭曲。<br>- 針對障礙者，AI（如手語氣象主播）可提升資訊無障礙性。 |
| **資金與成本** | - OpenRouter 點數成本低（≈ $5/30 次），可預算在活動經費內。<br>- 若要大規模（1 000 份意見）建議提前購買足夠點數，避免現場中斷。 |
| **後續持續性** | - 以 **NGO‑政府合作** 為主軸，建構「AI‑審議」的生態鏈（NGO → SenseMaker → 政府部會）。<br>- 透過報告交付與媒體曝光，爭取未來的政府資助或公私合夥。 |
