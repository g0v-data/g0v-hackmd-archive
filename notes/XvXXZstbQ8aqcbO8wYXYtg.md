# g0v Summit 2024 公務人員學習時數登錄

為了鼓勵公務人員參加 g0v Summit 2024，大會設置「公務人員學習時數登錄」機制，請公務人員朋友踴躍利用！

:::warning
一天要登錄一次，才會算當天的時數哦！
:::

## 給場務組

1. 引導公務人員至大會報到處「公務人員學習時數登錄」
2. 掃描 QR code
3. 在表單輸入姓名（本人完整姓名）、身分證字號
4. 送出後，系統就會自動記錄你的登錄時間
5. 資料彙整後，會送交 [...] 完成時數登錄

> 怎麼完成時數登錄待確認 [name=知澔]

## 需要準備

- [ ] 「公務人員學習時數登錄」立牌或手板 [name=場務組]
    - 已寫入指標系統[name=CPY]
- [ ] QR code 電子檔 [name=知澔]
- [ ] QR code 列印紙本數張 [name=場務組]
    - 已寫入指標系統[name=CPY]
- [x] Google 表單本人 [name=知澔]
    - 表單連結 https://forms.gle/AF44TCABHNN3zhr57
    - 為保護個人資料，表單開在 g0v Summit 2024 工地 > 合作伙伴 > 僅限合作伙伴組

## 資料處理

- 由合作伙伴組知澔負責

## 個人資料保護聲明

- 透過「公務人員學習時數登錄」表單蒐集的個人資料，只會用於公務人員學習時數登錄，絕不會移作他用，也不會交予時數登錄無關的第三者。
