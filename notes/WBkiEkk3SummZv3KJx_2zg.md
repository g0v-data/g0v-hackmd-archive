# 門牌規則整理

> [點此觀看原始內容](https://g0v.hackpad.tw/xTdfXNEsrJC)


從 [https://hackpad.com/SBCWxR9PLfp](https://hackpad.com/SBCWxR9PLfp) 移過來的，上次編輯時間 2016/2/3

這邊整理一下台灣的門牌有哪些規則，在解析時可能需要注意

- 欄位有 省、縣市、鄉鎮市區、村里、鄰、路、段、巷、弄、衖、衕、號、樓
    - 省只會有臺灣省和福建省 (現在常常省略不寫)
    - 當有路的情況下，鄉鎮市區、村里可以不給(一個縣市同一名稱之會有一條路)，但有的情況沒有路名，那鄉鎮村里不能省略
    - 鄰可省略不給
- 幾種地址狀態
    1.  沒有路名，只有號(金門馬祖為主，除此以外只有苗栗有一個)
        - 範例
            - 連江縣東引鄉樂華村6鄰153號
            - 苗栗縣公館鄉大坑村8鄰197之22號
        - 這種時候唯一識別為 \[縣市、鄉鎮、村里、鄰、號\]
            - 鄰也應該要加入唯一 key ，連江縣有 7 對門牌是號都一樣只有鄰不一樣
                - Ex: 連江縣南竿鄉介壽村 4鄰102之1號 5鄰102之1號 是不同地方
- return code
    - 1 - 完全吻合，毫無懸念
    - 2 - 完全吻合的地址有兩個但是很接近，
