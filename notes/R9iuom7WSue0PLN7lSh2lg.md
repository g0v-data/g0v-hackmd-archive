# #57 群眾外包：以 Disfactory 農地違章工廠回報為例

### ▪️ 課程簡介

這堂課將先介紹群眾標註的概念與案例，以及運用群眾標註時需注意的狀況及因應方式，接著再帶到台灣案例。台灣案例由長期關注違章工廠議題的NGO－地球公民基金會，說明違章工廠在台灣的現況，以及制度面遇到的阻力，並延伸到地球公民基金會如何藉由與g0v社群合作，借用科技與群眾的力量，以群眾標註等方式，打造一個多方協力的合作模式，來促成制度的改變。

### ▪️ 學習目標
1. 能了解群眾標註的概念、案例與常見狀況
2. 能了解台灣違章工廠治理現況，有哪些制度，以及公民團體努力的標的
3. 能了解在選擇將問題「群眾外包」時，要思考哪些面向的問題
4. 能嘗試自己去想像生活中遇到的哪些問題適合以群眾外包來處理，又或是為什麼不適合

### ▪️ 課程延伸思考問題
請選擇一個你在日常生活中特別關注的、具有公共性的問題。請說明你如何理解這個問題，並探討這個問題是否適合透過群眾外包來解決或改善現狀。如果不適合，請說明你的想法；如果這個問題有群眾外包以外更好的處理方法，也可以提出。
(舉例而言：路燈壞掉、生活圈下班交通壅塞.......)

1. 除了群眾外包，還有哪些可能的解決方式？
2. 有沒有既有的解決方式？既有的解決方式有什麼問題？這些解決方式有可能做一些調整或改善嗎？
3. 這個問題會影響到誰？誰應該來處理？牽涉到哪些利害關係人？
4. 如何創造群眾參與的誘因？
