# 我在抗議!!

> [點此觀看原始內容](https://g0v.hackpad.tw/LSrK6jRQcI7)


## 起源

    因為看到社會上發生很多不公平的事情可能太多人又加上台灣人民好傻好天真，容易淡忘，所以希望有個平台可以記錄社會上所發生大大小小的事情，並讓使用者可以自己選擇要參加怎麼樣的抗議，抗議的形式非常的多元，從活動到自己行動上的抵制都能記錄，但為了避免有人參與抗議之後就不管了，所以固定周期(可能1個月)就會詢問使用者是否要持續的參與。
    也可以讓使用者們追蹤不公平事件的後續發展，並記錄下是哪個集團集所有利益關係人，讓我們可以更知道並記錄台灣所發生的事情，而不會讓那些集團掛勾並且被抓包之後就想不了了之。



## 母題

- 議題與社會運動的結構
    - 當事人

        - 受害方與問題方的角色
        - 受害的程度
    - 支援群體或奧援群體
        - 社會運動團體
        - 政治立場者
        - 同情者
        - ~已經封神的~意見領袖
    - 解決問題的方法
        - 散佈議題
        - 制度解決
        - 抗爭行動
- 散佈議題的手段
    - 擺攤，擺文宣品
    - facebook粉絲頁
    - 講座或展覽
    - 組織社運團體，舉辦營隊
    - 記者會
- 議題與社會運動的結果
    - 新聞或負面新聞
    - 法律代價
    - 表面方案
    - 實質方案
    - 無結果
    - 照片、影片、心情日記
    - 同儕、~異性~朋友
- 如何捲動並拉升一般人對於社會議題，平均程度的關心？
    - 刺激情緒的事件
    - 拉人：擴大參與
    - 口碑引起支持度

## 結構

- 議題
    - （定義）由許多事件彼此關聯的因果脈絡。
- 使用者
    - （素人）「我想知道某某議題是什麼。」
    - （熟知者）「我想知道某某議題最近怎麼樣了。」
    - （社會運動學者）「我想知道某某運動，有哪些參與者加入然後又退出。」
- 世界時間軸：
        每個事件所組成的時間軸
- 單一事件:
    - 參與者
        - 當事人
        - 聲援者
        - 媒體
    - 參與者之間的關係圖
    - ~與該事件有關的其他事件(日期)~
    - 事件發生的日期
    - 地點
    - 詳細事件內容
    - 相關~連結~新聞
    - 相關討論（含外部網頁或blog）
    - 相關議題
    - 相似的事件
    - ~相簿/影片~
    - 對該事件的討論(留言板)
    - ~我在抗議~簡要的陳述
        - 抗議的名稱(EX:拒絕購買林鳳營公司相關產品)
        - 正在參與人(可連結FB看到有哪些朋友正在參加)
    - 事件的產出
        - 訴求
        - 正反面的影響
        - 數位檔案紀錄（含照片與影片）
- 大事記
    - （定義）許多單一事件彼此拉出關係之後，所構成的複雜脈絡。
    - （呈現）以時間為基礎的清單。
    - （呈現）以分群為基礎的 layout 。（資訊地圖、資訊圖表）
    - （呈現）由所包含事件的影響，所構成的變化圖。
    - （呈現）參與者的變化圖。

## 現有案例

### facebook

- 粉絲頁，主控抗議平台，議題散佈，公佈活動
- 朋友圈，散佈議題，再製議題，拉人
- 抗議方的操作力強
- 可以仿冒平台
- 議題的成長健全
- 保存短期事件的熱度，不容易翻閱舊資料，但容易炒熱事件
### 獨立媒體

- 例如[4am.tw](https://4am.tw)、獨立媒體、風傳媒
- 中立的平台
- 有自己的取向
- 議題需要經過審查才能發佈，有基礎的公信力
### PTT

- 議題及群眾吸附力強
- 群眾參與程度差異高
- 需從民粹意見中漸漸培養議題
- 謠言多，方便於傳遞小道消息
### 社運專屬集資平台

- 例如[VDemocracy.tw](https://www.vdemocracy.tw/)，[flyingV.cc](https://flyingV.cc)
- 需揭露集資資訊的責任
- 受政府財經部門限制，例如[flyingV](https://flyingV.cc)受到限制需自我審查
[Protest Net](http://www.protest.net)
- 提供公開發佈抗議活動的管道
- 有電子報

## 實作

- 系統
- 工具
    - 資訊圖表產生器
        - （發想）因為在 facebook 或 twitter 的時代，資訊圖表 (infographic) 是很流行的媒體格式。
        - （定義）資訊圖表是在有限的圖片尺寸中，以圖片元素的編排，視覺化呈現出豐富的數據或相對數據關係。
    - 字卡產生器
        - （發想）字卡在 facebook 也很流行，藉由一句有梗的句子，或一句極短的笑話，可以達到宣傳效果。
        - （定義）字卡是一種退化的資訊圖表，主要用在呈現簡短但有效的訊息。
        - 用類似 markdown 的特定符號及文字，自動產生字卡。例如，輸入 "\[捍衛民主\]" 可以得到一張 300x200 黑底白字 gif 圖片，上面寫著「捍衛民主」。
