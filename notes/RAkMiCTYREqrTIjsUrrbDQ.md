# 2025-09-23 <br>花蓮馬太鞍溪堰塞湖溢流-災後清理工作 <br>
:::info
1. **盡量避免個人前往**，請與有組織的單位報名，或與當地取得聯繫再行前往。
2. **審慎評估自行能力與專業程度再前往**，災區需基本的自我照顧能力，照顧好自己才能照顧別人。
3. 災區道路受損狀況嚴重，災後大型機具、救災車輛湧入，**目前進光復的車輛已經擁塞，如欲前往者，盡可能搭乘火車前往**。
4. **保護自身安全，尊重當地的需求**，垃圾自己帶走。
5. **請不要帶太多個人隨身行李**，基地沒地方放。
:::


### 心理準備：
   * 大量志工短期湧入，可能不會有太多準備好的資訊與流程。
   * 現場發生摩擦與爭執，記得不要咎責，要當責。
   * 可能遇到動物屍體，味道會比較重。

* 基本穿著
    * 不怕髒、快乾材質的衣服、長褲（泥水開始腐臭，易觸發過敏反應）
    * 雨衣、雨褲或防水衣物（視現場情況而定） 
    * 高筒的防滑雨鞋
        * 可放鞋墊+厚襪，提升舒適度
        * 塞褲管進雨鞋，防止小腿肚磨破皮 
    * 換洗衣物一套`（回程換穿、舒服休息）`
* 個人防護
    * 袖套、遮陽帽、頭巾`（防塵、防曬、防皮膚受傷）`
    * 防塵口罩、安全護目鏡（視現場情況而定） `（現場粉塵多）`
    * 防護手套（厚布或防割、止滑材質） `（泥巴混著玻璃、碎石、釘子等尖銳物）`
    * 頭燈、備用電池(至少三組)
    * 護腰（視個人需要）
    * 可以將自己包包整個包起來的大塑膠袋 `（防止包包弄髒）`
* 其他
    * 濕紙巾、個人餐具碗筷 `（減少在災區用水、使用一次性餐具）
    * 水與乾糧 `（照顧自己快速補給）`
    * 毛巾 `（擦汗）`
    * 個人藥物&醫藥防護包：優碘、酒精、OK繃、醫用膠帶、紗布、棉花棒、外用抗生素（軟膏或粉狀）、個人抗過敏藥物`（泥土內多尖銳物，割傷照護）`
    * 緩解肌肉疲勞的用品

### 清理工具：
* 方鍬／尖頭鏟 / 鋤頭 / 十字鎬／高壓水槍
`（很多泥巴乾了，圓鍬不好用，要先用十字鎬、方鍬、高壓水槍把泥巴弄鬆）` 
* 大型垃圾袋 / 集裝袋／ 布袋 `（裝泥土用）`
* 水桶／單輪泥水推車 `（倒泥巴、裝水）`
* 刮水器 `（趕水、趕爛泥，家具底部縫隙刮除汙泥）`
* 撬棍 ／鋸子（手鋸或鏈鋸）／鐵鎚 `（用於拆除工作）`
* 大型垃圾桶（臨時堆置廢棄物）
* 清潔劑 / 消毒水
* 拖把、掃帚、抹布（視需求補充）

### 大型機具支援（視實際狀況調派）：
    * 小型挖土機 / 推土機（用於處理大規模瓦礫或道路障礙）
    * 發電機（若當地電力尚未恢復）
