# g0v slack 版名改名草案 20220312

(1) 頻道內修改版名方式

頻道內的成員，可以提出該頻道版名修改的討論，並鼓勵該頻道成員提供意見，並且主動 tag 頻道「建立者」，因為建立者有版名修改權限。經過頻道成員們達成共識，即可由建立者修改版名。

頻道建立者的帳號名稱，可以在各頻道的簡介資訊欄中查看。


(2) 向 admin 提出修改協助

若無法聯繫頻道建立者，每 1 週連絡一次頻道建立者（例如於該頻道 tag 建立者），連續 3 次無回應，可申請由 slack admin 協助修改版名，但應提供「頻道內版名修改討論紀錄」供 admin 作為判斷依據。

請至 #g0v-slack 頻道，貼出頻道改名討論串訊息網址，讓 admin 了解有此需求。


(3) 修改後，請主動通知社群

頻道名稱修改後，請將修改結果內容（至少包含新的頻道名稱），張貼至 #general 頻道，讓社群參與者周知，以及讓「專案資料庫」的各類專案可以修改新的名稱。
