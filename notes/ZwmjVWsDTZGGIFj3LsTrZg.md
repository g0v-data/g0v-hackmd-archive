- [x] **DATA.GOV.TW**
政府資料開放平台，雖然目前有近13000筆資料，但絕大多數都是用不到 or 沒人用的垃圾
且資料上架與下架，都沒有相關資訊（LOG?），和社群連結也很少
1.  如何評比資料，判斷哪些是垃圾
2.  網站介面問題（類布告欄），搜尋功能不佳（關鍵字無法對應到想要的內容）
目標：
1.  從資料品質評比談資料內容格式化標準化
2.  政府網站內容結構化、格式化
3.  如何供社群下載（資料格式？）　Export to GitHub
    1.  資料可以更有效利用、闊充
    2.  政府單位可以從GitHub使用量、被關注量來判斷資料的被重視性

- [x] **_為什麼都市計畫要開放_**
目前都市規劃還是偏向獨裁
1.  僅有少數選項，如社子島開發、松山機場遷建
2.  和主政者任期有關，如桃園鐵路高架化／地下化、台中ＢＲＴ
都市規劃專業者的目標：消除都市規劃這項專業　→　幫助居民規劃自己的空間
- 都市計畫資料庫化（資料共享、減少資訊落差）

- [x] 公寓大廈管委會　與　都市更新
管委會之組織結構、工作
1.  日本有「區分所有法」及其他相關法規可以確保管委會可以推動社區規定、大樓維護與整建等
    1.  平時就有收管理費、維護修繕費
2.  在台灣大規模維護與整建等則被列在都市更新相關法規中（可向市政府申請經費補助，但大多數人不知道）
    1.  重建之流程規劃，管委會參與度低，使流程不佳
請大家多關心自己之住宅、社區、管委會

- [x] 健保核刪
1.  健保核刪是否該公開？審查機制到底有沒有問題？
2.  醫師建議你的選項到底是因為它是最佳方案，還是最不會被核刪？
3.  看病（醫院先墊錢）　→　醫院向健保申請補助（申報）　→　健保不買單、不付錢（核刪）
- 而且核刪後還有放大處罰機制
4.  傳聞　　每年有要求固定比例之核刪（即使是必要支出？）
5.  數字公開才能有討論，但目前完全沒公開，合不合理、被核刪風險都未知
6.  各醫生都有部分資料，透過身分審核機制、個人上傳，把資料聚集，讓資料公開
- 進一步使健保局願意公開資料
7.  如何讓健保永續經營？
