### 主題一：怦然心動的Code for Japan summit 2019
9月間一群g0v參與者前往日本千葉，參與 Code for Japan Summit 2019，會中與日本及美國的參加者有許多交流與分享，這些參與者將分享會日本官民合作案例、與日本社群交流的心得、對會議組織與形式的觀察等等。

### 主題二：以公民科技為專業？美國公民科技工作經驗分享

自 2014年 進入civic tech的世界後， Amy曾經先後擔任Code for America Fellow，以及Consumer Financial Protection Bureau中Technology and Innovation Fellow，目前則是擔任18F Consulting Engineer。
Amy 將分享在這三個不同的崗位，相同與不一樣的體會，希望與會者對專業的公民科技工作有更深的了解。

講者：
Amy Mok
18F Consulting Engineer, Previous: CFPB Technology and Innovation Fellow, 2014 Code for America Fellow
