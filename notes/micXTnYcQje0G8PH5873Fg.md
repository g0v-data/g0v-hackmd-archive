19:15-20:45 讓大家針對自己感興趣的專案個別帶開，可能是想法的深入討論或是單純多寫幾行程式碼
20:45-21:00 個別專案分享一下當天的記錄，最單純的就是秀 github commit log 去說做了些什麼，或是如果有些草圖也歡迎，希望內容都可以放進 hackmd 做個記錄
