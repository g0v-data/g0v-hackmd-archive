## 加入 g0v 黑客松


1.  **逛逛** **g0v，瞭解狀況**
    - g0v 到底是什麼呢？
        - [g0v 官方網站](http://g0v.tw/)
        - [facebook 專頁](https://www.facebook.com/g0v.tw)、[g+ 專頁](https://plus.google.com/u/0/102893633954450772289/about)
        - [緣起影片逐字稿](https://g0v.hackpad.com/-g0v-x1vIIhs38Bx)
    - 我想參與專案協作！[hackfoldr 協作資料夾](http://hack.g0v.tw/)
        - 有哪些進行中的專案可以參加？[http://](http://g0v.tw/zh-g0v.tw/zh-TW/projects.htmlTW/projects.html)[g0v.tw/zh-](http://g0v.tw/zh-g0v.tw/zh-TW/projects.htmlTW/projects.html)[g0v.tw/zh-TW/projects.html](http://g0v.tw/zh-g0v.tw/zh-TW/projects.htmlTW/projects.html)[TW/projects.html](http://g0v.tw/zh-g0v.tw/zh-TW/projects.htmlTW/projects.html)
        - 我也想開個專案協作資料夾。[http://hack.g0v.tw/about](http://hack.g0v.tw/about)
    - 我是程式設計師，show me the code！[github repo](https://github.com/g0v/)
    - 我不是程式設計師，除了 code 以外的檔案在哪呢？[google drive 共用資料夾](https://docs.google.com/folder/d/0B0NsS2a-Qx8ZN19uV1p6YWd6TXc/edit)
> 權限是 anyone can edit！ （所以檔案放到共用資料夾之前記得自己留一份備份）
> [name=ET B]

        - google drive 官方有推出桌面端程式，可以把共用資料夾的檔案同步到自己的電腦硬碟中，[按此下載](https://tools.google.com/dlpage/drive)
    - 最近 g0v 有什麼新聞？[零時公報](https://g0v.hackpad.com/ep/group/yZ9JT9UlJf4)
2.  **自我介紹、打招呼、看大家在做什麼**
    - 到 [g0v綜合論壇](https://groups.google.com/forum/#!forum/g0v-general) 的 [自我介紹討論串](https://groups.google.com/forum/#!topic/g0v-general/bp6nZlNTXls) 發文讓大家認識你
        - 加入 g0v 文化部的朋友，請額外到 [g0v 文化部論壇](https://groups.google.com/forum/#!forum/g0v-moc)  獨立發一篇較詳細的自我介紹，讓大家知道什麼樣的需求可以找你幫忙
> 因為現在 [http://hack.g0v.tw/people](http://hack.g0v.tw/people) 還沒有標籤分類的功能，理論上等標籤分類功能做好以後可以在上面貼標籤就好了，就不用詳細自我介紹了... XD
> [name=ET B]

            - 加入 g0v 文化部美術館的朋友，請額外閱讀 [g0v 美術館館長手冊](https://g0v.hackpad.com/g0v--2ozguMQjfV5)
        - 加入鄉民關心你專案的朋友，請額外到 [鄉民關心你論壇](https://groups.google.com/forum/#!forum/kuansim) 的 [自我介紹討論串](https://groups.google.com/forum/#!topic/kuansim/6rhrLsV7AeE) 發文讓專案夥伴認識你
        - 加入立法院專案的朋友，請額外到 [零時立法院論壇](https://groups.google.com/forum/?fromgroups#!forum/g0v-ly) 參與討論
    - 到 irc 聊天室和大家打招呼
        - 登入來聊天 [http://hack.g0v.tw/irc](http://hack.g0v.tw/irc)
> 如果使用一般 irc 軟體，伺服器為 irc.freenode.net，頻道為 [#g0v](https://g0v.hackpad.tw/ep/search/?q=%23g0v&via=Q185NMLUdIf).tw
> [name=ET B]

            - 如果有人給你++（讀作：加加），不要害怕，那是加分按讚的意思
            - irc聊天有些特殊用語請見  [g0v 黑話及語錄](https://g0v.hackpad.tw/MI7fGNIdygb)
        - 使用IRCCloud登入聊天室聊天 [https://www.irccloud.com](https://www.irccloud.com)
> 手機/平板界面可以用此管道
> [name=Weiting T]

            - 登入後，點 add a network > 伺服器選 freenode (會自動顯示為：_irc.freenode.net_)\> 下面 channel 打 [#g0v](https://g0v.hackpad.tw/ep/search/?q=%23g0v&via=Q185NMLUdIf).tw > 登入成功
        - 或者不登入看  [聊天室紀錄](http://logbot.g0v.tw/channel/g0v.tw/today)
        - 或者到[g](http://hack.g0v.tw/g0v-hackath4n/e8C6p3dQvzK)[0](http://hack.g0v.tw/g0v-hackath4n/e8C6p3dQvzK)[v廣播電台](http://hack.g0v.tw/g0v-hackath4n/e8C6p3dQvzK)看看快訊，找找坑，在當日留言板寫下到此一遊～
    - 加入 g0v 的 [facebook 社團](https://www.facebook.com/groups/g0v.general/) （人很多）
3.  **開始跳坑**
    - 把想法整理出來，開出需求、開始討論！看到想幫忙、想參與的項目，就把名字簽下去 →[零時政府第肆次國民大會黑客松](http://hack.g0v.tw/g0v-hackath4n/d3UYGxZtxw2) 專案構想 idea pool
    - 也可以在「[專案列表](http://hack.g0v.tw/project)」中追蹤或增加自己喜歡／想要的專案。
        - 新增時若選擇從 GitHub 匯入，可參考[這個格式](https://github.com/g0v/hack.g0v.tw/blob/master/g0v.json)。
