# ==Store & Place Manager Q&A==

## I have already implemented a real-link system, do I still need to use this system?

The focus is on providing a "real-link system." The "SMS real-link system" system is one of the methods of the real-link system, and it does not conflict with paper-and-pencil registration and the self-made forms of each store. The government will not force the use of this system.

This system is convenient for tracking the trajectory of the epidemic, which can ensure the accuracy of the information while taking into account the efficiency. It is recommended that you use it more.

## Do I need money to apply for this?

This system is completely free, please feel free to use it.

## How do I apply for a QRcode for the store?

Applications are open at 3 pm on 5/19.

1. Enter the "[SMS site QR Code registration operation](https://emask.taiwan.gov.tw/real/)" system.
2. Provide the serial number or ID number, mobile phone number, and after confirmation, you can get a QRcode containing the location code.

The location code is randomly generated and does not contain business registration or location information.

Any type of agency, unit, or individual business person can apply for a new QRcode at any time.

## How do you create branches? Is it the editor of the head office? Or the shop number?
The head office can generate the QR Code of each branch, or authorize each branch to add the branch number (only numbers) to the head office's store code and then manually generate the QR Code.

## I have not used the "mask real-name system" pre-order system before, how do I obtain the QRcode?
You can log in through multiple methods such as "Mobile Authentication", or "Health Insurance Autotoll", "TW FidO", "Health Insurance Card", "Natural Person Certificate" and so on.

## If people fill in their phone number, how can the store confirm it?
Instead of filling in the phone number, people use their mobile phones to scan the QRcode to send a text message to 1922, and show the completed screen of the text message to the staff of the place. 1922 will record the mobile phone number of the text message.

## I am a mobile vendor, where should I post the QRcode? What should I do if the guests will not bring their mobile phones?
This system is one of the methods of the actual linkage system, and it does not conflict with paper and pen registration and the self-made forms of each store. If there are things that are difficult to adopt, other methods can still be selected.

## What should I do if my QR Code is stolen and posted in another location?

If you find such a situation, please notify 1922.

## If the public is recommended to install the "Taiwan Social Distance" app, can it replace the text message?

Welcome to recommend the installation of the "Taiwan Social Distance" App, but stores that need to implement the actual linkage system still need to do so through any means including the "SMS actual linkage system" system.

(According to the guidelines for the actual linkage system, the Taiwanese social distancing app is currently unable to provide the necessary contact information for the epidemic.)


## Nowadays, some company numbers or residential communities require real-name registration when people come, so what identity do they choose? (0519 Press Conference Yang Guang asks questions)

The main difference between a legal person and an individual store is whether there is a unified organization. If there is, the legal person store is used. If not, like an apartment building, as long as there is a representative, you can use the personal store to apply.

## In this app, does the SMS outgoing area have settings for how long to keep the data and how long to delete it? Now there are many stores that need to use the google form designed by the Ministry of Health and Welfare. Will this form be used in the future? Still consider calling for the use of the current app. (0519 press conference TVBS question)

This is not an app, so there is no question of how long the mobile phone will last. The outgoing text messages will stay in the mobile phone, and whether it is deleted or not depends on everyone.

The only record that the SMS program will leave is the location code. There will be no store address, latitude and longitude information or other information, so even if the SMS is stored in your mobile phone, there will be no data leakage.

Even if the epidemic investigators go to him or share some statistics, the store will only know the footprint time and will never have any information. In addition, the telecommunications company will delete the newsletter 28 days after receiving it. This is a requirement of the guidelines for the implementation of the newsletter system.

Previously, google forms still existed. We did not want to replace any existing implementation schemes. We provide another implementation option, hoping to make it convenient for everyone. This is entirely up to the store.

## What action does the store need to confirm at the door?
Make sure that the customer scan code has brought out the store code, and the customer has sent the real-link SMS to 1922.

## Is it mandatory? (2021.05.19)
At present, to encourage and promote, protect ourselves and protect others, but if the intensity of the epidemic escalates in the future, when the use of it is mandatory, we have already started. (This answer time is 2021/05/19 11:30)

The content of the short message will be stored in the telecom company to which the mobile phone belongs, and used for epidemic prevention purposes only. It will be deleted after being kept for 28 days.
