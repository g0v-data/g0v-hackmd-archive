# 臺北街區政策倡議 GPU - Grid Practice Unit

台北街區．城市的 GPU - Grid Good Practice Unit

路網型態進行道路分級
把握車流道路，人本道路
形成「生活街區」範圍

「生活街區」是實踐市政創新的最小單元
從西門町徒步商圈到永康街徒步街，從迪化街街區保存到科技產業園區規劃
依山坡地安全社區，以及承洪韌性的街區規劃
不論是大學城、產業聚落、夜市商圈等
以街區為基礎的台北城市營造策略，面對具體需求，提出政策創見
