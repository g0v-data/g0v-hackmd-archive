# Awesome g0v 更新與結構化計畫

人坑松整理計劃
---------
g0v 社群是由「人」、「坑」、「松」組合而成，之間有許多的關聯
* 人 <-> 坑
    * 人會挖坑
        * 人起草 hackpad, hackmd 共筆
        * 人在黑客松提案
        * 人在 slack 上提出想法開始做事
        * 人會在 github 上開專案
        * 人會演講介紹 g0v 或是介紹專案
    * 人會跳坑
        * 人在 hackpad, hackmd 補充意見填寫更完整
        * 人會在 g0v 黑客松跳坑
        * 人會在 github 上提供
* 人 <-> 松
    * 人會主辦松
        * 有大松、小松、summit ..
    * 人會參加松
        * kktix 或是 fb 活動
* 坑 <-> 松
    * 坑在松被發起
        * 大松會有提案表，可以知道這次大松有哪些 slide 和共筆產生
    * 坑在松被做了更多東西
        * 大松會有成果報告，可以知道這次坑多了什麼


目標
----
* 研究者想要知道某個坑的發展脈絡可以更容易了解
* 已經很久沒進展的坑有機會再被復活繼續接手發展
* 要找做過相關專案或研究的人可以更好找到
