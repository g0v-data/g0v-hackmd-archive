# 小小聚#12 20220310 <span class="label label-success">小小聚</span>

目前的結論是不追求短時間讓 KOL 分享 go viral，而是兩個月一個週期的 campaign，搭配地球公民基金會的各種議題行銷、教育推廣（包括線下活動），目標讓在各種管道接觸到農地違章工廠的人，可以實際動動手指參與，並且兩個月後有實際可以檢舉的案件。因此不追求短時間大量轉發辨識完幾萬個點，而是讓已經有興趣的人順順玩完、理解議題和感覺自己有貢獻，地公也有數據跟政府談。
因此優先順序會有一些改變（我認為的，都可以再討論）

- 產品面：
    - 文案清楚傳達目的和議題，讓使用者知道辨識完之後對倡議的貢獻，e.g. 辨識出來的會送公文檢舉
    - 設計清楚導引辨識流程
    - 為了兩個月內可以有倡議結果，一開始需要先用一份比較少份的工廠名單，集中辨識完。 @deeper 會再提供初始工廠清冊，再跟 @酸酸的 @YAlgorithm 討論後端，以及也許可以應用 @yellowsoar 昨天分享可以抓到地號面積範圍的方式。之後小聚需要討論。
    - 不積極發想設計 fancy 的分享頁面（不預期大量社群轉發）
- 行銷面：
    - FB: 會分享大家來找廠的連結
    - IG: 會再發想比較 IG 原生的議題行銷，不一定要主推大家來找廠，因為連結在 IG 上很難擴散
