- [x] 賓

**==胸==**

```
重量訓練

1.機械推胸

2.雙槓臂屈伸

3.平板槓鈴臥推

4.器械夾胸

5.上斜啞鈴握推

----------------

腹部訓練
1.懸吊抬腿

2.椅子抬腿
```

```
飲食控制

午餐 - 雞肉,菜類,南瓜(學餐)

訓練後 - 高蛋白乳清

晚餐 - 地瓜,雞胸肉,花椰菜
```

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_c87f7e2f089338fbaa1440a7810b5f17.png)

 - 備註：1.平板槓鈴臥推腰部緊貼平板 2.眼睛位於槓鈴桿正下方 3.肩胛骨後縮下沉 4.握距略寬於肩(握把靠近虎口)
