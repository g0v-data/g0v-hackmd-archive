# CfA codeacross/brigade hangout notes

> [點此觀看原始內容](https://g0v.hackpad.tw/w1ON8D4jaNH)


時間：2014/2/11 09:05-09:45
參與：g0v/clkao, CfA/Lynn

### 前情提要

[Code for America](http://codeforamerica.org/) 舉辦 [CodeAcross 2014](http://www.codeforamerica.org/codeacross-2014/) 活動 (2/21-23)，以 [Beyond Transparency](http://beyondtransparency.org/) 為今年主題 ，同時響應 OKFN [International Open Data Day](http://opendataday.org/) (2/22)

### 會議內容

- 互相介紹，並討論 g0v 如何及是否加入 CfA network，成為 [Code For All](http://www.codeforamerica.org/about/international/)  Brigade 之一。
- g0v 社群如果決定成為 Brigade，可以在 2/22 成立，針對活動當日 CfA 會提供一些支援（活動補助金、貼紙等等）。
- 成為 CfA Brigade 優點：
    - 資源經驗分享
    - 成果更快帶入其他地區
    - 媒體資源 networking
- 其˙他摘要：
    - CodeForAll (international brigade) 自主運作，有些國家有仿效 CfA fellowship program（送人進市政府工作一年，發掘問題、帶到 community 解決），但是經費是獨立募集的
    - clkao 問: CfA Brigade 一般來說都是城市層級的運作，似乎沒有聯邦層級。Lynn 認為解決和市民第一線接觸問題，一般是地方政府。g0v.tw 目前許多 project 都是和中央政府相關，clkao 認為，也許因為地域、行政層級，多數專案在台灣適合同時和中央及地方溝通。如求職小幫手，去和每個縣市勞工局協調釋出格式並不經濟，因此可同時從中央主管單位下手。Lynn 表示這個觀察非常有趣，希望之後能多分享 g0v 的經驗。
    - CfA Brigade 建議角色分為 : Leader、Story Teller、Community Organizer、Delivery Lead (人形 hub)、Municipal Partner. clkao 說在 g0v 也蠻類似的，只是每個角色都有多個人，各 project 又略有不同，彼此重疊。Lynn 表示這個分法也算是 CfA 運作後，自然生成的生態，算是寫下來讓新地方比較能快一些進入狀況，並且會針對各種角色有 training program。
    - Brigade 非常強調與政府合作，因為才能快速讓解決方案實際被套用，所以才有 fellowship program. clkao 表示：經過三四次黑客松後，已經開始有公務人員主動想參與、把問題帶到社群，有點像是 reverse-fellowship。Lynn: 靠這比 fellowship 還酷
