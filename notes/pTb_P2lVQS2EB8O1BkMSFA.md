# 聲量分析 + 吉祥物角色

說明：採用關鍵字聲量工具，來協助吉祥物推動承辦單位，知道自己推出的角色效益

工具列舉：
- Google Trend 用角色的名稱字詞，搜尋熱度的趨勢變化 
    - 使用頁面：https://trends.google.com.tw/trends/?geo=TW
        - 可以跨角色比較
    - API
        - 使用教學
            - https://www.maxlist.xyz/2018/06/17/python_google_trend/
            - https://ithelp.ithome.com.tw/articles/10195071
        - 可以嘗試批次掛上角色名稱，取得該角色的聲量折線圖
    - 工具討論
        - Google Trend 著重在搜尋熱度
        - 不過可能角色若沒有非常熱門，Google Trend 顯示出不出結果
        - 但不代表該角色並沒有經營，例如使用 FB 粉絲專頁日常經營
- OpView社群口碑資料庫
    - https://www.opview.com.tw/product-trend
    - 試用：https://photos.app.goo.gl/VChYWEirEMHTL42t9
- i-Buzz
    - https://www.i-buzz.com.tw/api/application/1

- 更多工具：https://www.gvm.com.tw/article/53311

# 想用這類工具探討什麼題目？歡迎提供

建立單一角色圖鑑網頁，包含「活躍度圖表」
- [圖鑑畫面構想簡報](https://docs.google.com/presentation/d/1iu7xUrfRoRDe3feWRxfa74iwD9-2dJbRCJvjlc5ysCc/edit)
- 活躍度圖表，例如嵌入 Google Trend 
    - 貓裏喵 https://trends.google.com.tw/trends/explore?q=%E8%B2%93%E8%A3%8F%E5%96%B5&geo=TW
        - 可以嘗試批次掛上角色名稱，取得該角色的聲量折線圖

把聲量表現資料，與該單位的業務 KPI 相關資料，放在一起探討
- 業務機關 KPI 項目，例如活動人次、銷量等有時間格式的資料

從聲量表現，歸納一些類型與手法
- 虱目魚小子?
- 沒有充裕行銷預算的期間，如何保溫？
    - 在大型活動時間點之間，定期辦抽獎，可能會是本益比較佳的保溫方式
    - 經費模式，採用開口合約概念？
    - 實體景點，用角色名稱取名，例如貓裏喵公園，可讓此名稱不會完全仰賴行銷活動，也能透過景點被查找的方式觸及至更多使用者

舉辦一個 慰靈祭
- 用免費的 Google Trend 找出被冷落的公部門吉祥物
- 如何定義被冷落？
    - 方案一：抓一個數值當基準，分成以上或以下
    - 方案二：全部的角色聲量表現的數值後段角色
