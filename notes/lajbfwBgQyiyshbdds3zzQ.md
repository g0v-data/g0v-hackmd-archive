# 【數位協作工具】 HackMD 與 Slack 使用



## 授課老師
analeigh


## 堂數
1

## 課程星河：
### 北極星（HMW）：
v1.0

掌握數位協作工具使用，讓線上溝通變得更有效率！

本課程將會解說 g0v 社群常用協作工具的入門使用方法，學員可以透過數位共筆 HackMD，學習基礎 markdown 語法，熟悉文件協作與整理 。並搭配社群溝通軟體 Slack，讓團隊在遠端工作以及管理多項專案上，能夠更有效構通。

### 設計原則：
- 步驟拆解越清晰越好，能讓學生一步一步跟著動手做
- 讓學生練習工具越來越熟練
- 越能讓學生操作數位協作更順暢
- 讓專案團隊溝通越有效率

### 限制條件：

- 一堂課以 50 分鐘為單位
- 預錄影片
- 課堂包含講述、工具使用步驟解析、實作練習、多元評量

### 課程模組（起終一二三）：
#### 第一節-如何運用 Slack 與專案團隊、g0v 社群建立有效溝通？

1. slack 的應用場景與特色 / what／3 min
2. 如何加入 g0v slack / how／3 min
3. 運用 g0v slack 跟社群夥伴互動、特色 g0v 頻道簡介 ／what／4 min
4. slack 重點功能簡介／what & how／5 min
	1. 加入頻道／創立頻道
	2. 傳送訊息：在頻道、私密群組、私密訊息
	3. 訊息發布重點功能：連結、黑體、條列、特殊符號、插入檔案、指名討論＠、重要段落標記 pin、星號備忘清單、排程傳訊
5. 實作：在 g0v #intro 發布第一則自我介紹訊息

#### 第二節—如何運用 HackMD 建立線上共筆文件，與專案／社群夥伴線上同步協作
1. HackMD 的應用場景與特色 / what／2 min
2. 案例：g0v 社群如何運用 HackMD，在特殊社會事件情境下，展開大型線上協作 ／what／5 min
3. 如何建立你的第一份 HackMD 共筆 / how／3 min
	1. 註冊HackMD 帳號
	2. 在 g0v HackMD 開啟一個空白共筆
	3. 學習基礎 markdown 語法
		1. heading
		2. bullet point
		3. link
		4. frame
		5. form
		6. quotation
		7. color/font size
		8. insert image
		9. 網頁縮圖顯示設定：tags/title/image/description
	4. 實作：用 HackMD 建立你的源力增能營學習歷程（目錄＋2 堂課堂筆記）
4. 為什麼掌握協作工具很重要？

## 課程評量向度

因為本課程的評量向度無標準答案，夏日源力增能營時，將以自評為主。若為可即時互動的課程形式，則可改為由授課者觀察學生在課程中實做、開放練習的結果判斷。

#### 第一階：能夠按照課程影片的步驟，清晰了解數位協作工具的功能與特色
- Slack：成功加入 g0v slack，並在 #intro 發布第一篇自我介紹，讓社群更認識你
- HackMD：完成 HackMD 註冊，開啟第一份空白 HackMD 文件
#### 第二階：能夠掌握數位協作工具的功能，一步步完成設定的初階實作項目
- Slack：了解 Slack 各個功能的使用方法，開始在社群展開討論
- HackMD：掌握基礎 markdown 語法，成功完成第一份 HackMD 共筆的架構與內容
#### 第三階：根據對社群／專案的熟悉度，能夠有效運用科技工具進行數位協作
- Slack：以 g0v slack 為例，能夠流暢與社群參與者在 slack 各個頻道上互動
- HackMD：能夠依據不同專案類型，在 HackMD 上建立共筆與他人協作
