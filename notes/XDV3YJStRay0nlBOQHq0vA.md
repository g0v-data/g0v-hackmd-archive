# 臺灣政府規定
**入境旅客需進行檢疫措施
Quarantine measures required for incoming passengers**

1. 航班抵臺48小時前登入系統輸入之資料，均屬無效。
<font color="#f00">
2. 自重點高風險國家（印度、巴西、英國、秘魯、以色列、緬甸、印尼及孟加拉）入境旅客， 須入住【集中檢疫所】，入住時及檢疫期滿PCR檢測。其他旅客須入住【防疫旅宿或自費入住集中檢疫所】，抵臺時及檢疫期滿前PCR檢測。
</font>
3. 為進行社區電子追蹤關懷，請入境旅客配合使用我國電信業者門號及以個人手機(1人1機)進行申報，限特殊狀況(如：12歲以下孩童)且居家檢疫地點相同者，得共用手機申報。
4. 自臺灣本島入境者，檢疫地點不得為離島；自離島入境者，居家檢疫地點為離島。本人同意檢疫通知書以電子表單方式收執。

# 簡訊檢查站
分區狀況：
1. 有台灣門號旅客：至集中區填寫檢疫表並等候衛福部簡訊發送，收到簡訊即可離開此區域，不須等候工作人員帶領。
2. 無台灣門號旅客：需至電信業者處辦理台灣門號，完成後前往有門號者集中區填寫檢疫表並等候衛福部簡訊發送。

> 異動紀錄：
> 2021/05 為每批人潮統一由現場防疫工作人員帶離此區。

# 動植物檢查站
有攜帶動植物（含肉製品者）者，須實施非洲豬瘟檢疫，並填寫非洲豬瘟檢疫資料表單給動植物檢疫工作人員。

若未依規定主動丟棄者，將罰款3000NTD。


# 自動通關 / 人工通關
此處會由海關人員檢查入境相關資訊、登機前PCR檢驗報告，且旅客需要用手機填寫隔離文件資料，以上皆完成後才可通過。

若走自動通關仍需檢查登機前PCR檢驗報告及填寫隔離文件資料。



# 領行李
旅客直接至行李轉盤領取行李，現場將有緝食犬值勤檢查行李。

> 附記：
> 該行李已由地勤相關人員於運輸時實施外部消毒

# 接機大廳
接機大廳人流須聽從現場工作人員引導，無告示牌等指示，旅客於此處填寫紙本文件資料並領取"抗原"快篩試劑一盒（於居檢/居隔10-12天時自行使用）、量杯一個+貼紙一張（PCR檢測用）。

因無相關告示牌等指示，需要較多工作人員協助導引。
**<font color="#f00">
補充：抗原快篩與PCR檢測不同，現場可能因現場狀況而導致接收資訊不完全，旅客須多加注意此項。
</font>**


> 異動紀錄：

# 口沫採檢站
此處為PCR檢測站，設置有19個獨立篩檢站，旅客須於篩檢站裡將口水吐至量杯中（需2.5ml），並交給現場採檢人員來完成口沫採檢作業，檢驗結果將於隔離D+3~D+5時由里幹事通知。

> 附記：
> 入境時為D-Day，24小時候才算D+1，後續以此類推。

> 異動紀錄：
> 2020/10-12 未設置
> 2021/07/20 篩檢站滿編從5站擴編為19站。

# 防疫計程車站
區分狀況：
1. 往北部人員：旅客依照告示牌及現場人員指示搭乘計程車，司機會針對行李實施酒精消毒作業；若人數大於2台車則須分車，請旅客務必配合。
2. 往中南部人員：旅客依照告示牌及現場人員指示搭乘巴士

此區人潮容易聚集，需自主保持1.5m之社交距離。

> 異動紀錄：
> 2020/10 無工作人員指揮引導

# For 一線工作人員及相關單位

旅客建議事項
1. 可提供快速簡單之圖形化流程說明
2. 人力輪班須注意現場狀況進行彈性調動
3. 各站點所填寫之資料重疊率過高，建議完成水平整合及數位化
4. 各站檢查之綠色憑證重複次數過高，是否可整合為統一審核方式及相關配套措施
