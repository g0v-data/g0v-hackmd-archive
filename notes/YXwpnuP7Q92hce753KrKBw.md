# 零時小學校 4th 專案孵化競賽共筆



::: info
**【零時小學校專案孵化競賽】**
g0v 零時小學校希望邀請學生、老師，以及任何「想用行動解決生活／社會需求」的行動家，一起用開源協作的方式做專案，把想像變成可能！徵件主題不設限，而 22 歲以下青年提案將有保障名額的機會唷！計畫包含 g0v 公民科技專案資深參與者的一對一輔導、提案優化工作坊、資源媒合、入選獎金，專業評審也將從初選到決選 Demo Day 給予回饋。
- 想了解更多專案孵化競賽資訊，請至零時小學校官網：https://sch001.g0v.tw/
:::
