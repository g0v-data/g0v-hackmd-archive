# 2-2 心法

隨著數位的發展，人們表達的管道與方法越發多元，易於「發聲」的時代，需要有更多「行動」的人。

此外，AI 時代
- AI 針對有範圍和已經存在的內容，已經可以學得比人類更快更好，傳統考試範圍和標準答案的

「好棒喔！但我應該無法，我只會ＸＸＸ」、「但我不會寫程式可以參加嗎？」，這是許多人第一次聽到 g0v 零時政府的時候立即的反應，會先否定自己「做出行動」的能力。然而，「行動」其實很簡單，把想法寫下來與他人分享、查資料、整理資料、翻譯、寫文案、做圖、修改一個程式的 bug，都有助於一個公民科技的專案誕生，人人都可以在專案中貢獻一己之力，成為改變社會的力量。



- 學習如何從情感引領到思考、從思考化為行動？（不要純批評以及純情緒，要行動）
- 集體智慧：群眾智慧＋人工智慧
- 不要問為什麼沒有人做這個，先承認你就是那個沒有人。
- 尊重現實，尋求解方
- Release Early, Release Often. 盡早釋出，經常釋出
- Talk is cheap, show me the code. 空話少說，放碼過來
- Everyone's business, everyone helps. 眾人之事，眾人助之
