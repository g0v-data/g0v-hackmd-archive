# 幻物平台構想

> [點此觀看原始內容](https://g0v.hackpad.tw/hMSmTcDsOd7)

fb 社群：[不要再買了！免費的幻物與幻務](https://www.facebook.com/groups/392133004299683)，提供一個以物易物、也容許勞務交換的場域。隨著使用者增加，社群也在構思「幻物平台」的各個環節與操作期待：
> 歡迎幻物社團的朋友一起共筆！
> [name=che l]


使用者
- 提供
    - 上傳可供交換的「幻物」或「勞務」
    - 提出自己想要換到的
    - 有照片為佳
    - 地理資訊，協助其他使用者判斷
- 針對這些幻物/務，給予標籤
- 「標籤系統」本身需要由參與社群來定義，不一定是傳統物品分類習慣，這部分需要社群討論
    - 標籤的UI可參考憲動盟野生官網：[http://g0v.github.io/don-constitute/public/grassroot-10-tapcpr-20150201](http://g0v.github.io/don-constitute/public/grassroot-10-tapcpr-20150201)
    - 幻物（借用或交換）：媽媽小孩的一批，農業農產食物，手作品的一批，專業工具器材的一批，耗材一批，衣物裝飾品一批，生活用品一批.....
    - 幻務：旅遊（包含移動，住宿空間提供或實際帶領），農事協力，教育陪伴，身心靈療癒，專業咨詢，技術教學分享，資訊門路提供......

如何促成一次交換
- 如何增進使用者接受海量即時的交換資訊呢（雖然說錯過也是一種緣份啦XD）
- 使用者可以追蹤自己關心的幻物類型，並且系統可以通知該幻物已出現
    - 可參考  [TAAZE｜讀冊生活](http://www.taaze.tw/)  徵求二手書的方式，在釋出物／務的同一個頁面，開放有相同需求的人徵求留言
    - 使用者可自由選擇，匿名徵求或是公開徵求，匿名徵求者只對擁有該物品的人（或同時參與討論的人）顯示個人資訊
- 幻物的同時，也有機會釋出「協助送貨」的需求（參考一則社群經驗高雄to台北，網址待補）
- 熱門的幻物，會出現「排隊情境」，目前既有的做法是看 fb 內的留言先後順序
- 雙方需要確認交換細節
    - 表達「同意」
    - 物品運送方式、或是會面
- 實際進行交換

幻物結束後
- 個人幻物心得分享圖文（[參考網址](https://www.facebook.com/photo.php?fbid=10206511703967158&set=gm.417607301752253&type=1&theater)）
- 物品已幻出，之後的狀態更換，以往是FB介面手動打字修改，能否更簡易完成
    - 應該也是可以用tag功能來做
- 幻物心得是否與幻物頁面做連結

公告訊息
- 不定期發起幻物市集實體活動
- [各種寄送方式歸納整理](https://www.facebook.com/notes/%E4%B8%8D%E8%A6%81%E5%86%8D%E8%B2%B7%E4%BA%86%E5%85%8D%E8%B2%BB%E7%9A%84%E5%B9%BB%E7%89%A9%E8%88%87%E5%B9%BB%E5%8B%99/%E5%90%84%E7%A8%AE%E5%AF%84%E9%80%81%E6%96%B9%E5%BC%8F%E5%8F%83%E8%80%83/410488619130788)
- 輔助新手的圖文

技術討論
- 關於 FB group
    - 技術上是否沿用呢？類似路殺社模式，FB+app
    - 如果技術上沿用，個別使用者可以有以下功能（皆可由使用者自行選擇是否開放，此舉等於將幻物功能並置社群聯結的功能，人與人的關係也是重點這樣）
        - 自我介紹、交換文分享、過去的交換紀錄、欲徵求項目、追蹤的幻物／務類型（可選擇是否要被搜尋，願意被哪些人搜尋，其他使用者可透過此功能搜尋到有相同興趣的人）、過去參加或舉辦過的活動顯示
        - 使用者想找尋相關幻物時，可以選擇透過物品全面搜尋，或是只向特定區域的人搜尋，或是只對設定的社群搜尋
        - 系統可透過當事人的紀錄，自動偵測其興趣並提供建議的項目（可自行選用是否開放）
    - APP功能希望可以有地區搜尋功能，特別適用於借用東西
- 思考，技術化操作，而非原先透過留言的方式，是否可能弱化人與人之間因交換而產生的聯結與單純開心？
    - 使用交換而非金錢，就可以保有這點了，是嗎？
    - 標籤的活化，可帶動走向與潮流，由使用者自行創立是重點，系統要如何保有新鮮流動，並兼顧小眾需求
    - 從虛擬頁面到實體認識交流，是持續創造社群動能的火花，創造並保有這份認識的意願和可能性很重要（如舉辦實體幻物趴，或相關永續生活的活動，持續推坑（？）（我是新手，但興趣參與實體社群的活動策劃與相關教育，因開了一家以分享互動為目的地新空間，或許又適合的資源可共享）
    - 與價值觀相似的社群有所互動交流。
    - 舉辦幻物故事分享，增加社團凝聚力，教育新手或還未接觸幻物人士共襄盛舉。
