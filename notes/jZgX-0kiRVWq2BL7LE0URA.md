- 標題：Agentic Citizens: Can AI represent us in public deliberation? 
- Presenter: Josh 
#### 簡報架構
- TFGA & vTaiwan [name=peter] [name=yeeting]
- 既有的 vTaiwan 程序介紹 [name=peter]
    - 先利用網路與數位工具徵集意見
    - 邀請利害關係人參與
    - 整理相關意見，找出共識最低的
    - 藉由實體討論
- 既有的 vTaiwan 程序遇到的問題[name=peter]
    - polis 不會收集到大家的資料，難以調查參與者的 demographic data
    - 實體活動與線上活動的參與者高度集中在台北，且具有相似的教育背景與年齡區間
        - self-selection bias: 因為我們通常是邀請多元利害關係人參與，所以來參加的人很多是本來就對這個議題有興趣，可能會出現缺乏代表性的問題。
- 突破：利用 AI agent 來解決前述的問題 [name=peter]
    - 先用 pilot discussion 搜集多元利害關係人的意見
    - 利用問卷，藉由社群媒體散佈來做隨機抽樣
    - 藉由問卷
- 議題：反詐騙的社群媒體審查 [name=peter]
    - 為何會選擇這個議題？
    - 相關的背景資訊介紹與新聞整理
- pilot discussion [name=peter]
    - 參考去年的活動共筆
    - 整理初步意見
- Agent 設計 [name=josh]
    - 如果要做 AI agent 的話，問卷設計上有哪些不同？
- Agent 結果 [name=josh]
- 限制：
    - 可靠性不足以取代 vTaiwan 既有的程序
    - agent 如何 align with 使用者
        - 能否創造一個機制是讓使用者可以自己給 agent 回饋
        - 目前只收集一次的資料
    - agent 的討論品質會與 model 的能力高度相關
- 建議
    - 這樣子的方式可以補足 vTaiwan 現有程序上的不足
    - 信任與 alignment 會是關鍵
    - 基礎建設：社群、問卷（實際的 prompt) 還有 AI model
