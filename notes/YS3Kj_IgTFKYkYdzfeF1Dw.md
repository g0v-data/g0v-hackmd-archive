 云端运算例子：Amazon Web Services (AWS) EC2
 
* 說明使用法
* 注册和登录
注册一个 AWS 账户，并登录到 AWS 管理控制台。
* 启动实例
在管理控制台中，搜索并选择“EC2”服务。
点击“Launch Instance”按钮。
选择一个预配置的 Amazon Machine Image (AMI)。
* 连接实例
实例启动后，在控制台中找到实例，点击“Connect”按钮。
根据实例类型
* 管理实例
可以在 EC2 控制面板中启动、停止、重启或终止实例。
配置负载均衡和自动扩展（可选）。

*   使用方法
* Amazon EC2 (Elastic Compute Cloud) 允许用户租用虚拟服务器（称为实例）来运行应用程序。用户可以通过 AWS 管理控制台、命令行界面（CLI）或 API 启动、管理和终止 EC2 实例
* 工作原理
* 用户通过 AWS 控制台或 API 提交 EC2 实例启动请求。
AWS 提供的资源池中选择适当的硬件，并根据用户选择的 AMI 启动实例。
实例启动后，用户可以访问和配置其操作系统和应用程序。
使用 EBS 存储来保存数据，并通过 VPC 管理网络流量。
