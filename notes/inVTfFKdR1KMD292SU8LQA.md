# 沒有人罷免指南 N0body Recall Manual (N0RM)

## 專案目的

- **It's about time**：隨著第二階段罷免案已全數送件，進入投票前的關鍵時期，亟需維持社會討論熱度，讓議題不被遺忘。
- **整合寶貴經驗**：多個罷免團體在實際執行過程中累積了豐富且珍貴的經驗，這些經驗值得被記錄、整理與分享，避免重複踩雷，並為未來行動提供參考。
- **採用開放授權**：本專案預計文字部份採用 CC BY（姓名標示）授權，網頁部份採 MIT 開源授權，確保所有內容可被自由存取、引用與再利用，促進資料的流通與知識的累積。
- **便於散佈重製**：文件以 HackMD markdown 語法編輯，方便團隊協作與 AI 存取及訓練，也利於日後資料延伸、重製與跨平台分享，提升專案影響力。
- **確保長期存取**：借助 g0v HackMD 及 GitHub 託管，確保未來資料維持可存取性，避免過往像是保管文件的雲端帳號權限丟失，造成資料消失的困境。
