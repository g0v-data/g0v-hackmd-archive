文／林冠廷

仰望夜空時，我們總有幾位朋友可以信手捻來，指出各個星座所在位置，或是透過大三角判斷季節。但除了遠望之外，如何將數不盡的天文開放資料轉化為教育資源，拉近這群天文愛好者與星空的距離，並讓他們對天文教育有所貢獻？具有天文學家背景的工程師蘇羿豪，想藉由開源技術與天文領域長才，建立以開放資料協作教材的天文教育社群，培養不分年齡的「公民天文學家」。
 
### 從開源新手到開創社群
蘇羿豪是中央大學天文博士，研究主題是藉由分析黑洞吞噬星體過程所發出的光，以了解其物理機制。研究所期間，他發現國外天文學術界普遍已在使用開源的程式語言 Python 抓取、分析觀測資料。實際嘗試以 Python 用於自身研究後，蘇羿豪體驗到其門檻低易學、開放文化、社群龐大、應用廣泛等魅力，便與天文所的同學組織 Python 討論會，並成為社群活動常客，舉凡 g0v 大松，或 PyCon、COSCUP 等開源研討會都能看到他穿梭其間。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_140abe453749fea7537f54342947a929.jpg)


受到 g0v 與國外的天文黑客松（Astro Hack Week）啟發，並秉持研究者應公開原始碼的「開放科學」精神，蘇羿豪與學長陳正忠創辦 Astrohackers in Taiwan（天文黑客在臺灣）社群，關注 Python 應用、開放科學與公民天文學議題，並透過黑客松與工作坊，讓大眾認識開放天文資料，協作天文數理教育教材，達到公民參與天文推廣教育的目的。他並實際以太陽系外行星的開放資料，開發互動式網頁「歸途」，藉由遊戲化方式讓使用者了解何謂太陽系外行星、描述星體距離的常用單位等知識。也因為「歸途」介面簡單清晰且資訊豐富，讓蘇羿豪有機會在 PyCon 中將成果分享給社群成員。

 ![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_7fffc951251cce9d1968ad4b167a5b46.png)

 
### 零時小學校開張，黑客松大升級
除了自己開發教材，蘇羿豪希望透過實體活動，培養社群貢獻的能量。2017 年至今，他與母校興大附中老師林士超合作，利用學校場地舉辦三次的全日黑客松，學生與社會人士都能參加，希望帶著高中生一步一步吸收知識，實際產出天文相關的開源專案。他說：「我高中時就想走天文研究，雖然有參與過墾丁天文台的觀測實作課程，但當時鮮少有機會接觸開放的天文資料。現在舉辦黑客松，就能讓高中生用『專案式學習』接觸天文，使用開放資料協作能解決自身學習痛點的專案。」

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_4bb7c18be4627c39bfad884c75b5bcef.png)


蘇羿豪坦言，首次舉辦黑客松，難免會面臨教學與實作時間不均的問題。「但藉由零時小學校的提案，讓我更聚焦在『參與者就是要使用開放天文資料』，並且幫他們訂好方向，即製作教材，用於自學並提供老師或大眾教學使用。」經過不斷的調整，今年 3 月舉辦的黑客松不但圓滿落幕，更有同學在一天內產出教材，以開源授權釋出。此外，他從事教育的碩班同學潘康嫻，也與他討論如何改作這些教材以應用在教育現場。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_4ed9cd8631f28874be190553e0ceec5f.jpg)

 
### 從「零」開始建立公民天文社群
從 2015 年接觸 Python、參與黑客松及開源社群至今，蘇羿豪看到了藉由開放協作的方式，讓群眾參與貢獻成為「公民天文學家」的各種可能。這也賦予了專案名稱裡面饒富趣味的「[0]」。數字零以及中括號，代表程式碼一連串元素的首項。他笑著解釋：「公民天文學家不只做教育，也有研究和更多面向，所以從零開始，之後也將擴及到一、二、三、四、五⋯⋯」。有了初步成功，他預計於未來參與更多天文教育活動，尋找師生需求，協力尋求解決方案。

絢麗又奇幻的滿天星斗，蘊含著宇宙誕生之謎，也是人類爭相競逐的探索目標。未來，蘇羿豪將陪伴著他一手打造的社群，從代表教育的「零」發展，打造屬於社群的燦爛星空。
