#### 裝修規劃:

#### Q & A
1. 工班 : 自己找要注意的 
- 西門 : 對工程不懂給點錢比較好。分包對工班不熟也不知道合不合理。工序問題, 到底做得對不對。現在師傅很會欺負人只能做他對的事。他會用自己最習慣的工法做, 你告訴他做他覺得麻煩就會說不能。自己有辦法判斷就可以自己來
- 找監工 : 但也是要確定監工是有用的。如果堅工了解他曾經工作的態度和內容是可以信任的就可以交給他。
- 自己統包會有點辛苦, 不見得不行。要忍受一天到晚接到電話, 處理事情。
- 師傅會說 : 你當初就這樣講。他理解的跟你講的不一樣, 很有可能發現這種問題。師傅錯了打死不會承認, 因為他要花工。
- 西門 : 不會出這狀況 > 施工圖畫非常完整。只畫3D圖是不行的。畫圖不偷懶就可以不用去工地。要確定你寫得夠清楚
    - 曾經4合一冷風機, 現場師傅電買了110v, 應該用 220v, 師傅直接說圖是用110v, 我就拿了圖跟他說是220v, 他已經裝上去是不能退。因為有那張圖師傅只能摸摸鼻子。
    - 我只相信白紙黑字, 寫很清楚比較不容易有爭議, 因為沒有人想賠錢。發現要賠錢都會把罪推到別人身上。死凹他沒有錯

- 工班的溝通 : (西門)不太知道工法得請教他要用什麼工法 對施工工序材料不熟悉只能去請教師傅 施工是比較專業。他說的材料可以先標註在報價單上 (什麼材料 幾次 轉角會不會要多處理 加玻璃網子)我做是一定要加(轉折點)-剝簽網, 好處地震裂縫有網子不容易破

- 推薦工班 : (西門)師傅不會找自己麻煩因為工程都有保固。正常工班會都顧自己品質 直接詢問也是會告訴你。 看你要求到什麼程度, 寧花錢也不想以後麻煩 (龜毛)。可用也是不行, 可用也是可用。設計公司不一樣, 不能只做到可用一定要做到超耐用。

- 比價 : (西門)估預算可以先問還有沒有調整的空間。沒有想調整就是沒有一定想要做這個案子, 願意調-就是有心。不建議一項項議價, 奇摩子比較不好, 直接砍總價就好, 不要一項項工項討論價值 (總包)。

- 學裝修經驗 (總包): 西門 - 業主想去工地隨時想去就去, 業主付錢的最大
- 讓人奇毛子壞 : 改來改去 講好要進場了才改 (材料都叫好了要進場就白去了)
- 第一次去 > 工程費不要給太爽快。10-20% 保留, 做完還是有保留, 除非確定完全沒問題。進住一-兩個禮拜確定都沒問題, 不然錢都付了有問題叫師傅來會愛來不來。
- 哲: 詢價可以問到多細
    - 西門 : 工法/ 材料要一開始就先講好。比三家都一樣才有辦法比, 每一家都有自己的寫法你不知道有沒有漏掉。以為比較便宜可能漏東漏西結果沒有比較便宜。可能有個方法(話術)> 問第一家, 符合你需要, 第二家問一樣的問題聽他講怎樣, 第二家沒有很符合需求, 照第一家說法: 改成這樣....可不可以。如果估出來, 問比較高價的，預算有限能不能改成什麼什麼做法

- 拆除/清運:
    - 西門 : 價差宇宙無敵大只能多問, 沒有公定價, 粗工沒有細膩品質問題。拆除會在工地待整天, 一離開回來線可能剪壞了 什麼拆壞了。拆除會有工具費, 有些用切 動到其他機器 ... 安全問題, 要非常注意安全問題, 玻璃/磁磚一定要帶手套。裝的時候不能凸出麻布袋, 碎磁磚非常利, 總電一定要全關, 瓦斯/水一定也要確定全關閉才能做拆除。不然會非常可怕。現在拆除通常都是外勞, 台灣人做勞力很少。有些人很瘋狂有些人一拆完就全部給你丟掉。有做過業主剩下的材料非常貴想保留, 就通通被丟掉了。
    - 拆除非常強調安全 : 該戴手套就戴手套 該戴帽子就戴子, 釘子...都要非常小心 。
    - 外籍 : 通常會有一個會說國語但不一定是正常。(西門)雖然配合很多年但也不敢跑, 每次狀況都不一樣。
    - 垃圾車不滿也是算一車/ 有的會算半車。目前垃圾車市場比較大，說多少就多少

- 趨勢/禁忌: 
    - 西門 : 我們通常不會做這件事(公開價目)。比如木工報給設計師的工資2k-6k都有, 不同師傅程度工資是不一樣的。其他不懂的人就會說2.5k為什麼報5k。師傅工的細膩程度是不一樣的 會引起誤會。可以放總價但不要有細項
    - 最敏感大家最在乎價錢, 工序大家不懂就算拍施工照片也看不懂。
    - 報價單價絕對不會公開, 價差非常大。不是本行人不會知道價差的道理, 對師傅是不公平的。

- 業界忙碌程度: 
    - 西門: 問我不準, 我的工班是要搶人的
    - ddio: 以我們的工程 > 算短的工程
        - 西門 : 看師傅要不要做, 要做就會儘可能喬時間給你。我的師傅就沒在欠錢欠工程的。不是工程短。有的師傅覺得東西只有一點點。有時候做半天一點點就做完了，我那天有空就可以來你這收掉。就看師傅有沒有想幫你解決這件事 他想做就會說你要的這天我不可以，但我哪天可能可以。
        - 工班時間 : 約師傅不可能今天約明天到, 最少前一個禮拜預約。我的工程不接受delay我會先做好所有的準備。要給師傅自己安排工程的時間, 給的時間愈充足就愈不會延誤工期。最少提前一個禮拜。

        - 西門: 廣告說去過巨匠就可以當室內設計師。門檻很低就算隨便做也是可以的。眉角。多久會有問題做下去才知道
