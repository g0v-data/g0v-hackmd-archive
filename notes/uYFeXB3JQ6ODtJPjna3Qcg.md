# 學生專案，如何聯繫議題團體與公部門？

## 說明

察覺問題，找出關心的主題後，會不會有相關的 ngo 組織、公部門，可以洽詢請益呢？


## 步驟

您認為您關注的主題，是屬於哪些類別呢？
- 1-SDGS-終結貧窮
- 2-SDGS-終結飢餓
- 3-SDGS-健全生活品質
- 4-SDGS-優質教育
- 5-SDGS-性別平權
- 6-SDGS-乾淨飲用水
- 7-SDGS-人人可負擔的乾淨能源
- 8-SDGS-良好就業和經濟成長
- 9-SDGS-工業化、創新及基礎建設
- 10-SDGS-消弭不平等
- 11-SDGS-永續城鄉
- 12-SDGS-負責任生產消費循環
- 13-SDGS-氣候變遷對策
- 14-SDGS-海洋生態
- 15-SDGS-陸域生態
- 16-SDGS-和平、正義和公平的制度
- 17-SDGS-全球夥伴關係
- 18-行政制度變革與民主治理
- 19-新聞訊息與媒體識讀
- 20-開放資料與開源社群基礎建設
- 各項目的詳細描述，可參考行政院國家永續發展委員會的網站
    - https://ncsd.ndc.gov.tw/Fore/nsdn/archives/meet3

如何找團體？
- 可以至 g0v slack #edu 詢問
- 從「政府開放資料集」找相關單位
    - 每一份資料集，都會有一個發布這份資料的單位
    - 資料詢問與答覆，
- NGO 與基金會搜尋網站？
- g0v 專案 https://g0v.hackmd.io/@jothon/intro/https%3A%2F%2Fapp.awesome-table.com%2F-MnVz4_I6B5OzsFZUcaq%2Fview
- 公民科學與群眾參與計畫資料集 https://g0v.hackmd.io/@chewei/cs/
- GRB https://www.grb.gov.tw/


通常與團體之間可以有哪些層次程度的交流方式？
- 視訊會議訪談
- 實體訪談
- 電話訪談

找到可能相關的團體或公部門之後，如何介紹自身的專案進度？如何展開聯繫？
- 先寫信
    - 有模板或大綱嗎？
        - 介紹我們是誰
        - 想了解什麼、想詢問什麼
        - 提供自己團隊的 Landing Page 或是 Hackmd-書本模式網址
        - 提供自己團隊的聯絡方式，可以考慮用 googlegroup 
- 依照回信狀況
- 打電話
    - 上班時間
    - 避免午休時間
