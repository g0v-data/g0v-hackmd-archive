# g0v tour guide 個人主線任務劇本

> [點此觀看原始內容](https://g0v.hackpad.tw/QADGWZSU2Dw)

整理自 [g0v 教育部 hackfoldr](http://hack.g0v.tw/g0vMOE)、[leve1up 小學校](http://g0v.github.io/leve1up/)、[新手教學 pads](https://g0v.hackpad.tw/ep/group/otfOozEJyvl)、[零時的學習 pads](https://g0v.hackpad.tw/collection/A0o8nonStzd)

### 本文目標

攤開所有人資 / 教學的 link，變成樹狀故事線，之後可以變成 g0v tour guide 的劇本，直接轉成 web game 之類的。

### 使用方式

- 本文件：直接看，邊看邊點連結
- 未出世的 web game：使用某個神奇的 hackpad 2 webgame converter 把劇本自動轉成可以選選項的 web 介面，或直接用 leve1up 的形式也可以
