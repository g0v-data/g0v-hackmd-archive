#### 海洋保育地理資訊平台（海洋委員會海洋保育署）網址
https://iocean.oca.gov.tw/OCA_OceanConservation/GIS/MyGIS.aspx

#### 描述
平台分成四大類型及資料集
 
| 類別               | 資料來源               | 備註                               |
| ------------------ | ---------------------- | ---------------------------------- |
| 1 海洋生物目擊紀錄 | 民眾上傳回報紀錄 | 有回報紀錄經由專家判識後皆在此呈現 |
| 2 垂釣成果紀錄     | 民眾上傳回報紀錄 | 有回報紀錄經由專家判識後皆在此呈現 | 
| 3 海洋生物擱淺紀錄  |  民眾上傳回報紀錄 | 農委會林務局建置之TCSN擱淺資料庫，涵蓋了自民國83年起迄今所有經通報之鯨豚、海龜擱淺紀錄。 |
| 4 主題圖層     | 各類民間及官方資料 | 例一 珊瑚體檢：台灣環境資訊協會  
| | | 例二 海洋污染事件:環境保護署、海洋環境保護署|
