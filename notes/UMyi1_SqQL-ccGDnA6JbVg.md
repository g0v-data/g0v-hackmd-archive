# 雙溪高中俐瑜老師訪談綱要
:::info
時間：109年11月26日（四）19:00
地點：板橋車站一樓星巴克
訪談人：CoTeach 專案成員 Ada
:::

### 教學經歷
- 除雙溪高中公民科以外，是否有其他任教學校、科目與課程，以及任教年級分佈
### 新課綱-素養導向課程設計
- 瞭解老師先前社會領域素養導向課程設計的經驗 
- 新課綱是否增加教師之間協作需求？先前協作的經驗為何、有沒有受到現有平台什麼樣的限制？

### 網路/科技使用習慣
- 個人慣用的社群網站
- 習慣閱覽的內容網站/新聞網站/多媒體平台
- 生活中規劃事情時會使用的軟體或網路服務

### 任教學校科技使用習慣
- 在學校裡老師間會使用哪些通訊軟體
- 學校工作時常使用哪些文書處理軟體或網路服務

### 教案使用
- 個人使用教案輔助教學的頻率，以及是否因人/科目而異
- 和課程內容相比，教案在教學中扮演的角色為何
- 如何決定是否要在課堂中使用某個教案
- 教案使用後如何評估教案對學生的學習是否有幫助


### 教案內容
- 一個好的教案應該具備哪些元素
- 嘗試過哪些長度的教案？怎樣的教案長度教學效果最好？
- 目前使用的教案會使用哪些多媒體素材？例如：影片/圖片/音源/podcast/電影

### 教案製作前準備
- 通常是什麼原因會想製作新的教案
- 如何構思教案idea
- 如何構思教案大綱及內容
- 開始製作前會花多少時間蒐集相關素材
- 會使用哪些資源做事前研究/蒐集素材
- 是否在製作前找人討論交換想法

### 教案製作
- 完成一份教案所需時間
- 是否會和其他老師合力製作一個教案
- 通常會使用哪些軟體製作，為什麼
- 一學期大概需要製作幾個新教案

### 教案共享
- 校內外提供的教案交流管道與社群
- 是否常使用其他老師製作的教案
- 是否會主動和相關科目老師分享自己製作的教案

### 社群參與
- 除了教案交流之外，平常還會參與哪些教學相關的社群？怎麼知道這些社群的？
- 社群參與對於教學能力成長是否有幫助？有的話有哪些具體的幫助？
- 任教學校老師參與社群比例
