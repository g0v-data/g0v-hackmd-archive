# 《鯨典RP》規章 
## 📢【規章說明】
鯨典RP共有兩個主要規章必須遵守與詳讀，分別為 **【演出者規章】、【扮演規章】與『其他規章』**。
▲ **【演出者規章】**：意旨 **『演出者本身』** 之相關規範。
▲ **【扮演規章】**：意旨 **『扮演角色』** 之相關規範。
▲ **【其他規章】**：意旨 **『後續設定』** 之相關規範。
▲ **【記點制規範】**：共有兩種記點單位，分別為 **『警告』** 與 **『記點』**。
```bash
#『記點與警告』相關
"警告一支" = 兩個"警告半支"
"記點一支" = 三個"警告一支"
"記點三支" = "強制離鎮，依情節『黑名單』"。

#『違反規章』以『告訴乃論』為最高原則。
#『檢舉』依『申訴日』來算，如超過4天演繹時間以上之檢舉，則不與審理。
# 採取『事件制記點』而非『次數制』。
# 黑名單：不得加入後續所有【鯨典RP】相關企劃。
```

## **:pencil:【__演出者規章__】**

```md
#第一條、請保持愉快的心情以及負責任的態度加入，禁止將情緒帶入或帶出RP。
        【違反："警告一支至記點一支"】

#第二條、請勿在未確認事實前，就散播他人操作判斷Fail RP之言論。(不限任何公開平台)
        【違反："警告一支至記點一支"】

#第三條、請勿在『RP』與『演員企劃群之DC頻道』，進行R18露骨行為等描述。
        【違反："警告一支至記點一支"】
        ※ 專指：『(沒意義/沒必要)之腥羶色之描述，並噁心不尊重別人』。

#第四條、請勿在『非RP演員企劃群之DC頻道』，進行『暴雷或預告劇情』等行為。
        【違反："警告兩支至記點一支"】

#第五條、請勿對『腦補』談論之相關演員，進行『質問或尋求答案』等行為。
        【違反："警告兩支至記點一支"】

#第六條、演出期間，產生任何『演繹糾結問題，請在『演出完畢』後，進行『開會調解』(須一名管理組)或是『填寫申訴表單』。
        【違反："警告兩支至記點一支"】

#第七條、鎮民彼此之間私交內容一律不涉入，同時勿將私交關係混入RP之中。
        【違反："記點一支"】            

#第八條、非自願原因(戲下)需殺青，需自行與相關劇情的演出者進行溝通。
        【違反："記點一支"】
        
#第九條、嚴禁演出者對種族、膚色、性別、性傾向、言語、宗教信仰、政治傾向、國籍、經濟狀況、家庭背景、長相、身心障礙等進行攻擊、針對、歧視與差別待遇。(不限任何公開平台)
        【違反："記點一支至三支"】

#第十條、發生上述無所提及之爭端，企劃與管理組，決定記點內容與公告。
        【違反："警告半支至記點三支"】
```
## **:pencil:【__扮演規章__】-《__主要禁忌__》**
```bash
【任意違反："警告一支至記點二支"】
【Extra Rule："相關規範之補充"】
#第一條、禁止超能扮演(PowerGaming, PG)：
         ▲ "做出非常人的行為或是利用遊戲機制達成特殊目的之不良行為"。
         Ex："狂跑步而不喘。包含，剛做完激烈運動等行為"。
         Ex："受傷表現完全沒事，接受忍痛之演法，但需要有演繹相關行為"。
         Ex："演出不同傷勢情節，接受假裝之演法，但內心戲需要呈現出來"。
             
#第二條、禁止混淆扮演(MetaGaming, MG)：
         ▲ "將RP外的資訊帶入到RP內，造成平衡與情報不對等。"
             Extra Rule #1-1：任何條款私訊內容皆不於申訴範圍。
         Ex：透過其他平台得知角色A位置，進行追殺或刻意跟蹤之行為。
             ▲ "有效證據：直播平台時，擁有該玩家之帳號出沒於當時的截圖"。
         Ex：透過非企劃提供之其他平台進行位置信息等交流。
             ▲ "有效平台：Discord的演員企劃群"。
         Ex：因演出問題，對於玩家A不爽，在RP演出中，一直針對角色A至退出伺服器為止。
             Extra Rule #1-2：會針對當前劇情與人設問題來裁決。
         Ex：利用遊戲外的資訊塑造自己的角色，包含戲外的推測與提示等。
             ▲ "遊戲外資訊：『(非人設表/劇情)所能提供之資訊』"。
             Extra Rule #1-3：在戲外時間進行解謎行為時，在戲以不可能的(描述/動作/時間成本)來完成。

#第三條、禁止濫殺(Death Matching, DM)：
         ▲ "肆意亂殺與攻擊(NPC/演出者)之行為。"
             Extra Rule #2-1：遭遇暴民進行反殺之行為，依照人設合理性與劇情進行裁決。
             Extra Rule #2-2：遭遇暴民攻擊致死之狀況，請RP正常演出。
             Extra Rule #2-2-1：如RP劇情不合理至無法正常演出狀況，參閱#2-2-1-1~2。
             Extra Rule #2-2-1-1：身旁無人，接受(黑畫面/下線)處理。
             Extra Rule #2-2-1-2：身旁有人，則繼續RP演出，請勿情緒問題馬上下線。
             Extra Rule #2-3：請勿直接使用槍枝進行爆頭攻擊。
             Extra Rule #2-4：NPC也是人，請視為居民對待。
             Extra Rule #2-4-1：請勿因為NPC不回你話，直接進行攻擊；劇情問題不再此限。

#第四條、禁止仇殺(Revenge Killing, RK)：
         ▲  "死亡復活(頭暈甦醒)後，立即尋找兇手報仇，牽涉混淆扮演與超能扮演。"
             Extra Rule #3-1：混淆扮演，你不一定知道是誰把你弄傷的。
             Extra Rule #3-2：超能扮演，必須考慮你的受傷環節來決定是否能立即動身。
             Extra Rule #3-3：指使別人復仇，皆不在此範疇。
#第五條、禁止混淆情緒(Out-Of-Character Hit, OOC Hit)：
         ▲ "將(現實/遊戲)情緒帶到(遊戲/現實)世界。"
```

### 〈基本相關準則〉

```bash
#第一條、請保持(尊重/友善/包容)的態度，將鯨典RP視為另一個人生。

#第二條、演出請假，請給予RP戲上演繹解釋。
        Extra Rule #4-1：請告知企劃管理組與相關『劇情人員』。
        Extra Rule #4-2：無闡述相關理由給予與相關角色告知(狀態/行為)，累積消失4天以上，強制離鎮處理。

#第三條、請確保電腦之硬體相關需求，必須能正常(錄影/直播)之一。
        Extra Rule #5-1：電腦CPU底線：開(直播/錄影)跑GTA5(低畫質)能30~60fps以上。
        Extra Rule #5-2：電腦記憶體：10GB以上(建議12GB以上)。
        Extra Rule #5-3：電腦配備：麥克風、滑鼠(鍵盤)；手柄玩家會有部分功能無法使用。
        Extra Rule #5-4：網路需求：因應環境狀況，以『演出者』自身能忍受為基準。

#第四條、變裝採取認型不認聲，部分職業則有特殊規範。
        Extra Rule #6-1：儘管有口癖之原因，除非有證件或親口承認，否則皆為不同人。
        Extra Rule #6-2：對於變裝人士，皆為陌生人的存在。
        Extra Rule #6-3：對於變裝成其餘鎮民樣貌，會依照人設合理性進行裁決。
        Extra Rule #6-4：禁止隱形裝扮，儘管解釋成『高級光學迷彩』。
        
#第五條、(劇情殺/殺青/離鎮)請順其RP劇情與角色心態而行。
        ▲  "劇情殺：該RP角色真正意義上死亡，不接受死而復生的後續劇情安排。"
            Extra Rule #7-1：假裝劇情殺的情況，則不再此限。Ex：裝死、隱瞞真相。
        ▲  "殺青：演出者完結，該RP角色的整體演出。"
            Extra Rule #7-2：殺青人員，如有需要後續劇情演繹，請洽『企劃管理組』。
        ▲  "離鎮：演出者因記點強迫離鎮，將由企劃管理組，決定後續該RP角色演出與解釋權。"


#第六條、禁止利用(伺服器/FiveM/TS3/DC)漏洞來獲得好處。
        ▲  "好處：獲得非常人所能執行之行為，獲益於演繹之成本或目的。"
        Ex：被人追殺時，使用DDoS攻擊(相關人士/伺服器)，免於被追到的問題。
        Ex：透過程式漏洞，進行刷錢、複製車拿去(賣/拆車)、刷槍或使用修改器。
        Ex：在車子上，進行(採集/釣魚/販賣)等行為。
        Ex：使用非本企劃允許之光影模組。
        
        
#第七條、請勿播放有版權之音樂。
        Extra Rule #8-1：播放音樂的演出者，有義務調查是否有侵權情況。
        Extra Rule #8-2：如在審核違反規章之情形，發生消音等狀況，會採取連坐法。
        Extra Rule #8-3：如遊戲內建聲音，發生消音情況，則與『企劃管理組』回報。

#第八條、演出違反鯨落世界觀或人設之情況，依情節給予記點。
        Extra Rule #9-1：違反人設且無解釋性，記點兩支~直接離鎮。
        Extra Rule #9-2：違反世界觀說法，警告半支~警告一支。
        Extra Rule #9-3：演出的任何行為，遵從扮演角色與合理性出發。

#第九條、請勿使用(T)聊天框進行文字聊天。
        Extra Rule #10-1：5秒內使用/clearall指令，可以免於懲罰。
        Extra Rule #10-2：聊天框之玩家文字訊息，請勿拿來進行RP演繹。
        Extra Rule #10-3：聊天框之系統文字訊息，通常皆為個人所獲得之資訊，可以進行RP演繹，依照情況可能為(細微的聲音)。

#第十條、遇到(海嘯/萬安/BUG狀況)情況，請繼續RP演繹行為。
        ▲  "海嘯：該RP角色發生斷線之相關描述。"
           Extra Rule #11-1：相關上線之人員請繼續RP演繹，無須關黑畫面。
           Extra Rule #11-2：斷線之人員須告知相關劇情人員真實情況。
        ▲  "萬安：伺服器發生發生嚴重狀況，需全體RP角色下線，需全體黑畫面處理。"
           Extra Rule #11-3：直到全部鎮民重整完畢後，統一恢復正常畫面並進行演出。
           Extra Rule #11-4：遇到物品怪異之情節，請與『企劃管理組』進行緊急處理。    
        ▲  "BUG狀況：RP角色遭遇BUG，發生的所有詭異情節。"
           Extra Rule #11-5：請通報『企劃管理組』進行緊急處理。

#第十一條、可以紅燈右轉。

#第十二條、/me指令相關規則。
          Extra Rule #12-1：進行錯誤使用，相關細則於Extra Rule #12-1-1~12-1-4。
          Extra Rule #12-1-1：說謊，記點兩支。
          Extra Rule #12-1-2：呈現可聲音演繹的內容(RRR、www、蛤)，動物無此限制。
          Extra Rule #12-1-3：使用顏文字或符號內容(Owo、A_A、@@、^^、0.0、???)。
          Extra Rule #12-1-4：洗頻。          
          Extra Rule #12-2：可以呈現(受傷/動作/情緒描述)。          
          Extra Rule #12-2-1：受傷描述，代表絕對真實與客觀，幫助拋接球與別人瞭解傷勢。
          Extra Rule #12-2-2：動作描述，代表絕對真實，輔助演繹行為。
          Extra Rule #12-2-3：情緒描述，代表真實，但對方可以選擇是否將情緒部分，拿來進行演繹。
          Extra Rule #12-3：記點採事件制，而非次數。     
         
```

### 〈受傷相關準則〉
```bash
#第一條、所有受傷相關演繹，請使用/me來進行真實描述。
        Extra Rule #13-1：無提供me描述時，醫治人員描述判斷為主。
        Extra Rule #13-2：受傷休息時間，按照當時RP所談論之說法為最高法則。
        Extra Rule #13-3：判斷依序：RP說法>>DC溝通>>私訊溝通。[合理性為主]
        Extra Rule #13-4：發生超能扮演問題，企劃管理組會給予新的受傷解釋。

#第二條、黑白畫面(昏厥)：依照當下情節，決定是否能保留記憶，以合乎當下RP狀況為原則。
         ▲ "昏厥：整個治療行為中，皆處於昏迷不醒狀態"。
        Extra Rule #14-1：侷限性問題，請不要在治療的過程中，進行攻擊行為。
        Extra Rule #14-2：請依照當下能順利演繹RP來進行必須之拋接球行為。
                          Ex："需要自己上病床躺著，必然需要你自己上病床"。

#第三條、黑白畫面(裝暈/裝死/昏迷)，容易觸犯超能扮演，請依照當下情況進行合理性判斷。
        ▲ "裝暈/裝死/昏迷：整個治療行為中，處於假裝昏厥的狀態"。
        Extra Rule #15-1：侷限性問題，演出掙扎送醫等行為，請不要把送醫人員打死。
        Extra Rule #15-2：裝暈或裝死的狀況，事後恢復活動後，仍然需要受傷演繹。
        Extra Rule #15-3：容易觸犯超能扮演，請謹慎使用，依情況合理性與人設狀況來判決。

#第四條、未醫治之受傷，請隨著時間與相關環境思考，是傷口(惡化/恢復)之相關演繹。
        Extra Rule #16-1：(骨折/斷骨/重大受傷)不接受自然恢復，以傷口惡化為基準。
        Extra Rule #16-2：不接受會讓自己隨意死掉的人設，劇情問題當然除外。

#第五條、尊重生命的價值。

```

### 〈劇情相關準則〉

```bash
前言：這裡專門規範『預定劇情』、『設定劇情』、『新增設定』。

『預定劇情』："將有一份相關演員所認同與討論之劇本，並按照該劇本進行後續RP演出。"
            【通常參與人員皆為"正收益"，讓彼此角色塑造"更加強烈"。】      
            
『設定劇情』："為了讓劇本合理所需的鋪陳與設定，又名『安排橋段』"。
            【是補足"未來"的劇情，而非過去或現在的設定。】
            
『新增設定』："為了讓背景或與其相關人員能夠合理存在的設定，又名『Set設定』或『補設定』"。
            【是補足"過去"的設定，而非現在或未來的設定。】

#第一條、『預定劇情』與『設定劇情』無強制力。
         Extra Rule #24-1：絕對強制力的狀況為：這是一個全體RP演員共同參與之劇本。
         Extra Rule #24-2：不能強制未參與之鎮民進行避開之行為。
         Extra Rule #24-2-1：如果有『演出者本身』說出刻意搗亂之言論，則可以進行檢舉。
         Extra Rule #24-3：請勿大肆宣傳。
         Extra Rule #24-4：如需要額外劇情人員幫忙進行跑龍套角色，請參見#26-4-1~2。
         Extra Rule #24-4-1：跑龍套角色皆為『殺青人員』，則無繳交企劃書。
         Extra Rule #24-4-2：跑龍套角色包含『上帝或非殺青人員』，則需要繳交企畫書給予企劃管理組。
         Extra Rule #24-5：整體劇本設定包含6人以上(不包含上帝/殺青人員)，建議填寫企畫書繳交給『企劃管理組』。
         Extra Rule #24-5-1：如未繳交，發生違規合理性之檢舉，會加重懲處記點。
         Extra Rule #24-5-2：如有繳交，企劃組會背書劇情，如有問題則記點企劃組。
         Extra Rule #24-6：如果發生認為Fail RP行為，所有劇本相關人員都會被相關記點。

#第二條、『新增設定』可以於任何時間點進行追加，但請知會『企劃管理組』。
         Extra Rule #25-1：無知會，相關之違規行為，通常有效。
         Extra Rule #25-1-1：如因違規行為而臨時追加之設定，依情節會有4次的寬恕機會。
         Extra Rule #25-1-2：如發生無法透過新增設定解決之違規行為，此條規範無效。
         Extra Rule #25-2：有知會，企劃管理組會與你進行相關討論。
```


### 〈犯罪相關準則〉
```bash
#第一條、一個犯罪出動人數最多6人。
        ▲ "犯罪：該所屬社會規範之明文禁止之行為"。
        Extra Rule #26：聯合犯罪的人數是採取累積制的。
                        ▲ 解釋："一個大型犯罪行動，只要目的與其相關犯罪行為之動機為一致(目的/受益方)。儘管拆解為四場3人犯罪行動，也視為12人，而非當成四場3人犯罪"。
                        Ex："為了支開警力，先行叫3人小隊去進行犯罪，趁空檔，執行真正的犯罪計畫"。
 
#第二條、持槍人士與對抗勢力對峙時，需先進行火力壓制才可正式互相傷害。
        Extra Rule #27-1：情況互換時，此條款也通用。
        Extra Rule #27-2：遠距離狙擊行為，必須要有先警告與暗示，否則不能直接執行。
        Extra Rule #27-2-1：遠距離狙擊手，直接進行(爆頭/斃命)傷害行為，繼續RP演繹，並記點一支至兩支。

#第三條、拯救人員與解救人質關係，由相關職業討論後為基準。
        Extra Rule #28-1：遵從相關職業規章。
        Extra Rule #28-2：如沒有職業規章，採取解救人質為原則。
        Extra Rule #28-2-1：人設原因而放生人質行為，不在此限。
        Extra Rule #28-2-2：人質於搶奪過程，咬舌自盡等自殺行為成功，可視為劇情殺。

#第四條、搜身相關。
        Extra Rule #28-1：犯罪、毒品相關、搶劫相關之地點，可無條件進行搜身。
        Extra Rule #28-2：正常情況，請對方同意才可進行搜身動作。
        Extra Rule #28-2：如果強迫進行搜身，會依照雙方人設合理性進行探討與當下情節為準。
        Extra Rule #28-3：如果違反#28-2，則會事後還原當時所搶劫之物品。
        Extra Rule #28-3-1：如果雙方都沒有VOD佐證，還進行檢舉則雙方記點，並不補償物品損失。
        Extra Rule #28-3-2：如果某單方沒有VOD佐證，則無條件判無VOD演出者輸，儘管其他演員角色有辦法進行相關佐證。此規則套用所有相關檢舉。

#第五條、犯罪份子的犯罪車輛，執行犯罪完成後30分鐘內，不能進行(停入車庫/拆車/賣車)。
        ▲ "基本原則：不能讓車子不見，可以破壞車體，要讓搜查的人看到車，還能知道是誰。"

#第六條、犯罪份子進行特殊金錢交易時，請將相關交易內容至少要放在身上一個RP時間，才可自由使用。
        ▲ "相關交易內容：黑錢以及可賣成黑錢之物品。"
        Ex："假設8點時，獲得黑錢；需要到明日8點時，才可自由使用"。
        
#第七條、犯罪分子進行犯罪後一小時內，不能入住所屬房子內部與提早下線。
        Extra Rule #29-1：如果牽涉到關服時間，請繼續加戲一小時。
        Extra Rule #29-2：最後可犯罪時間，為正常演出結束前半小時，不包含加戲時間。

```

### 〈人設相關準則〉

```bash
#第一條、雙重人格相關。
        Extra Rule #30-1：需給予兩份人設表。
        Extra Rule #30-2：明確制定人格之間關係、人格切換等相關描述。
        Extra Rule #30-3：禁止三重人格以上之人設。
         
#第二條、不允許愉悅犯的出現。

#第三條、不允許性交易之職業為現在主軸的人設。
        Extra Rule #31：可以有性交易背景，但不能以性交易相關職業為開場。
                        Ex："可以說自己以前是妓女，但是現在不是"。
        
#第四條、設定龐大集團背景。
        Extra Rule #32-1：請給予組織的目的、時間軸、相關人員與劇情邏輯。
        Extra Rule #32-2：如發生難以解釋之合理性，相關組織人員各記點。

#第五條、牽涉『歷史事件設定』之人設，企劃管理組需與劇情組討論。
        Extra Rule #33：如企劃管理劇情組發生混淆問題，可以申訴。
        
#第六條、企劃管理組，有權利索求該人設的細節問題。

#第七條、第二人設相關。
        Extra Rule #34-1：第一人設未殺青之前，不能使用第二人設。
        Extra Rule #34-2：第一人設殺青後，不能後續RP中出現。
        Extra Rule #34-3：請勿有以第二人設為出發點，輕易讓第一人設殺青或劇情殺。
        Extra Rule #34-4：需於填寫人設表時，就決定好是否有第二人設，並繳交兩份人設表。
        Extra Rule #34-5：不接受演出期中，追加第二人設之需求。
        Extra Rule #34-6：不接受轉讓第二人設給予其他演出者。
        Extra Rule #34-7：不接受為了讓劇情更精采與完善，而殺青第一人設說法。
        Extra Rule #34-8：會參考演出者相關演出經歷，來決定是否通過第二人設。
        
#第八條、上帝相關。
        Extra Rule #35-1：企劃組/管理組/技術組/支援組，可飾演上帝角色，進行BUG臨時搶救或恢復工作。
        Extra Rule #35-2：上帝飾演跑龍套角色，需要給予企劃管理組進行說明與理由。
        Extra Rule #35-3：上帝新增劇情角色(Ex:派遣警察、醫生等等)，需要事後解釋劇情背景與人設。
        Extra Rule #35-4：上帝是不能與演出鎮民進行深度演繹互動的。
        Extra Rule #35-5：上帝的角色切換，要有一定的合理性。
        Extra Rule #35-6：上帝的角色切換的視角，需要謹言慎行與塑造。
        
#第九條、社會相關。
        Extra Rule #36-1：擔任掌權者會相關人設之規範。
        Extra Rule #36-2：如要新增姓名家門，企劃管理組會與你進行討論。
        Extra Rule #36-3：戲上新增家門姓名，各社會有相關之規範。
        
#第十條、成長性相關。
        Extra Rule #37-1：被檢舉人設相關問題時，因應之前等相關劇情問題，而有角色(立場/心境/個性)之轉變，皆可以提出反駁，而抵銷檢舉。
        Extra Rule #37-2：無VOD佐證，則需要召開相關人員之會議，來談妥成長性之角色變化問題。
        
#第十一條、滅族相關。
          ▲ "發生某社會之某家族，演出人員完全(殺青/劇情殺)之情況，而導致相關社會受到嚴重衝擊"。
            Extra Rule #38-1：允許相關之家族，以背景NPC之角色，來進行輔助演出。
            Extra Rule #38-2：輔助定義：不會進行主動演繹行為，即為此家族之代表而工作。
            Extra Rule #38-3：如發生確切意義上，該家族完全滅門，則不啟用規則#38-1~2。
            Extra Rule #38-4：發生滅族之情況，請第一時間通知『企劃管理組』。
            Extra Rule #38-5：此條款相關規範，會依照當前RP環境狀況，進行隨時修改與新增。
```


## **:pencil:【__其他規章__】-《__邊界條款__》**
```bash
#第一條、本條款全部皆由白鯨『威爾家族』、灰鱷『巴洛家族』與拉瑪『巴奇達魯』勢力掌權者，代表該宗教意願所制定。如違抗與觸犯，其全體鯨落島民應有其共識，並施予譴責與懲罰。

#第二條、各勢力所劃分之區域，皆為該勢力擁有之所有權，不得侵犯其他區域之生活權。

#第三條、每年由三方討論一時間召開邊界大會，來重新擬定邊界條款之內容。
        
#第四條、邊界大會期間，各方代表皆須參加，並於『鯨落的遺骸』進行開會。

#第五條、邊界大會期間，各方代表皆須提出『我方的價值』，來說服其它方對於該方的存在必要性。

#第六條、邊界的調整與契約，於邊界大會後，裁量這一年新的邊界範圍。

#第七條、邊界的跨越，皆可視為『入侵者』的存在，入侵者可由被入侵方任一人射殺。

#第八條、(觸犯/不遵守)任一條款者，經雙方協調無果後，皆流放於『鯨落的遺骸』，並賦予『流放者』之稱號。

#第九條、『流放者』可被任何區域之人，以競標之方式來進行購買，交易人為當前掌管『鯨落的遺骸』為代表。

#第十條、邊界的(劃分/搶奪)，皆以契約來達成，由掌權人簽訂契約。

#第十一條、於邊界發生之鬥爭，皆以所屬相關之區域進行協調。
```
