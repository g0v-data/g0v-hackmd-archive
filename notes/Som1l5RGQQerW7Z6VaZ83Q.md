# 計算機概論
-  電腦基本概念
    - 資訊處理循環(Information process cycle) = IPOS
        - 輸入(input)
        - 處理(processing)
            - 將輸入的資料經過計算、轉換成有意義格式的資訊
        - 輸出(output)
        - 儲存(storage)
- 一位元(bit)是電腦資料能處理的最小單元
- 電腦系統(computer system)主要可分為<font color=red>硬體</font>及<font color=red>軟體</font>兩類
    - 硬體
        - 看的見、摸得道的實體元件及相關裝置
    - 軟體
        - 看不見、摸不到但存在電腦硬體內部，可用指令指揮硬體要做的事，又可分為
            - 系統軟體(system software)- 一連串的程式碼所組成，用來提供硬體基礎架構、基本服務以及控制的核心程式，以便驅使其他程式能正常運作
                - e.g. operation system(OS)-整合並控制電腦內部的功能，提供使用者和電腦互動的使用者介面(UI)
           - 應用軟體(application software)- 以作業系統的功能、服務為架構，藉由電腦的功能來完成使用者特定任務的程式
- 資訊差距
    - 因數位落差引發彼此在資訊吸收能力獲取的能量的差別
- 都市的智慧化程度評斷方式
    - 智慧經濟
    - 智慧治理
    - 智慧環境
    - 智慧居民
    - 智慧交通
    - 智慧生活
