# vTaiwan 個人資料處理政策與同意書

因應 vTaiwan 之後會有的合作，以及舉辦活動上的需求，故有必要根據我們的活動與內容，制定保障參與者與社群成員個人資料的處理政策與同意書。

## 基本原則
- 隱私保護：涉及個人資料與隱私資訊的內容不得公開。
- 告知：在舉辦活動與收集個人資訊前必須進行告知。
- 賦予退出與更正權利：對於個人資訊部分，必須要賦予退出與更正權利。
- 透明：在社群治理與相關決策上盡力遵守透明公開義務。
- 尊重：尊重社群貢獻者與活動參與者的個人權利與意願。
- 降低風險：在應用於人工智慧時，致力降低AI訓練結果的偏誤。
- 社群治理：任何相關政策必須經過社群決定。

## 涉及人工智慧運用的告知責任
### 說明與一般性規範
- 由於 vTaiwan 之後會有一些將意見徵集結果用於人工智慧的運用，故
    - 在運用參與者個人資料之前，必須告知參與者，並賦予參與者更改與退出的責任。
    - 在運用參與者提供的意見之前，必須在活動前告知參與者，賦予參與者退出（opt-out)機制。
### 告知事項：
- 技術來源
- 技術合作平台
- 訓練資料來源
- 利用資料範圍
- 訓練方式
:::info
- 預先訓練，預測序列中可能出現的下一個/幾個字詞。並具一定的彈性，從合理和較不可能的字詞中挑選，產生回覆。（解釋LLM概念）
- 凝聚共識，但不存在完美準確的預測
- LLM 並非資訊資料庫，也不是檢索系統。因為 LLM 接受的訓練是預測下一個可能性最高的字詞，而非取得資訊。大型語言模型可能存在所謂的「幻覺」風險（實驗式民主，促進對話。盡可能運用於共識凝聚，然而目前技術仍可能有其侷限）
:::
- 負責與承諾
- 準確度
- 可能風險
  - 偏誤
  - 不恰當的回覆(False positives / negatives)
  - 對抗性提示造成的安全漏洞


## 活動表單的同意書
**vTaiwan 個人資料蒐集、處理及利用同意書**

為遵守「個人資料保護法」與相關法令規定，並保障當事人之權利，謹依法告知下列事項：
- 機關單位名稱：vTaiwan 社群  
- 蒐集之特定目的：vTaiwan 活動    
- 個人資料之項目：姓名與電子郵件
- 個人資料利用之期間、地區、對象及方式：
    - 期間: 限於vTaiwan 活動期間，以及舉辦後續活動之必要宣傳
    - 地區：台灣
    - 對象：填寫本表單之參與者
    - 方式：蒐集、處理、利用 
        - 蒐集：利用google表單等方式搜集
        - 處理：處理利用資料表方式彙整資訊
        - 利用：用於本次社群活動相關連繫與後續社群活動宣傳
- 依個人資料保護法第3條規定，當事人可行使以下權利：
    - 查詢或請求閱覽。
    - 請求製給複製本。
    - 請求補充或更正。
    - 請求停止蒐集、處理及利用。
    - 請求刪除。
- 依個人資料保護法第14條規定，當事人查詢或請求閱覽個人資料或製給複製本。
- 若遇行使第5及6條權利，請寫信至 vtaiwan.tw@gmail.com，本團隊將依法進行回覆。
- 如果您繼續填寫，代表您已明確知悉並同意 vTaiwan 個資收集同意書
