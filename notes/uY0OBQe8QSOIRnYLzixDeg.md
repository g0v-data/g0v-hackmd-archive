# 主持人

## 行前通知
- 主旨：【g0v Summit 2024】主持人行前通知
- 內文：
:::info
各位主持人好：

感謝各位願意擔任所屬場次的主持人，以下是相關的注意事項。

主持人注意事項：
1. 請在活動開始前十分鐘完成講者 / 工作人員報到並抵達場次所屬場地，如果有任何問題，請回覆本封郵件或傳簡訊至 Peter / 0913080120
2. 在工作人員 / 講者報到後，請至工作人員休息室（VIP室）找 Peter 領取紙本場次議程資料。
3. 各位在所屬場次都會有議程助理協助，惟在有外國講者的場次，可能會需要主持人協助溝通。
4. 主持人在議程開始前，請協助傳達議程資料上的相關説明，包含CoC重申、場地規則重申、Unconf 與閃電講報名重申（該部分會因為議程開始時間不同而有所不同）。相關資訊均會連同議程資料一併提供！
