# 裁判書小幫手

> [點此觀看原始內容](https://g0v.hackpad.tw/AoHSTKneEw7)


現在的裁判書查詢界面使用上有些麻煩，首先是各法院要分開查詢，無法一次查詢全部，查了之後也很難分享給別人，因為連結貼給別人別人也連不進去，因此想做個 Chrome Extension 來解決這個問題

### 技術相關

- 用 Chrome Extension 的作法，可以讓 Chrome 直接去司法院抓網頁內容，不需要有後端 server 。
