# Crowdtribute<br>公民科學與群眾參與計畫<br>案例蒐集與討論

公民科學與群眾參與計畫，是公眾可以自主參加，從過程中認識與學習主題知識，並進一步成為計畫的貢獻者。例如在都市生活週遭中，將所見生物拍照上傳到 iNaturalsit 愛自然網路平台，成為生態研究與環境觀察的共享成果；對於文史主題感興趣的朋友，可以一起來追蹤臺北城城牆石，從當年岩場開採到城牆拆除後石材四散的位置，紀錄臺北都市發展歷程中重要的序章。

由民間持續整理各個計畫的基本資料，已累績超過 200 個計畫，涵蓋「天文、防災與減災、氣象與氣候、環境調查、生物調查、社會文化與生活慣習」等類別，也進一步標記出關鍵設備，例如拍照工具、望遠鏡、物流寄件 ..等；也標記具有特定時節與時段的計畫，例如新年數鳥活動、探討夜間光害的照明偵探團等。

歡迎分享：
1.  你所知道的公民科學案例，或是群眾外包的計畫
2.  你認為初衷類似的計畫
3.  公民科學專論書籍資訊
4.  你認為與公民科學十分相關的領域，例如儀器自造、群眾募資、環境影響評估法的通過、部落自主運動...等等
5.  浮現的討論，例如「科學」與「公民」的認定...

## 浮現的討論

- 「科學」，如何能被稱之為科學？是針對「方法」嗎？不同學門有什麼不同看法嗎？
- 「公民」，與人民、普通人，有什麼不同？
- 「自然觀察」和「調查」兩者的區隔又是什麼？
- 如果說大數據分析，像是蒐集一般人的就醫紀錄來分析健保用藥，這個與公民科學有什麼差別？
- WikicitY  這類以軟體和技術發展出來的群眾發展算嗎？群眾募資平台這類新的服務算嗎？公民科學重視的是案例還是知識論？或是著重在工作方法與流程上的改變？


## Pol.is 意見問答，籌備規劃內容

- Pol.is 可以發起一個關於議題的意見問答 (實際運作方式，可參考 [UberX 自用車載客 pol.is 討論頁面](https://pol.is/3phdex2kjf)、[Airbnb Pol.is 題目蒐集](https://g0v.hackpad.com/vTaiwan-Airbnb--Jqv5FwD48vn))
- ==『你認為怎麼樣才算是「公民科學」？』線上意見表達網址：====[https://pol.is/7nai2k](https://pol.is/7nai2k)==
    - 當您希望提供您的觀點，請以「我認為...」作為文句開頭，謝謝。
    - 提問：
        - 我認爲公民科學與傳統科學並沒有太多的差異，差別只在於特定工作項目的分散化執行而已。
        - 我認為一般人透過參與公民科學計畫，可以更瞭解科學的研究方法。
        - 我認為公民科學除了專業研究之外，重點是脫離專業學術圈的範圍，開放讓更多一般民眾參與。
        - 我認為公民科學的定義，需要從實際執行的計畫經驗中歸納。
        - 我認為公民科學計畫，與自己製造儀器、DIY 工具的風氣有關。
        - 我觀察到似乎各種科學領域，都有所謂的公民科學的計畫。
        - 我認為明確知道自己正在參與科學計畫的非科學家，就是公民科學中的公民。對照大數據 big data 應用中的一般人的資料，當事人不一定知道自己的相關資料或衍伸資料正在進行某個科學研究，這就不同於公民科學的公民。
        - 我認為生物調查雖然有比較嚴謹的科學方法論要求，不過民眾的在地觀察與相應的紀錄資料、佐證素材等，也應該被重視。
