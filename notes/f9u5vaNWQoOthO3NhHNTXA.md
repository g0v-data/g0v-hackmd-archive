## ==必修== 新手村：社群介紹、專案協作心法、協作工具、專案提案入門

| 課程主題 |課程內容 |講師| 修課時數|
| -------- | -------- | -------- | -------- |
| 【基礎概論】<br>科技與社會議題的交會：g0v 如何協作| 多中心化的公民科技社群「g0v 零時政府」如何集結一群人，以公民精神面向社會議題，並結合數位開源的技術，透過專案解決問題？這堂課程將會與你分享 g0v 社群如何從零開始協作，進而生成面向政治、教育、環境、媒體等 818+ 多元領域的專案。|Isabel Hou|2|
| 【基礎概論】<br>開源概論| 「開源」（open source）不僅僅發生在軟體界，乘著開放的概念開始盛行，開源也開始被運用在不同領域的創作，例如開源字體、開源可樂等。究竟開源的意義與影響力為何？我們如何在不同領域運用開源思維？一起來談談開源美好的部分吧！|Teemo|1|
| 【基礎概論】<br>我們與專案的距離|本課程將分成兩個部分，第一部分從專案的性質談起，幫助大家釐清生活中遇到的問題以及可以如何以專案採取行動，並選擇專案執行所需的工具。第二部分則聚焦專案上路後的夥伴、團隊經營與專案永續問題。|崔家瑋／Peter|2|
|【基礎概論】<br>g0v 帶你來提案|「我有一些想法！我想要解決一些問題！我想和一群人一起做事！」，此堂課將會運用 g0v 社群長期以來的提案經驗與架構，藉由引導協助學習者覺察與分析問題，並將想法化為數位開源協作專案。|林怡廷／Tiff|1|
| 【數位協作工具】<br>數位工具手把手教學：HackMD & Slack |掌握數位協作工具使用，讓線上溝通變得更有效率！本課程將會解說 g0v 社群常用協作工具的入門使用方法，學員可以透過數位共筆 HackMD，學習基礎 markdown 語法，熟悉文件協作與整理 。並搭配社群溝通軟體 Slack，讓團隊在遠端工作以及管理多項專案上，能夠更有效構通。|Analeigh|1|
| 【數位協作工具】<br>協作不想吵架？快用版本控制系統！|分組報告的 ppt 或 word 檔都傳來傳去，不知道是不是最新還有誰改了什麼，這樣的哀號時常聽到，那如果專案更大協作的內容更繁雜怎麼辦呢？版本控制系統就是一個很好的解決辦法，本課程將以 git 為例，讓大家認識版本控制的管理精隨。|Denny Huang|1|

|【Wikidata】 只要三步驟，把書籍變成資料集|古人有云：書中自有黃金屋。 但是在數位時代下，各種好用的分析工具面對到紙本的書籍卻不知道從何下手。本次課程將會用三個簡單的方式，使用 Wikidata 這個平台，帶大家將紙本文獻轉成開放資料集。讓你可以快速串聯多方文獻，最大化你的研究成果！|Allen|TBC|
|【[阿美語萌典](https://amis.moedict.tw/)】原住民族語協作-以阿美語萌典為例|阿美語是台灣原住民族語，專案將協作門檻較高的族語，透過程式處理後，成為不用懂族語也可以參與的計畫。|Lafin|1|
