【PyConTW 2021】歡迎索取 g0v 專屬優惠碼
===

PyCon，亦即 Python 年會，是全球 Python 社群的年度盛會。PyCon 由各地同好自發籌辦，而台灣自 2012 年開始也有了自己的 Python 年會。即將邁入第十年的活動歷程，乘載著全台各地自願付出的志工們，為推動台灣程式領域交流持續努力。

📍 PyCon Taiwan 2021 第十屆年會活動時程📍
會議時間：2021/10/02 - 2021/10/03
會議地點：線上進行 
官方網站：http://tw.pycon.org
購票網址：https://pycontw.kktix.cc/events/2021-individual

今年 PyCon 首度推出「社群優惠票」，以期透過優惠的購票方式與台灣各大程式應用開發社群有更多互動，也希望藉此聚集台灣各領域應用 Python 的組織與個人，交流彼此的程式使用經驗，分享 Python 最新的技術發展。

**PyCon 本次提供「g0v零時政府」 30 組的「社群優惠票」，社群夥伴能以優惠價格 TWD 1,200 購買今年度 PyConTW 線上門票＋Pysafe 禮包。** （一般票原價 TWD 1,800）PySafe 是今年的大會小禮，內含防疫物資、紀念 T-Shirt、紀念酒杯及小貼紙。
