<span class="s1">村上春树（日语：村上 春樹／むらかみ はるき Murakami Haruki ，1949年1月12日－），日本小说家、美国文学翻译家。热爱音乐。</span>

*   <span class="s1">29岁开始写作，第一部作品《且听风吟》，即获得日本群像新人奖</span>
*   <span class="s1">1987年第五部长篇小说《挪威的森林》在日本畅销四百万册，广泛引起“村上现象”[1]。</span>
*   <span class="s1">村上春树的作品写作风格深受欧美作家影响的轻盈基调，少有日本战后阴郁沉重的文字气息。被称作第一个纯正的“二战后时期作家”[2]，并誉为日本1980年代的文学旗手。</span>

<span class="s1">简体版封面及书名</span>

*   <span class="s1">村上在這本書《關於跑步，我說的其實是……》中藉著記錄整理自己的經驗與思緒，分享他在跑步中所理解的人生以及寫作觀。至於書名是什麼意思？其實這本書的日文原名就是以外來語思維所寫的（What I Talk About When I Talk About Running），這個名稱明顯是向他喜愛的美國小說家瑞蒙．卡佛名作之一《當我們討論愛情》（What We Talk About When We Talk About Love）致敬。（博客来 台湾）</span>
