## 海外軌：美國版天龍特工地 (OpenSTL the St.Louis Code for America Brigade)
* 定義空屋的問題：導致區域變得殘敗的原因
    * 有一座空屋出現，沒有被定期維持，不是只有導致人們不想買，也會因為隔壁是空屋所以其他人更不想買，會像病毒一樣影響整個社區
    * 如果沒有人住在這個社區，就會沒有人支付稅金，政府沒有錢可以去維護路、去修整整個區域，漸漸擴散開到整個區域
    * 整個區域對人的印象就會非常的差，對政府來說維護變得非常昂貴
* 我們怎麼開始的
    * 我們有一個國慶 civic hacking
    * 市長說請用 data 做一些事情
    * 這些資料非常不好，很多欄位都沒有弄好
    * 我們有好幾個政府部門跟這個議題有關
    * 大概一年後，我和另外兩個人（政府方），我們透過地址去找到這些空屋的擁有者，才能終究把資料串在一起
    * 政府資料對不起來，搞不清楚到底哪裡有空屋哪裡沒有，無法掌握真正的狀況
    * 我們了解政府有太多官僚所以我們沒有找政府，我們只跟他們拿資料，主要是找社群
