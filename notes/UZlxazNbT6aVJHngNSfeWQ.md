# 工作內容

我們的工作就是透過IT技術的導入，建構適合旅遊產業完整解決方案的平台服務，並應用在美食、住宿、遊程、購物、交通、票券產品上，建立標凖溝通機制，帶全世界的人到世界各地遊玩。

我們鼓勵工程師嘗試新的技術，較為資淺的工程師會有人帶領輔導，讓你逐步學習、功能規劃、系統架構設計等，快速累積專業技能，寫出可持續維護的高品質程式碼。

## 這個職缺，我們希望你/妳是：

1. 孰悉PHP/MySQL程式語言，能獨立開發作業者
2. 具備良好的Coding習慣及對新技術抱有好奇心
3. 具備基本正規化之知識，了解資料庫設計
4. 具備串接及整合不同第三方系統服務的能力
5. 具備程式追蹤除錯的能力
6. 願意改善或透過工具來使自己的工作流程更有效率
7. 擅長溝通協調、團隊合作、態度積極一同達成目標
8. 了解資料結構及演算法


## 這個職缺，主要的工作內容有：

- 串接第三方系統API服務
- 設計及開發系統API服務
- 設計及開發旅遊電子商務網站管理後台
- 協助提升平台效能與反應速度
- 透過數據分析找出正確的開發方向

## 加分：
- 熟Linux Nginx MySQL PHP (LNMP) 架構
- 曾使用 Laravel 框架開發相關經驗
- 熟悉 CI/CD 流程，有相關經驗
- 熟任一NoSQL，ElasticSearch
