# 鄉民關心你 常見問題

> [點此觀看原始內容](https://g0v.hackpad.tw/nvw5cFwVmab)


### 所以這是什麼？


一個企圖處理蒐集，整理，行動，串連，追蹤的群眾參與社會議題的平台。

是串接事件和人的媒介，是對話的一個窗口，
它存在的目的，最基礎是備忘，其次是串聯和關注，
更積極的是通往後續的「行動」，整套都接通了，才是鄉民關心你的全貌。

### 感覺使用上很複雜，這會有人要用嗎？


瞭解社會議題本來就是一件很耗時間的事，
你一定需要花時間整理公文，新聞，Blog 文章等等。

而在瞭解後即是採取行動，行動後還要追蹤結果。
解決社會問題本質上就是這麼複雜，本質上就是囉唆。

 天底下沒有一蹴可及的事。

 你若不自己思考整理而只依賴懶人包的資訊，
 那懶人包的作者就會幫你決定你的立場。
 而你，就只是個棋子。

 世界上沒有絕對客觀，但讓我們更接近的方法之一，
 是花時間研究吸收多方說法，以番茄取代番茄醬。

 我們相信每個人都不僅此而已，
 如果你願意，鄉民關心你樂於提供軍火。

### 為什麼不用 Evernote 當作儲存媒介？


也許你會想說可以用 Evernote 來蒐集網頁啊!
然後用它的  API  來兜網站, 這樣很多東西就不用自己做了。

嗯... 不過... 我們不想這樣。

因為 Google Reader 的退出，
我們確信少數保證不會無預期消失的方式，就是開放源碼。

### 我還沒爬過文，不過這好像跟....某專案有重疊ㄟ....


人生最有趣的就是這個"好像"，
從8萬英呎往下看，人跟豬看起來都很像。

似曾相識或許是因為真有此需求，因為所見略同，
這是個自由國度，就找自己順眼的專案加入吧，未必要是這裡。

### 我發現這個專案跟我想的不太一樣，不是我想做的。


你隨時可以退出，不需要感到抱歉。

重點不是這個專案想做什麼，
重點是你想做什麼,
你想解決什麼問題。

### 為什麼這個專案用的東西我幾乎都沒聽過？


因為有趣，因為愛... 為了蔚藍清淨的地球，
因為學習永不止息。
