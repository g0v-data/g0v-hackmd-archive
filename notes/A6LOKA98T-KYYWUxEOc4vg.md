# COVID-19: How Can I Help? 我如何提供帮助？

## Task Tracker for COVID-19 Projects | COVID-19 项目的任务跟踪器 

Objective of this document: help people that want to help, quickly find a way to help. 

本文档的目标：帮助想要帮助的人，快速找到帮助的方法。

I believe this objective can be achieved through task tracking, or project management. This is also an experiment by me to help track g0v tasks in general. 

我相信可以通过任务跟踪或项目管理来实现此目标。 这也是我的一项实验，旨在帮助总体上跟踪g0v任务。

If you have a project, feel free to add it! I have added several projects as an formatting example. Feel free to modify if you have better ideas, or if you think your project needs a different format. 

如果您有一個項目，請隨時添加！ 我添加了幾個項目作為格式示例。 如果您有更好的主意，或者您認為您的項目需要其他格式，請隨時進行修改。
