# g0v 影響力問卷調查結果

## 前言
「g0v.tw 台灣零時政府」是草根式的公民社群，致力於加深公民對社會的貢獻以及彼此間的連結。透過 g0v 社群，你可以在這裡尋找志同道合的夥伴，以草根的方式**實踐你的理念**，並**將成果以開放授權模式釋出**，讓更多的人可以站在你的成果上接力賽跑。

初次認識 g0v 的新參者，通常會難以理解「社群」的概念，g0v 不是一個非營利組織，也不是法人，最概括的說法是「一群貢獻自身能力來改變社會的人」，不只是「想改變」，而是「**真的造成改變**」。

g0v 社群開創出台灣公民社會與政府互動的新模式，讓台灣公民有增加影響政治決策的手段，在國際上曾獲「**2018 林茲電子藝術大獎**」電子社群優異獎，並與歐洲開放知識基金會(Open Knowledge International)、美國 Code for America 等著名公民科技社群並駕齊驅，為**全球前三大開放公民科技社群之一**。

為了量化「這些改變」，在 g0v 社群籌辦雙月黑客松的工作小組「g0v 揪松團」，依照樹冠影響力投資執行長 <u>楊家彥老師</u> 提供的影響力地圖理論，設計問卷並回收分析，首度開展出 **g0v 台灣零時政府的影響力地圖**。

## 問卷設計
問卷填答自 2021/1/23 g0v 第肆拾貳次黑客松開始，持續開放社群參與者填答，透過不同的 g0v 實體活動、g0v 線上社群平台（Slack、FB 等）以及 g0v 揪松團電子報，邀請 g0v 參與者填答，截至 2021/3/18，我們共收穫 86 份問卷回覆。

在問卷中，我們依照**參與者的參與程度**，大致可以分類為「參與者」、「志工參與者」、「專案參與者」，不同身分將會填寫不同的問題題組。依照**參與者的影響力層面**，分為個人層面的「經濟財務價值」、「人際價值」、「感性價值」、「技能價值」，以及對社會影響層面的「社會價值」、「文化價值」、「環境價值」等進行答案歸類。

# 2021.01.23 - 2021.03.18 問卷結果整理與分析

## 我們的發現
- g0v 社群參與者在社群中達到高程度的**人際互動滿足**和**社交練習**。
- 比起自我技能的提升，對於參與者而言，**貢獻社會的情感滿足**更能在 g0v 社群實現。
- g0v 社群的不可替代性在於「**跨界專案的實踐力**」，不只是提出構想，而是能讓**想像落地實現**，成為改變社會的專案。

## 問卷分析
![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_8370a6d028708fa3931fa9352e7a6091.png)
g0v 社群的傳播管道主要為「口耳相傳」以及「線上社群平台」，由曾經參與過社群的參與者，藉由聊天或分享，將 g0v 社群讓更多人知道並參與。

第二大類為「特定專案或 NGO 」，可得知或許某些專案參與者一開始並不認識 g0v，而是透過參與專案後，才漸漸了解並踏入社群。

最後值得注意的是，g0v 社群資深貢獻者唐鳳，在採訪或專案中持續協助 g0v 曝光，她的宣傳效果甚至勝過電子新聞的報導。

**個人層面影響力**的調查結果，可以知道 g0v 社群參與者，有 6 成贊同能夠在社群「<font color = brown>提升對社會議題的了解程度</font>」，並且肯定 g0v 社群提供的感性價值。另一方面，從影響力價值層面來看，11 個選項中，關於<font color = brown>人際價值</font>的選項重複被選擇，長期社群參與者總說：「**有很多人在這裡找到共度人生的夥伴。**」問卷結果剛好可以印證這個觀點。不只是人生伴侶，長期專案的夥伴，也有很大的機率是從坐在同一張桌子吃炸雞開始。

**社會層面影響力**的調查結果，能看見有高比例的參與者認同 g0v 專案在「<font color = brown>協作方法與開源理念的推廣</font>」和「<font color = brown>增加跨界合作與專案發展的行動力</font>」，對社會產生影響。藉由 g0v 常用的協作工具，能夠降低專案跨界的門檻，讓大家用同一套行為準則工作，減少無效溝通，增加專案成功的比例。

而也有過半數的專案參與者認為，g0v 專案達到「<font color = brown>促進社會溝通與討論的風氣</font>」，用許多議題光譜工具，來讓參與者可以就議題的歧異點正反辯論與理解。

深入探討每個選項的影響程度，可以看見 **g0v 的獨特性與優勢**，除了前段提及的影響，在「<font color = brown>台灣國際交流的能見度</font>」提升方面，許多專案為此付出了努力，例如與國外參與者合作專案以及參加國際研討會等。而在社群媒體不實資訊紛飛的年代，g0v 社群致力於「<font color = brown>良好的議題討論與媒體識讀</font>」。g0v 活動參與者必須遵守 [g0v 行為守則](https://g0v.hackmd.io/s/COC)，無論在網路上或是實體活動，彼此用尊重包容的態度對話，許多社群專案在設立之初，也蘊含著媒體識讀，以及異溫層友善溝通的願景。

*<font color="gray">「[g0v 國際交流工作小組](https://g0v.hackmd.io/@chihao/rJmYFi0YS)」與日本、韓國、香港社群定期共同辦理「面海松」，並彙整國際交流、工作坊與會議資訊，讓社群個人參與者或專案看見國際公民科技發展與趨勢，並讓社群個人參與者和專案有機會將 g0v 社群推廣出去，將國際新知與交流成果帶回台灣。*

*在 COVID-19 疫情爆發時，g0v 社群的[防疫專案](https://g0v.hackmd.io/@kiang/mask-info)如雨後春筍般萌芽，從一位工程師做的「超商口罩地圖」開始，引發一連串的公私協力，以及更多的防疫應用，例如：「患者足跡追蹤」、「人潮熱點顯示」等，讓國際看見台灣，驚艷不已。*

*「[Cofacts 真的假的](https://cofacts.g0v.tw/)」Line Bot 闢謠機器人，最初是為了幫家中長輩辨別網路養生文章的真實性，吸引許多志工編輯，進行工人查證後，在人工回傳到手機用戶的 Line 中，目前趨勢科技、Mygopen 等，都有串接 Cofacts 的後台資料庫。「0archive 零時檔案局」透過追蹤不同網路媒體或社群平台文章的發布時間與路徑，辨別出互有關連，或別有用心的誇大資訊。</font>*

## 影響力地圖的下一步

### <font color = brown>坑主訓練</font>

為了讓來提案的坑主，增加運作專案的技能，g0v 揪松團會於 2021 年舉行**坑主小聚**、以及**坑主引導師工作坊**，讓坑主不只有提案，更有引導眾人「做出答案」的能力。

### <font color = brown>國際曝光</font>
g0v 社群持續於國際媒體及國際公民科技社群曝光，g0v 揪松團召募 2-3 位志工與實習生，為 [g0v.tw 官網](https://g0v.tw)、[揪松網](https://jothon.g0v.tw)和推廣文宣，進行**英文翻譯**，降低國際參與者參與 g0v 社群的門檻，期待能夠**複製 g0v 社群經驗並推廣到全世界**。

### <font color = brown>新手友善</font>
參加 g0v 黑客松的參與者中，約有 4 成是第一次接觸 g0v 社群活動的新手參與者，在 2021 年，對於「新手友善」的課題，預計會由社群基礎建設的「**新手村專案**」持續推動，包含修改 g0v 官網、建立新手入村的流程小遊戲、新手導覽團等，讓社群如同涓涓細流，生生不息。
