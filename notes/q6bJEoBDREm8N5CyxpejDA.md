3【資料與應用】
- 樹木監測 / 感測 / 硬體裝置

- 樹群推論 / 樹覆率 / 綠覆率 / NDVI / 森林損失計算 / 街景綠視率 / 樹高
- 潛力用地推論 / 用地盤點 / 認養業務窗口 / 必要資料或 API
- 用地與現地調查 / 踏查輔助工具
- 基地配置演算法參考
- 環境營造溝通討論工具 / 營造歷程呈現與責信工具
- 綠化-樹木-樹群 效益評估與增進對策 / 軟體工具 / 研究與論文
- 苗圃位置資訊 / 苗木申請窗口
- 植物園 / 保種機構 / 技術機構 / 環境教育場所 / 林場

- CSR 企業社會責任與種樹行動
- 異業合作 / 跨領域業務議題結合種樹 / 賽事會展產業連動 / 社會福利政策與就業方案
- 已知籌備企劃推動中的用地
- 外交儀式植樹
- 公部門提供認養養護的機制 / 長期認養之指標個案
