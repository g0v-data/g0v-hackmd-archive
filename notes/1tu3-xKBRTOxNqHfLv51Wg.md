# 給非工程師的 GitHub 使用課程

可講內容
- Git 基本介紹
    - 了解版本控制的概念
    - commit, pull, push, diff, blame
    - commit log best pratice
    - branch
- issue
    - 介紹專案開發管理流程
    - issue 流程 open => 處理 => close (invalid, duplicate, wontfix)
- fork, Pull Request
    - 介紹開源的複製模式
- github page
    - 可以自行架設靜態內容，包含網頁、文字、圖片...
- gist
    - 可以丟些簡單的小文件
- data preview
    - CSV, GeoJSON 等預覽


可用範例
- 不要用程式碼當範例，可以用法律或是寫劇本當作範例
