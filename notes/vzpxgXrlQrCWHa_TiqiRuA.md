## **開源（Open Source）** 

**開源（Open Source）** 也就是將專案內容（可能是程式碼、文字、討論等）以[開源授權（Open Source license）](https://opensource.org/licenses)或[創用CC 授權（Creative Common License / CC License）](https://cc.ocf.tw/%E9%97%9C%E6%96%BC%E5%89%B5%E7%94%A8-cc)等方式公開出來，讓其他人能夠再次利用或直接參與該專案內容。

常見的情境像是軟體工程師在 GitHub 上面看到某個專案（套件），只要遵照該專案的開源授權，就可以將該專案的原始碼拿來直接使用，也同時因為原始碼公開，你也可以直接協助改善、改作該專案。

或是你在做投影片時，想要放一隻可愛的貓貓圖片，但是你沒有，你可以上網搜尋有 CC 授權的圖片，依照作者的授權，你就可以使用那張圖片而不用擔心侵權。

以及近幾年大家常使用的字型 — 思源黑體也是使用開源授權，所以你可以自由的使用在各個地方，也有基於思源黑體的授權與原始碼而產生的思源柔黑體與思源宋體等字型。

因為專案或作品本身開源了，所以人們可以更自由的去重複使用、延伸、應用現有的內容。


## **開源精神**

**開源精神**到底是什麼呢？前面提到的都是具體的行為，那為什麼大家會希望推廣開源這個做事方法呢？

開源精神最核心的永遠是人，透過 **「分享、參與、行動」**，人們將自己的創作、專案分享出來，透過分享尋找因為共同的信念、喜好聚集的同好；透過參與專案，去產生理念與技術的激盪，甚至從中尋求新的刺激與討論；藉由行動去改變現況、推廣你想分享給其他人的事物或理念。

而且在合力工作的過程中，我們的能力會愈來愈好，能做的事也愈來愈多，因為每個人擁有不同的能力、不同的專長，彼此截長補短，就能達成一個人時做不到的事情。當有人對你的想法、專案有興趣也能自主加入，或是協助改善。同樣的，你也可以接續其他人的想法，也或者其他人能夠接續你的想法繼續創作與延伸。
