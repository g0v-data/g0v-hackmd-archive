제2회 코드포코리아 컨퍼런스
The 2nd Code for Korea Conference

안녕하세요. IT기술로 사회문제를 해결하려는 시민들의 네트워크, 코드포코리아입니다.
Hello. This is Code for Korea, a network of citizens trying to solve social problems with IT technology.

2023년 10월, 국내 시빅테크(Civic tech) 분야 최대 행사인 코드포코리아 컨퍼런스가
개최됩니다!
In October 2023, the Code for Korea Conference, the largest event in the civic tech field in Korea, will be held. It will be held!

시빅테크, 시빅해킹 알아보기([링크](https://ko.wikipedia.org/wiki/%EC%8B%9C%EB%B9%85_%ED%95%B4%ED%82%B9))
Learn about civic tech and civic hacking (Link)

제2회 코드포코리아 컨퍼런스의 슬로건은 “Happy Happy, Hack Gathering! 모여봐요 즐거운 시빅해커즈"입니다.
The slogan of the 2nd Code for Korea Conference is “Happy Happy, Hack Gathering! Let’s get together and have fun, Civic Hackers.”

국내외에서 활동하고 있는 시빅해커들이 한 데 모여 ‘네트워킹을 해보자'는 주제입니다.
The topic is ‘Let’s network’ as civic hackers active at home and abroad gather together.

대만 최대의 시빅해킹 모임인 ‘g0v(거브제로)’, 부산 지역의 시빅해킹 모임인 ‘코드포 부산' 등 다양한 곳에서 시빅해커들이 모일 예정입니다.
In various places such as ‘g0v (gov-zero)’, Taiwan’s largest civic hacking group, and ‘Code for Busan’, a civic hacking group in Busan, civic hackers are planning to gather.

또한 2023년 한 해 동안 코드포코리아가 추진하고 있는 #지역화폐 #산업재해 #기후위기 #페미니즘단어사전 등의 프로젝트를 소개하고, 진행 상황을 공유합니다.
In addition, Code for Korea is promoting #LocalCurrency #IndustrialDisaster #ClimateCrisis #Feminism throughout 2023. We introduce projects such as these and share the progress.

제2회 코드포코리아 컨퍼런스는 10월 27일 금요일 오후, 국립중앙도서관 국제회의장(1층)에서
열립니다.
The 2nd Code for Korea Conference will be held on Friday afternoon, October 27th, at the National Library of Korea International Conference Hall (1st floor).

- 행사 일시 : 2023년 10월 27일 (금요일) 13:00 ~ 18:00 
- 행사 장소 : 서울 서초구 반포대로 201, 국립중앙도서관 국제회의장 (1층)
