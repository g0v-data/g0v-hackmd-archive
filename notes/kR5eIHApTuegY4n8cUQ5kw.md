# 智慧國土-規劃治理交流會

> [點此觀看原始內容](https://g0v.hackpad.tw/qDA1GlkqLZ1)


### 分享題綱：

1.    您認為國內在推行智慧的規劃與治理方面所遭遇到的困難與瓶頸（可從人才、政策、產業或其他層面）？
2.    您認為如何透過先進科技科技協助政府提供更好的規劃與治理方式，讓有限的資源能夠有效的分配，而國發會再這個層面上能夠做怎樣的推動或措施。
3.    您認為再推動智慧的規劃與治理方方式時，民間單位與政府部門如何攜手改變現行的方式。
4.    請您分享世界各國如何透過先進的科技協助政府提供更好的規劃與治理的方式。

### 社群提問：

by jimmy
應用相關問題：
- 對於不斷發展的「國土」相關產業，缺乏地理資訊應用的整體指導方針，未來智慧起來要如何管理及分析現況？舉例而言，像農委會提供補助種電的太陽能板，完全無法知道地理座標和面積，僅能從空照圖和地號略之所在位置，現在已經出現無耕作的農地亂種電申請補助，政府卻無力[視察](http://www.newsmarket.com.tw/land-factory/chapter-03/)，更無法計算其政策之光電效益。若已經有無人機、衛星等科技，為何不搭配整合應用？
- 先進科技應該輔助現有法規執法、評估風險，但看到政府有許多案例，空拍看[燒稻草](http://news.cts.com.tw/nownews/society/201506/201506221627896.html#.VZIRG_mqpBc)、砂石濫墾、[水土保持](https://g0v.hackpad.tw/FTBOsiElsPY)，甚至土石流等，但智慧國土監測，卻無與**環境影響評估機制連結**。這等於智慧的很智慧，但是真正要蓋的時候，卻矇著眼睛不看研究、不看數據，還是一樣照蓋、照推不知道有沒有智慧的國土利用。這真的有效果嗎？
- 衛星圖應用為例，國際先進的技術，可以在空拍後30分鐘內完成校正釋出，供災害管理、環境評估等應用。福衛的空拍每天都有，但圖資卻需要層層關卡，一天至數天，耗費人力後才有辦法出圖，做不到像 [Tomnod](http://www.digitalglobeblog.com/2015/04/26/digitalglobe-opens-access-to-satellite-data-to-support-disaster-response-efforts-in-nepal/) 的 Crowdsourcing 應用。若基礎圖資取得的技術是如此老舊，又要如何智慧起來？

開放（open data / open source ）相關問題：
- 民間單位合作，若有政府委外招標的工作成果，若無機密資訊，理當以公共性、可再利用為考量。例如地理資訊系統是否採用 Open Source 軟體為基礎進行開發，以利下個廠商、甚至民間，都有機會在有限資源下利用？

先進科技應用於國土規劃議題時，應該要有什麼樣的態度？
- 應以國土分區作為基礎
    - 參考[〈國土的治理與挑戰〉作者：廖本全｜整理：許暐咏](https://www.facebook.com/notes/%E5%85%A8%E5%9C%8B%E5%8D%80%E5%9F%9F%E8%A8%88%E7%95%AB/%E6%96%87%E7%AB%A0%E8%BD%89%E8%B2%BC%E5%9C%8B%E5%9C%9F%E7%9A%84%E6%B2%BB%E7%90%86%E8%88%87%E6%8C%91%E6%88%B0%E4%BD%9C%E8%80%85%E5%BB%96%E6%9C%AC%E5%85%A8%E6%95%B4%E7%90%86%E8%A8%B1%E6%9A%90%E5%92%8F/850437304988759)
    - 原住民基本法，行政程序，區域計畫，國家公園，第四次森林調查與林地分級...
- 於「所有權與開發權」之外，健全「規劃審議權與資訊公開權」的細緻作法，強化空間四權的均衡。
- 回應其他部門所面對的未來趨勢
    - 例[中華民國人口推計（103至150年）國發會](http://www.ndc.gov.tw/Content_List.aspx?n=84223C65B6F94D72)
    - 例回應氣候變遷調適與制裁，國內調適綱領
    - 例公民參與國土治理，各種層次的銜接，行政程序，公民科學，計畫合作，行政延伸
- 創造新的「規劃思維」
    - 減災整備，社會與環境脆弱度評估（方法與案例）、pre-disaster planning
    - 原民會，傳統領域工作坊
    - Food system planning, 大費城經驗、聯合國規劃方法
    - 足跡規劃，連江縣、蘭嶼
    - 關於污染
    - 溫減
    - 流域治理
    - 海岸與近海
    - 產業的成本效益評估，涵蓋外部成本，辯護式規劃，專家證人
    - 產業現況，園區，上下游，締約與產業衝擊分析資料
