# Linear Independence
:pencil: $Definition-Independence$
Let V be a vector space over F. A nonempty subset S of V is said to be linearly dependent if there exist distinct vectors $v_1,...,v_n$ in S and scalars $\alpha_1,...,\alpha_n$ in F, not all of which are zero, such that:
$$\alpha_1v_1+\alpha_2v_2+...+\alpha_nv_n=0$$
A set which is not linearly dependent is called linearly independent.
If S=$\{v_1,v_2,...,v_n\}$, then we say that $v_1...v_n$ are linear dep/indep.
