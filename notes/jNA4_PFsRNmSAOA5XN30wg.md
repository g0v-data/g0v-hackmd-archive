# （坑）如何監督航空城開發

> [點此觀看原始內容](https://g0v.hackpad.tw/kl9DLRK16dV)

/*
轉呂先生(去年新聞人物呂阿雲老翁小兒子)的訊息。因為航空城開發利益錯蹤複雜，本人想拋出如何監督這件事情的可能方法！
\-\-\-\-
[https://www.facebook.com/wordfisher/posts/10152628686333738](https://www.facebook.com/wordfisher/posts/10152628686333738)
關係到我們全台灣納稅人的「航空城」！
 航空城開發案即將在本月底通過了！這個全台有史以來最大的開發案，在我自己看來似乎只有我們這些被徴收戶、炒作的官商在乎！或許我是錯的！
 航空城預計花5500億人民的納稅錢！5500億如果一億給一位小老百性，可以造就5500位億萬富翁！如果反面思考，以葉世文貪污2600萬，航空城可以造就2萬200個葉世文！！這麼大的一塊肥肉晾在陽光下，真的只是為那一條跑道嗎？當然沒這麼簡單！
  小時候我的夢想是去台北打天下！後來我在台北上班後，我的夢想是去美國發展！美國迪士尼學校畢業後，我的夢想是回家鄕打造我自己的動畫王國！這40多年 來，菓林村一直都沒有太大變化，除了近幾年空軍機場搬走變安靜許多，再來就是機場捷運缐快要開通了，我們從小到大期待的城市發展似乎慢慢形成！但近幾年這 些官員高喊航空城，一下子從擴增跑道、航廈，突然間整個大園瞬間被吞沒！原本以為我們因為航空城的發展"得求了"！家鄉變城市，但誰知道我們生活一輩子的 地方要發展起來了！但我們在地人享受不到，慎致要把我們趕到土地最差最不值錢，房屋仲介絕對不要的墳墓區！好死不死就在阿公阿媽的祖墳內！這一切真是太荒 唐了！但這就是真真實實的台灣政府！！
 相信全國人民大家都希望我們有一流的國際機場！但如果你知道花這麼多錢，把整個大園人都趕走，一切移為㔻地！我們只是多了一條跑道！多了一個航廈！然而第一、第二航廈一様沒有一個整體性、沒有一個國際水準的設計性！你還會這麼期待航空城嗎？花這麼多錢到底進了誰口袋呢？
  我們當然希望國際機場有世界水準，但這餅劃這麼大，而最後成就只是一條跑道和第三航廈？！那來的世界水準？！但如果全國民衆團結起來監督這個開發案！監督 政府！或許我們能夠擁有一個全球性的國際機場！！航空城開發案即將在本月底通過了！一但通過就很難去監督它了！「航空城」絕對不止是大園人的事！航空城是 台灣每一個人的事！文/呂文忠
*/

音樂、藝術可以改變世界！
 1994年的我服務於上華唱片，每天與公司員工為歌手做平面設計、記者會佈置，記憶中我們公司歌手有齊秦、許茹芸、動力火車、李翊君、裘海正、熊天平、許美靜等等，那是一段非常令人懷念的時光！
  2004年在美國迪士尼的學校（calarts)，開始與學校音樂系學生一起製作我的第一部手繪動畫「鞋子」，在從來沒有合作過的情況下，驚覺音樂的力量 無限浩大！它把一個平淡無奇的影像付予新生命！真的非常神奇！並且譲我在大學一年級就入圍學生奧斯卡金像獎，在大一就和奧斯卡扯上関係，只能説自己的運氣 很好！
 ”鞋子“
[https://www.youtube.com/watch?v=dhuYNRtT7Y4&list=PL3FAE984ADB56FA8A](https://www.youtube.com/watch?v=dhuYNRtT7Y4&list=PL3FAE984ADB56FA8A)
 2012年底房子剛蓋好，把所有欠的款項付給營造廠後，心想這下終於可以好好的闖事業，雖然身上有龐大貸款壓力，但只要心??的信念不變，總有一天一定能夠還清的！
  但就在完工的第一個月，突然接到通知家園被航空城劃為公園和道路預定地，那一陣子，每天都在煩惱如何面對這突來的徵收案！晚上睡也睡不好，吃也吃不下，惱 子??想的都是如何解決這個突來的厄號。父親問我知不知道要被徵收？我當然知道呀！這麼重大的事怎麼可能不知道呢？我也不知道該如何是好，但也只能安慰他 老人家不要擔心，晚年還要面對這第三次徵收家園，諒誰也不忍讓他擔心。但誰也想不到11月9日航空城抗爭日，他竟然選擇這樣的方式離開人世，留下我們沒有 支字片語，唯有許許多多的自責與不捨。接下來這一連串的抗爭惡夢才真正開始，我看見人性與政府最醜陋的一面和人世間最溫暖一面～～。
  我們很慶幸目前有這些公益團體在幫助我們，但每個人的能力有限，在此時此刻我們需要「音樂創作人才」、「舞蹈」、「戲劇表演」、「影像創作」、「藝術」各 界的支援，把這一連串不公不義運用藝術表現出來，讓社會大衆更多人関注這個事件！由於自己是念加州藝術學院（迪士尼學校），対於藝術充滿無限的愛好！父母 親用賣菜錢供我去美國念藝術，我也希望能夠運用藝術使這塊土地變的更加美好！更有文化藝術特質！而不是一連串的醜陋～～，我們真的很需要藝文界的您協助！ 如果您知道有誰可以幫助我們，請您協助轉貼！謝謝！文/呂文忠
