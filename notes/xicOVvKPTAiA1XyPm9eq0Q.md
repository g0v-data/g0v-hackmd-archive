# 開放當責的「數位治理」

促進 (1) 公民社會 **不特定人 / 群體** (2) 專業者 (3) 政府 **機關**
之間 **有根據的 / 合理的** 信任關係

--> 開放政府、科學研究⋯ <-- multistakeholder approach

Action items
- 政府機關持續與公民社會、學者專家對話
    - 過程、諮詢對象應公開
- 政府積極開放資料
    - 協力修訂開放資料指引
    - 另可一併考慮涉及公共利益之企業、民間組織
- 國家積極保護個人資料自主
    - 保護人身安全、基本自由、隱私
    - 保護個人不受科技平台操弄
- 推動資料驅動的學術研究
    - 應符合科學原則、同儕審查
- 推動研究驅動的政策及法律制定
    - 多方利害關係人協力機制
    - 過程應公開
- 推動資訊判讀教育
- 確保基於資料及科學的公眾溝通
- 確保多元涵容、利於公共討論的資訊環境
- 開放政府行動方案、開放國會行動方案監督機制
- 共同維持公民社會動能
- ...

確保多元涵容、利於公共討論的資訊環境 [IORG]

- 國家積極保護個人資料自主，降低政府監控及科技操弄
- 政府及科技平台積極開放資料
- 推動資料驅動的學術研究
- 推動研究驅動的政策及法律制定
- 推動資訊判讀教育
- 確保基於資料及科學的公眾溝通

持續東亞各地公民黑客交流協作 [g0v]

- 各地開放政府、數位治理案例分享、實作技術交流
    - 口罩地圖
    - 開放資料進用
    - 多方利害關係人對話協作機制
- 各地數位治理相關法規協調交流
