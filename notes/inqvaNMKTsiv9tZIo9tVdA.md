在加入前，有些團隊的核心原則請你了解

- 任何人皆可參加
- 本專案無時程壓力，任何東西， It's done when it's done!
- 我們理解各人都有自己的人生規劃，你隨時可退出，或是跟大家說想暫時休息，也隨時歡迎你回來
- 不要因為做這個專案影響到原本的生活
- Just for fun :)

### 本專案運作風格


- 先求有，再求好
- 想要什麼，先做一點出來，別人才好幫忙
- 不需要等待別人丟任務給你，從自己的需求想起，從自己要用的東西做起, 如果有人起步較早，不妨加入他，如果這東西沒有人做, 那你就來做吧!
- 做事的人最大，沒有人是頭, 不需要向他問可不可以... 就直接做吧!  真的有爭議或問題我們再來協調
> 當然還是要考慮一下法律問題
> [name=HisnYi C]

- 最重要的, open source , open mind :)

### 現在


- 加入 [g0v](http://hack.g0v.tw/people) [鄉民關心你](http://hack.g0v.tw/people)[hub](http://hack.g0v.tw/people), 給自己加上"鄉民關心你"的標籤
- 加入 [github](https://github.com/)
- 加入[討論區](https://groups.google.com/forum/?fromgroups#!forum/kuansim)並發一篇自介文(單獨發一篇), [HisnYi Chen](https://g0v.hackpad.tw/ep/profile/BXGmz8TpDOs) 會幫你作一些協作工具的設定
    - 說明你是誰
        - 你的暱稱
        - 你的github id
> github id 主要是用來登入團隊工作系統
> [name=HisnYi C]

    - 你想做什麼
        - 想擔任的角色，可複選
            - 前端工程師: 負責 Web 或手機  App 前端開發
            - 後端工程師: 負責資料庫, API  開發
            - 營運工程師: 負責專案上線，開發環境準備，自動化工具開發
            - 文案寫手: 撰寫使用教學文，推廣文, 網站規劃
            - 視覺設計師: 視覺設計
            - 協調者: 幫忙與其他組織[鄉民關心你](http://hack.g0v.tw/people)串連，尋求合作或是行銷這個專案讓更多人認識參與
            - 值日生:
                - 幫忙招呼新人，引導他融入團隊，並且介紹他感興趣的任務使之參與
                - 處理實體聚會的庶務，像是發活動通知，便當購買等
> 角色是用來協助協調者知道成員偏好做的任務，而企劃這件事預期所有成員都會幫忙，所以就不特定分個角色出來。
> [name=HisnYi C]

### 有空做


- 在[這裡](https://github.com/g0v/kuansim/issues)認領想解的任務, 指派自己, 如果已有指派人, 請向被指派的人聯繫你願意幫忙
- 或幫忙蒐集可能有用的資訊到[待孵化想法](https://g0v.hackpad.com/--bw2FTvecSSm)
- 或貢獻點子到[待孵化想法](https://g0v.hackpad.com/--bw2FTvecSSm)
- 或逛逛規劃[文件](http://hack.g0v.tw/kuansim/)給些意見或是改善他
- 或自己在論壇詢問有甚麼可以幫忙，更歡迎發起自己提案!
