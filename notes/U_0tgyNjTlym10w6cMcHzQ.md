# g0v 獎助金許願 - 視覺藝術展演蒐羅平台
我是 Syuan ，我有一個 civic tech 專案的想法，雖然我有想法，但是我並不會建立平台。而建立平台之後，更需要各地的人，到藝術現場做紀錄。

歡迎任何人或團隊對這提案有興趣的，可以此提案內容來投 2020 年獎助金。可以修改這提案內容沒關係。另外若是有用這提案入選的話，我也很樂意提供技術上的協助。

本提案內容授權方式：CC-BY

# 請介紹此專案
建立一個具有各種展演紀錄的平台，讓無法臨其境的觀眾能夠在視覺與思想上，能彼此交擊。

# 你為什麼要作這個專案？（個人動機）
大多數的當代藝術，展演展期約在兩周至四周，分散在北中南東各地。要在那麼短的時間內，如何看見自己有興趣的展。
(這裡的前提是該展為免付費或可拍攝紀錄之展演，請勿侵害版權。)
或者換句話說，做一檔展演，要如何讓你的受眾看見。
(這裡期待被受眾看見的展演比較接近，替代空間展演，具啟發或公益性質，非純營利性畫廊。)
# 這個專案要解決什麼問題？（120 - 500 字）
解決多數展演，時程短，可接觸受眾少的問題。平衡媒體話語資源。
解決觀賞者因偏鄉、或交通問題，無法收看到對自己有感動、觸發之展演。
解決這種區位資源不平衡的狀態。以及當代展演受跨國媒體擠壓，接觸受眾稀少的問題。
# 你預計用什麼方式解決此問題？（120 - 500 字）
我預計利用各地的參觀者、空間經營人或是創作者本人自行提供資料，讓展演趨近一個，接近線上紀錄的方式，並期待有居住在接近該展演空間的人，去認領該空間之資訊，可做紀錄，或者蒐集該展演空間，網路上所提供資訊。

內容須包含，**展演名稱**、**展演論述**、**展演報導**、**展演藝評**、**展演作品表**、**展場各作品特寫照**、**各作品與空間互動環境照**、**展場空間照**、**展場空間平面圖**、**展場空間施作布展圖**、**觀展360動線影片**......。如有環境光影響之作品，亦可補充不同時分之照片。或有開幕演出、講座等活動進行，亦可補充若干照片，或**演出、座談影片**。

# 請列出這個計畫的目標對象，他們的需求、情境與使用動機？（120 - 500 字）
如果是國高中生，我期待的是，補充他們在學校不足的藝術教育，透過接觸、認識當代展演活動。認知到藝術展演是活的、而不是課本裡的死知識。
如果是藝術創作者，我期待的是，增進在對於空間展演的概念。網路上的視覺資料大多呈現平面圖像，較少針對空間環境對於一個展場之影響。我認為這有助於，增進創作者在如何展演的需求上開啟開闊的想像。
如果是藝術教育者，我期待的是，教育者能擁有更多可參考的資源，讓教授當代藝術上可以更游刃有餘，並可空出更多時間在教學之餘，從事自我藝術實踐。更能體察對於藝術創作過程的理解。
如果是展演主辦者，我期待的是，主辦者可以從展演的當下維護，更延伸考量到檔案紀錄的重要性。認知如何完整一份藝術檔案，甚至在未來該展演是可被重製、再演出的！
如果是觀眾，我期待的是，他們在現場觀賞展演之餘，可利用此平台，觀賞到難以抵達現場之展演，擴展自身對於藝術或生命之見解。
# 請說明你如何讓目標對象知道這個服務，並完成其第一次使用，以及在他有需求時願意再來用這個服務？（120 - 500 字）
我認為須讓展演主辦者，與空間營運人，皆能主動提供此平台網址，與此平台刊載之資料。並藉著開源的精神，讓參與現場的觀眾，也成為解說員，引導未能抵現場的線上觀賞者，了解該展演。
