# Taoyuan Gov Green City Project

## 2021.03 - 2022 平台成果
https://treeplanting.oram.gov.tw/

桃園市政府已完成可增進綠化用地之圖資分析方法，並進行全市可植樹用地盤點，加速機關間用地確認流程，第一年度即完成 300 處可推動地點。透過線上業務網站，讓企業與民間公益植樹資源可以聚焦具體用地，加速合作洽談時程。媒合民間公益資源與在地環境綠色服務產業，促進桃園在地園藝廠商、環境教育團體，與公益出資者共同實踐適合地區地景之植樹綠化成果，有效促進各用地管理單位增進綠化質量。

## 本計畫的工作特點

相對於臺灣好植地民間專案，本計畫有市政府主動投入，以下三個面向是縣市政府均能推動的重點：

一、市政議題
- 市長及秘書長主持專案會議
- 各單位業務中的綠資源作為
- 依照圖資分析結果，由各單位確認用地是否釋出

二、圖資用地分析方法精進
- 使用 NDVI 福衛五號 4 公尺解析度網格
- 使用桃園範圍全部公有土地，縣市整府內部可以使用公有土地 API

三、公私協力促進平台
- 舉辦行政溝通，能接洽各用地管理單位，包含中央 (例如經濟部水利署、農業部農田水利署、衛服部醫院機構、財政部國產署..等)、地方等相關單位
- 縣市政府主動引入企業公益植樹，並建置查詢平台，提供篩選器，讓多元綠化響應單位可以依照地景類型查找相關用地 (海岸區位、公園綠地區位、工業區區位、水環境區位..等)
