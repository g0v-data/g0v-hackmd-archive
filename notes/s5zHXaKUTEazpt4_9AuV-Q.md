# 醫療前線注藥流速計算

## 問題

有多少變數是需要現場輸入的？

- 藥劑的單位濃度 ex: (2amps) 500mg in 250ml => 2 mg/ml
- 體重 ex: 100kg
- 醫生指示流速 ex: 0.4 mg/kg/hr

試算
```
0.4(mg/kg/hr) * 100kg / 2(mg/ml) = 20 ml/hr # 最終流速
```
