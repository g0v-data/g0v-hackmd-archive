# 前言

本次視訊活動的主題是討論租屋政策。租屋市場有什麼問題？我們怎麼看最近行政院的300億的租金補貼?有了300億租補為什麼我還要們要倡議修法？

# **租屋市場有什麼問題？**
* 在高房價下，越來越多民眾必須以租屋作為居住選項。有觀眾提問，現在的年輕人要租多久才能買房子？
    * 根據2015年永慶調查，台北市首購族突破40歲，新北市突破38歲。近年可能更惡化。

過去租屋市場不受重視，租房子、存頭期款、買房子，是過去人對住宅的想像。租屋一直都不透明。

* 租屋市場最嚴重的問題是「70%~90%租屋市場是黑市」和房東房客權力普遍不對等。
    * 因為政府資料不全，租屋黑市只能以現有資料粗略推估：比方說政府對「租屋規模」就有「住宅狀況抽樣調查」、「人口及住宅普查」、「家庭收支調查報告的『家庭住宅狀況』」三種，數據可以差距到20-30萬戶。當政府要說租屋市場的規模有多大，就會用最大的數字，但是，當被檢討租屋黑市多嚴重就會用最小的數字。
    * 房東申報租賃所得稅也沒辦法拆出「租給住宅」的數量，只能分成「租給自然人」或「租給法人」。

* 之所以會有那麼嚴重的租屋黑市問題，除了政府長期漠視外，也是因為台灣是少數以小房東為主的租屋市場，較難控管。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_62c543e3e9f1c91b0c31e360b8e035bd.png)



租屋黑市會產生很多不良後果，包含政府完全無法掌握租屋市場供需情形、沒辦法透過政策管理租屋市場等等。
- 比方說，租客為什麼申請租金補貼會遇到困境？房東讓房客申請租補可以減稅，理論上是雙贏的。但是因為黑市的原因，房東本來就逃漏稅，因此減稅只是讓房東逃漏稅的事實曝光，房東就不會這麼做。
- 不得遷入戶籍、房東的稅金增加轉嫁到房客身上，這在法律規定上都禁止寫入契約。然而，我們在書局可以買的到房屋契約卻都還有這一條。
- ![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_316347456d230c540ac3b34a92ca0560.png)
![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_040955a2eacd0a2ef5d29a5e977353da.png)


* 這麼嚴重的租屋黑市在先進國家裡，只有台灣獨有。
* 在國外對住宅政策的思考模式基本上是「居住權>財產權」；但在台灣則是「財產權>居住權」。
* 國外有一些能保護房東、房客雙方的政策或法令，也都是因為他們「沒有租屋黑市」才可能做到。比方說：
    - 租金凍漲令。在經濟蕭條時期，國家讓房東減稅，並要求房東不得漲租。
    - 法國，冬季的禁止驅逐令。房東不能因為房客繳不出租金，就驅離房客，因為有生命危險。然而，台灣根本不清楚哪裡有租屋，無法實施這個政策。

巢運在4/15-5/15進行問卷，共5,011份有效問卷，其中有51.1%（2,562）份是租屋族：
* 近年來租金過高當然是困擾，但更大的困擾其實來自於租屋黑市下的居住不穩定與無法享有合法權利。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_f1b95fa73da39c2b8e9cdbe528dcdee9.png)

- 根據「2021內政部電信信令人口統計」做調查，中華電信用手機訊號做調查，結果發現台北、新北、桃園、新竹縣市、台中，夜間人口都大於戶籍人口，單是台北市就超過19萬人（佔戶籍人口7.3%）。
- 比方說某縣市戶籍人口是100人，結果晚上發現這個城市有120人，那這20個人是誰？除了移工、夜班人員等狀況外，很大一部分應該就是無法設戶籍的租屋族。
- 租屋族沒有公托或學區等權利，也沒有投票權，相當於二等市民，這也是政府不想改革租屋的因素之一。

# 政府的解方：300億租金補貼

現行的租金補貼：每年12萬戶，約57億預算。領取人所得限制為「最低生活費」2.5倍以下。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_2512fc808db8e94c1a23197dc964d73b.png)


300億租金補貼：一年50萬戶，約300億預算，領取人所得限制放寬為「最低生活費」3倍以下。
- 50萬戶，橫跨兩年，但是是今年兩個月到明天十月，總共一年份。
- 把月收入限制放寬→最低生活費「2.5倍」以下變成「3倍」以下
- 三口家庭，每月月收入約12-17萬以下的家庭就能申請。
![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_51784663d7b7fc0487dbb41265799f24.png)
>
我們的批評：
1. 資源錯置。租金補貼的目的是純粹「協助經濟負擔能力不足的租屋家戶」，但中產階級憤怒的並不是租不起房子，而是買不起房子，以及缺少好的租屋品質與權利保障等。
2. 根據巢運問卷調查結果，87%受訪者都表示發錢不能解決他們的問題。
3. 目前每年住宅基金虧損25億（且是在國庫每年補貼50億的前提下），財務該如何永續？如果只是選前大撒錢，是否就是一種政策買票？


![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_2fb78705b5f6c496c8d481be8dd7042c.png)







# **民間租屋市場改革方案**
我們有一個整體性的改革方案，核心觀點是透過持有稅改革增加租屋市場供給，並且透過蘿蔔加棒子讓房東願意浮上檯面透明化。
* 但今天由於時間有限，只細講現階段修法的部分，若大家有興趣可以再專門開一場為大家講解。



![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_3d959711ae71f46876a9c7d1a956ca18.png)

目前我們先提出的是部分修補的方案，也就是「讓租金抵稅制度合理化」與「糾紛處理機制改善」，並讓房東適用相關不平等合約的罰則。

因此，我們在今年2月提出了租賃條例與所得稅法的修法版本。核心內容是：
* 將租金抵扣額在「所得稅法」中改為特別扣除額，並提升額度至30萬，具體降低租屋族負擔
* 無效的地政調處機制應當參照消保申訴調解制度合理改進。
* 花費大量公帑，名為「健全租屋市場」的包租代管資料，也應盡速導入實價登錄系統。

還有一些細節要求，如目前房客要提前終止契約（且房東不可要求賠償）的要件需要「致難以居住」，但實務上難以認定；房客遲付房租被催告後房東可以把房客趕走，但並沒有加上「相當期限催告」的但書等。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_b910922b9870780a45773b03a1c77eaf.png)
* 目前申報所得稅是「標準扣除額」與「列舉扣除額」二選一，之後再抵扣「特別扣除額」。
* 租客申報租金抵稅，是放在「列舉扣除額」，額度12萬。但以一般中位數家庭而言，抵稅效果非常差，甚至在未扶養子女的狀況，申報租金抵稅還要繳更多稅，是不合理的制度。
* 必須承認，這樣的改革仍是很修補式的，房東仍然可能因為租屋黑市不讓你申報抵稅。
* 但現在的狀況是，即便房東願意讓你申報租金抵稅，這個制度還是不好用。

 
>
> ![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_fb10c0f77139c5fbd3cacfd5e783baff.png)

* 目前租屋糾紛有三種處理方式：「訴訟」、「地政調處」、「調解」。
    *  一般而言「訴訟」不划算，多數人都不會選擇。
    *  「地政機關調處」租賃糾紛，是租賃條例立法時新增的。但原本地政機關的調處都是大額度的土地問題，要聘滿8個委員來審查。因此處理租屋這種小標的，地政機關就會累積案件一起處理，使得調解曠日廢時。2017年租賃條例立法以來，台北市至今地政調處作成共0件，根本是沒用的措施。
    *  「調解」又分兩類，鄉鎮區公所或縣市政府的「民政調解」與消保會的「消保調解」。消保調解因為有消保官與正式公文書等，成功機率高達85.1%，而民政調解成功率只有34.2%。
* 但要適用消保調解，首先房東得被認定為「企業經營者」。這部分消保會在2016有過函釋，只要「反覆實施出租行為，非屬偶一為之，並以出租為業者」就是「企業經營者」。
* 所以只要房東不是第一次租房子，都會被認定為企業經營者，房客就可以用消保調解處理糾紛。
* 不過這裡還有兩個問題需要修法解決：
    * 首先，消保會的函釋不夠穩固，哪天改掉的話房客就又不能使用了，所以要把函釋入法才可以。而且這樣就能夠確保房東訂出不平等契約時，可以由消保法直接開罰。
    * 第二，消保法的框架是「顧客」針對「企業經營者」，所以如果好房東遇到爛房客，房東現在也不能用消保調解。我們希望目前沒有用的地政調解機制，針對租賃的部分比照消保調解的框架直接照抄，這樣房東也有很好的糾紛處理工具可以用了。
