# 立法院 API /立法脈絡介紹

[立法院法律系統](https://lis.ly.gov.tw/lglawc/lglawkm)
- 整理所有三讀通過法案
- 變動、修正皆可在此找到
- 也可看到單一條文的歷史脈絡（修正歷程）
- 可找到所有提案版本


[立法流程圖](https://www.parliamentarytv.org.tw/images/micropd/05-pic03.png)
- 一讀佮程序委員的差異，敢有記錄踮立法院系統

[立法院議事暨公報資訊網](https://ppg.ly.gov.tw/ppg/)
- 資料內容
    - 整理正審理的議案資訊
    - 只會顯示出程序委員會的版本
- 資料類型
    - 議事錄：會議摘要，通常兩天後產出
    - 會議紀錄：完成會議記錄，較久才會產出
    - ...

[立法院開放資料服務平台](https://data.ly.gov.tw)

- 尋找提案：[議案提案](https://data.ly.gov.tw/getds.action?id=20)
    - 只會顯示議案最新進度，看不見被退回幾次
    - [Ronny整理議案API](http://ly.govapi.tw)
    - [提案法律對照表](https://congress.boggy.tw)
