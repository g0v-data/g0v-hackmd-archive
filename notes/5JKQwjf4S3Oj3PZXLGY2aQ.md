## 1. 台灣社區農耕概況介紹

![](https://g0v.hackmd.io/_uploads/rylua6gNmGx.png)

- 台灣的城鄉結構
    - 大型城市，如臺北市、新北市、高雄市、臺南市、臺中市、桃園市等
        - 都市密集街區，如臺北市大安區
            - 街區內的社區農園
        - 城鄉交界地區，如臺北市北投區
            - 街區內的社區農園、農業區農地
    - 鄉鎮 / 農村社區 / 臨海漁村社區 / 地方創生地區，如臺南市新化區、桃園市新屋區、高雄市永安區
        - 農業區農地、街區內的社區農園
- 社區農園的定義：一群人運用生活中的公共空間，共同栽種
    - 因為是以生活者為耕種者，所以場域不限定在農業區農地
    - 各種類型舉例
        - 社區空地
        - 住宅大樓屋頂
        - 學校的空間
        - 高齡生活照護的空間
        - 社福機構的空間
        - 醫院的空間
        - 辦公環境的空間
        - 工廠園區內的空間
    - 營造途徑與動機列舉
        - 發現閒置空間 -> 活化使用
        - 先有綠化空間 -> 增加可食作物
            - 臺北市的社會住宅，指定設置可耕種的屋頂或露臺
        - 高齡生活與社福機構 -> 園藝栽種課程活動
        - 醫院紓壓 -> 綠化 / 可食作物
        - 辦公環境 -> 栽種可食作物
        - [案例數量較少] 先有堆肥設施 -> 開始延伸到耕種
