# 台南市議會點名

台南市議會在審查總預算時經常因為清點人數流會，為了深入這個問題所以嘗試整理相關資料，看看能否系統化處理這個苦差事

目前台南市議會有 57 個席次，其中 林宜瑾 任職期間成為立法委員，依規定不需要進行補選，因此現有共 56 席議員；當議員要求清點人數，入席的議員未過半則當次會議立即結束，也就是清點未達 28 位隨即中止會議。

定期會的重要工作是審查總預算，如果沒有審查完成就會延伸後續的臨時會繼續審查，對於台南的預算來說就是多一筆負擔，而且會造成許多工程的時程安排問題。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_bd5eda76146ec4edf3e5827f95ce4529.png)

上圖是 2021-11-12 的清點狀況，個別議員姓名後面接的是黨籍縮寫，透過顏色呈現未簽到、簽到且出席與簽到未出席狀態，整理下面欄位

1. seat 座位編號，從 郭信良 議長位置為 1 ，向右到底第一排 1-10 ，第二排 11-20 ，以此類推
2. name 議員姓名
3. party 代表政黨，或者是加入的政黨黨團，這個部份還有點模糊，特別是在個人黨籍跟加入黨團有差異的情況，不過衝突情況不多
4. status 出席狀況
    1. 未簽到
    2. 簽到且出席
    3. 簽到但未出席
5. time 清點時間，也就是議程進行中有議員提出清點人數，經主席確認後開始清點的時間

資料範例：
```
[
    {
        "seat": 1,
        "name": "郭信良",
        "party": "無",
        "status": 1
    },
    {
        "seat": 2,
        "name": "林炳利",
        "party": "無",
        "status": 3
    }
]
```

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_335304854be845c97562a045340f64bc.jpg)

如果主席同意，議會其實可以把清點人數結果放上電子螢幕呈現，只是在公開探討流會問題後，已經有 2 次發生主席跳過這個步驟直接宣佈流會的情況，增加了確認的難度

當上述情況發生，就得看當下有沒有錄影畫面拍攝當時座位出席情況，只是有時透過錄影畫面清點跟現場清點的數字不太一樣，目前除非人在現場，否則很難進一步確認；如果連錄影畫面都看不到，大概只能往前追溯發言情況，但議員經常有簽到出席且發言後立即離場的情況，光看發言其實跟清點人數當下的情況仍會存在落差。
