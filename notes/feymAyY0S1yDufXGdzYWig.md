# 臺南市議會問題蒐集

> [點此觀看原始內容](https://g0v.hackpad.tw/6EHRGkMNxmK)


- 議案資訊在開議前並未放上網路供民眾閱覽
> 第9次臨時會2017/3/28開議，但今天在(3/29)在議案資訊網站上仍看不到討論議案( [http://www.tncc.gov.tw/motions/default1.asp](http://www.tncc.gov.tw/motions/default1.asp) )
> [name=kiang]

- 議案資訊並未包含各種附件電子檔
- "臺南市議會錄影、錄音管理規則"( [http://www.tncc.gov.tw/DC.asp?nsub=L0A900](http://www.tncc.gov.tw/DC.asp?nsub=L0A900) )第五條， "未經公開之開會過程且涉及其他議員的影音資料" 這些影音資料如果未涉及個案隱私或國防機密等應該予以公開
- 個別議案應該列示表決結果，而不是只有提案人與連署人
- 每個開議期間應該列示出席情況
- "議事影音" 對應的會議記錄應該在完成後公開
