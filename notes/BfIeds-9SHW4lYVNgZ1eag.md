# 心理服務污染
- 定義：造成被服務對象的干擾、不舒服、服務無效、甚至是創傷，以及造成認知、價值觀的不當影響。
- 如何避免：
    - 服務前
        - 調查社區與服務對象的需求，服務內容要圍繞在社區的需求上，並做好充足的準備，避免產生無效服務、干擾等汙染。
        - 辦理專業訓練，了解被服務對象的現況、特質、價值觀、文化...等，讓自己帶著正確、欣賞與尊重的觀點進入服務場域，降低造成被服務對象不舒服、歧視、受創...等汙染。
        - 制定志願服務倫理或服務行為準則，提高對服務汙染的警覺心。
    - 服務中
        - 嚴守志願服務倫理或服務行為準則，志工夥伴相互提醒，也要虛心接受夥伴的提醒。
        - 尊重對方文化，避免價值觀的壓迫或歧視。志願服務不是用上對下的壓迫方式來傳遞「優勢文化」，而是對等尊重的相處，讓被服務對象保留選擇的權利。
        - 發揮同理心與欣賞的眼光，看到被服務對象的優勢，協助他們整合自身優勢與新的觀點。
        - 自身碰到或觀察到有價值觀的衝突時，應該要在團體會議中提出討論，透過團體智慧來集思廣益。
        - 用協助探索來取代給予答案。
    - 服務後
        - 進行服務反思與團體分享討論，並將反思與討論歸納出的經驗與知識，帶到下次服務的場域實踐，這樣不但可以對服務汙染有更高的敏感度，也能有效降低服務污染的可能性。

    - 希波克拉底誓詞：仰賴醫神阿波羅、埃斯克雷波斯及天地諾神為證，鄙人敬謹直誓：願以自身能力及判斷力所及，遵守此約。凡授我藝者，敬之如父母，作為終身同業伴侶，彼有急需，我接濟之。視彼兒女，猶我兄弟，如欲受業，當免費並無條件傳授之。凡我所知，無論口授書傳，俱傳之吾與吾師之子及發誓遵守此約之生徒，此外不傳與他人。我願盡余之能力與判斷力所及，遵守為病家謀利益之信條，並檢柬一切墮落和害人行為，我不得將危害藥品給予他人，並不作該項之指導，雖有人請求亦必不與之。尤不為婦人施墮胎手術。我願以此純潔與神聖之精神，終身執行我職務。凡患結石者，我不施手術，此則有待於專家為之。無論至於何處，遇男或女，貴人及奴婢，我之唯一目的，為病家謀幸福，並檢點吾身，不作各種害人及惡劣行為，尤不作誘姦之事。凡我所見所聞，無論有無業務關係，我認為應守秘密者，我願保守秘密。尚使我嚴守上述誓言時，請求神祗讓我生命與醫術能得無上光榮，我苟違誓，天地鬼神實共亟之。
    
    - 誓詞衍生出的醫學倫理（四大原則）
        1. 尊重自主原則:強調尊重病人的尊嚴與自主性，例如不隱瞞病人之病情及診斷、保護病人的隱私及強調知情同意的重要性。
        2. 不傷害原則:強調醫療人員謹慎地執行以達到適當的照顧標準。此原則的精神在於不要傷害、防止傷害以及除去傷害。避免讓病人承擔任何不當的、受傷害的風險。
        3. 行善原則:指在不傷害他人之外，進一步關心並致力提升病人的福祉。所有醫療行為一切應以病人的利益為前提。
        4. 正義原則:公平地分配醫療資源、尊重病人的權利及尊重道德允許的法律。

- 醫師的社會責任
    - 社會責任是指：出於自發性的動機與理念，來提高生活品質，創造更好的社會，並希望藉此行為對更廣泛的人群與社會帶來正面的影響
    - 醫學專業與社會有關，可視作一種社會契約（社會給予醫療人員特權，提供特定服務並承擔責任，醫療人員則同意將此特權用在提升社會福祉上，其次才是個人利益）
        1. 維護患者健康與安全： 醫師的首要責任是確保患者的健康與安全。他們需要透過準確的診斷、有效的治療和良好的醫療管理，幫助患者恢復健康或者緩解症狀。
        2. 尊重與保護患者權利： 醫師應該尊重患者的自主權，並確保他們的意願和權利得到充分尊重。這包括尊重患者的隱私、保密權，以及讓他們參與醫療決策的權利。
        3. 持續專業發展： 醫師需要不斷更新自己的知識和技能，以跟上醫學科學和技術的發展。這有助於確保他們提供給患者的是最新、最有效的治療方法。
        4. 公平對待所有患者： 醫師應該公平地對待所有患者，不論其社會地位、種族、性別、宗教信仰或其他身份。他們應該避免歧視，並致力於提供平等的醫療服務。
        5. 參與社區健康促進： 醫師可以通過參與社區健康教育活動、預防性醫療措施以及公共衛生計劃等方式，促進整個社區的健康。
        6. 與其他醫療專業人員合作： 醫師應該與其他醫療專業人員合作，共同為患者提供全面的、協調一致的醫療服務。
        7. 關心社會議題 : 他們應該致力於改善公共衛生，並參與預防疾病的努力。這可能包括教育公眾關於健康生活方式的重要性，或者參與疫苗接種和其他公共衛生活動。
        8. 關心健康知識的傳播 : 教育公眾關於健康生活方式的重要性，或者參與疫苗接種和其他公共衛生活動。
        9. 致力於提高醫療服務的品質和可及性 : 倡導更公平的醫療政策，或者努力確保所有人都能獲得他們需要的護理。
        10. 參與政策研究與製定：從政策面改善醫療體系、提升全民健康。例如，參與研究分析、建議反饋、監督評估、推動並落實。
        11. 對環境的關懷 : 醫療行業是一個資源密集型的行業，醫生可以在他們的實踐中推動更環保的選擇，例如使用可再生的醫療產品，或者推動減少醫療廢棄物。
        12. 分配醫療資源並防止病患擅用他們無權取得之服務：避免浪費及缺乏效率的醫療行為，分辨有效與無效的治療手段
        13. 對抗不公正的社會結構 : 努力確保所有人都能獲得他們需要的護理。
        14. 致力於終身學習：可以繼續提供最新和最有效的治療方法。這可能包括參加研討會，閱讀醫學文獻，或者參與專業組織。
        15. 跨學科合作交流：和其他領域專家合作，提供醫療衛生或其他領域多元化的視角與思維。
        16. 尊重他們的病人 : 應該尊重病人的自主權，並且在所有決策中都考慮到病人的需求和願望。
        17. 探討及推廣生死學:目前亞洲社會對於生死的議題仍較保守，探討相關議題的族群多為哲學家和宗教家；因此，有許多相關生死經驗的醫生應該投入探討和推廣生死學。目前社會上已有部分例子，如善終概念的推行、器官捐贈（取代保留全屍的傳統觀念）和推廣大體老師的概念。
        
    - 醫師的社會責任可粗分為「法律責任、倫理責任與自發責任」三種：
        1. 法律責任：法律是社會對與錯的法典，而醫師的法律責任明定在相關的醫療法典當中， 以規定相關醫護人員的責任、權利與義務。所以，醫療法典中關於醫師的規定可算是醫師社會責任的最低要求。
        2. 倫理責任：身為醫師應該「奉獻自己、尊重師長、以良心來從事醫業、重視人類的生命與價值、維護醫事專業的傳統、不做違反人道的事、不因他種因素給予病人不同的對待、與其他醫療團隊充分合作」等，說明身為醫師倫理責任的所在。
        3. 自發責任：指身為社會公民所應盡到的責任與義務。每個人基本角色本都包含專業面與社會面，關切社會事務本是每個人擁有的權利，只是在參與社會事務時應如何謹守專業範疇的規範與要求。（如在提及醫事倫理時，都有強調「醫生除了精通醫術之外，尚須修身立德，要發濟世的宏願，不要做出違背道德的事情。」而現今歐美國家地區的醫師自發性地「組織醫療人權組織、醫療服務組織、 醫療反戰組織、提出相關宣言、參與政治事務」等，都為此原則做了最好的注釋。）

# 病毒＆細菌
- 結構
    - 病毒：病毒的結構相對簡單。它們主要由核酸（DNA 或 RNA）和外層的蛋白質組成。病毒需要寄生在宿主細胞內才能繁殖。
    - 細菌：細菌的結構較為複雜。除了核酸外，細菌還有細胞壁、細胞膜和細胞質等組成。細菌是單細胞生物，可以在其他細胞內外存活。
- 大小：病毒比細菌小得多。病毒的大小約是細菌的10到100倍。
- 生存方式
    - 病毒：病毒必須依賴宿主細胞才能繁殖。它們入侵宿主細胞並利用細胞的機制來複製自己。
    - 細菌：細菌是獨立的生物體，可以在宿主內外存活。它們無需寄生在其他細胞內。
- 致病性
    - 病毒：大多數病毒對人體通常具有致病性，如流感病毒、感冒病毒等。
    - 細菌：細菌則是有益菌與壞菌。人體內有許多與細菌共存的菌種，其中大部分是有益的好菌。但某些細菌，如沙門氏菌、金黃色葡萄球菌等，對人體不利。
- 傳播方式
    - 病毒：病毒通常通過空氣飛沫、接觸或食物傳播。例如，感冒病毒和流感病毒可以通過咳嗽和打噴嚏傳播。
    - 細菌：細菌可以通過接觸、食物、水和昆蟲傳播。例如，沙門氏菌可以通過食物中的污染傳播。
- 生命週期
    - 病毒：病毒無法獨立生存，必須寄生在宿主細胞內。它們的生命週期包括感染宿主、複製自己的核酸，然後釋放到宿主細胞外。
    - 細菌：細菌是獨立的生物體，具有自主生存能力。它們可以在不同環境中生存，並以分裂方式繁殖。
- 藥物治療：抗生素可用來殺細菌，通常一種抗生素可對付數十種細菌，但要吃到一定療程否則細菌會產生抗藥性；但是抗生素並不能殺病毒，而所謂的抗病毒藥物通常只能針對特定一種病毒，還經常會失敗，最常見的例子就是愛滋病毒HIV，用以治療的雞尾酒療法，就是將不同種藥物結合來執行。

- 如果消失會對人類產生什麼影響？
    - 病毒在世界的作用：
        - 支撐生態系統
            - 噬菌體是海洋中細菌種群的主要調節器，控制細菌菌種不過度增生，壓縮其他細菌菌種生存空間。
            - 通過淘汰微生物，病毒確保產氧浮游生物有足夠的營養進行高速率的光合作用
            - 物種控制：當某一物種過剩時，病毒會傾向快速複製並消滅物種，為其他物種創造生存空間（如人類大流行病）
        - 維持有機體健康
            - 有些物種會依賴病毒生存，在競爭中取得優勢
            - 黃石公園例子：一種感染真菌的病毒使草變得對地熱土壤溫度具有耐受性，單靠真菌無法做到
            - 某些病毒能幫助抵禦病原體
            - GB病毒C型與延遲人類愛滋病病症進展有關
            - 皰疹使老鼠不易受到某些細菌感染，包括黑死病和李斯特菌，而對於人類而言雖然可能造成免疫疾病，但某種程度上亦賦予免疫益處，可視作一種共生關係
        - 未來應用：發展於藥物開發與疾病治療
    - 細菌在世界的作用：
        - 分解死亡生物體，使其轉化成生物生長所需營養物質
        - 分解垃圾與有毒物質，延遲人類對於環境的衝擊
        - 協助生物消化作用、製造所需維生素及礦物質
        - 牽制其他微生物生長（藉由競爭或毒殺），可進一步避免生物體生病

- 處理方式
    - 停止食用受污染食物：如果您意識到食物可能已經被污染，請立即停止食用。
    - 保持身體水分：儘量多喝水，以防脫水。
    - 就醫：如果症狀嚴重（如持續嘔吐、嚴重腹瀉、高燒等），請及時就醫。醫生可能會推薦適當的藥物來幫助治療。
    - 觀察症狀：注意可能出現的症狀，如噁心、嘔吐、腹瀉、發燒等。
    - 催吐：如果吃下食物的時間不長，在1～2小時之內，毒物還停留在胃內，可以試著把毒物吐出，減少毒素吸收。可以用手指、匙柄、筷子等刺激舌根部；或快速飲用冷鹽水（食鹽20克，加溫開水200ml冷卻後使用）、薑汁來催吐。
    - 勿服用非處方止瀉藥：因為會延長腸道內容物留滯時間，可能會助長人體吸收毒素。
- 預防
    - 遵守個人衛生原則，確保自身飲食健康
    - 遵守食品處理之原則 
        - 區分生食、熟食，避免交叉污染，食物需徹底煮沸
        - 注意保存溫度，區分冷凍、冷藏
        - 注意水、食材、廚具、人員清潔（傷口要包紮）
        - 廚師檢查 e.g. 聞、異狀
    - 注意外出用餐習慣 e.g. 避免冷食、生食、來路不明的食品
    - 食物來源提供、上游廠商 e.g. 注意衛生、食品來源監管、食品保存、清潔、定期審核
- 米酵菌酸
    1. 在印尼椰子發酵餅發現
    2. 一種強烈呼吸毒素→影響呼吸作用路徑
        - 抑制ADP/ATP 轉運酶(ACC)，使ATP無法離開粒線體
    3. 由發酵穀物、椰子等受到唐菖蒲伯克氏菌污染產生
        - 唐菖蒲伯克氏菌可分泌毒黃素、米酵酸菌
        - 在特定種類濃度之下才會分泌米酵酸菌
            - 受脂質的種類、濃度影響，決定分泌何種毒素 -> 脂肪含量低於20%的食材引發的中毒是由毒黃素 -> 產生米酵菌酸需一定的濃度
    4. 毒性極高，1毫克即可致命
    5. 其好發於室溫、酸鹼值中性下、發酵1-2天及有一定脂肪含量食物之中，故預防和抑制可藉由:15°C及以下低溫、短期保存及酸性環境。
    6. 症狀
        - 腸胃道症狀，例如噁心、嘔吐、上腹部不適、腹瀉等，還常伴有頭暈、全身無力等症狀
        - 嚴重中毒者會出現肝腫大、黃疸、皮下出血、少尿、血尿、煩躁不安、意識不清、抽搐、休克等症狀
        - 重症中毒病人多因呼吸衰竭而死亡
# 如何分辨真假訊息
1. 對標題保持懷疑態度：假新聞/假消息常會使用聳動的標題，如果標題內容令人難以置信，很有可能就是不實報導。
2. 調查消息來源：如果報導來自不熟悉的機構或組織，請調查它們的詳細資料，了解更多背景資料再對訊息的真實性進行判斷。
3. 查詢相關報導：如果有多個具公信力的來源都報導相同的內容，則新聞內容較可能屬實。
4. 只分享可信的消息：還沒確認新聞/消息的真實性之前，不要在網路上分享。
5. 善用各種假訊息查證管道（如：LINE官方提供之「LINE訊息查證」帳號、趨勢科技「防詐達人」等)，彙整各方提供的資訊後再對新聞/消息的真實性進行判斷。
# 中暑
- 定義：人長時間處於水分易蒸發、流失的空間或狀態下（如高溫或悶熱環境中），又未適時補充水分及鹽分，造成身體熱量無法順利排出，核心體溫升高，出現各種不適症狀。
- 症狀：身體燥熱、皮膚乾燥發紅、低血壓 、心跳加快、呼吸加快、神經系統異常（行為異常、幻覺、意識模糊不清、精神混亂等）、口渴、疲倦、全身無力、頭暈、頭痛、說話喘、激動、焦慮、昏迷、抽搐、大量出汗但也可能會無汗、臉色蒼白，嚴重時，患者還可能意識不清、昏迷或休克。
- 成因：中樞神經系統失調，包括體溫調節功能失常，加劇體溫升高，使細胞產生急性反應。通常是熱衰竭未及時發現所致，體溫可能超過40度。
    - Q：跟熱衰竭有什麼不同？
      熱衰竭成因：大量出汗嚴重脫水，導致水份與鹽份缺乏所引起的血液循環衰竭。多因熱痙攣、熱暈厥演變而來。患者可能流失過多水分，或有補充水分但卻缺乏鈉離子，體溫通常介於38度~40度，可以稱為中暑的前兆。
- 分類：根據衛福部，中暑可分為傳統型中暑跟勞動型中暑
    - 傳統型中暑：身體無法應應環境溫度和濕度
        - 高危險人群：年長、肥胖、慢性病、嬰幼兒、孕婦
    - 勞動型中暑：長時間在高溫高濕度的環境下工作所導致
		- 高危險人群：軍人、工人、在密閉無空調空間工作的人、農夫、運動員
- 預防：在外盡量在陰涼處行走、適時補充水分及防曬產品、穿著淺色衣物並保持身體涼爽
- 急救五步驟：
  1. 陰涼：將患者從高熱的環境中移至蔭涼通風處。
  2. 脫衣：鬆脫衣物、足部稍微抬高、平躺休息，若有嘔吐現象則側躺保持呼吸道暢通。
  3. 散熱：全身皮膚噴灑水，通風（扇風）促進排汗，將冰袋（毛巾包覆冰塊或冰涼飲料罐）放置於頸部、腋窩和鼠蹊部（因為這些部位血管較淺薄） 。
  4. 喝水：意識清醒者可給予稀釋電解質飲料（如運動飲料）或加少許鹽的冷開水。患者意識不清時，不可給予飲水。
  5. 送醫：可撥打 119 或自行送醫。
- 常見迷思
    - 吃退燒藥是沒有幫助的：熱傷害主要是因為體溫調節中樞異常，使身體產熱與散熱失調，因此對退燒藥劑並不會有反應
    - 利用擦酒精來散熱：若於體表擦拭酒精，會使體表的水分蒸散(暫時會有涼感)，但它卻會引起表層血管急速收縮，反而有礙身體的散熱

- 癌症減藥
    - 起因：以「最高耐受量」為出發點的概念可能不適用癌症藥物治療
    - 目的＆優點
        - 減少病人副作用以提升病人生活及護理品質
            - 例：Sotorasib (洛滿舒) 的使用頻率降低可減少其對患者所造成的腹瀉以及肝臟問題
            - 例：低劑量放療對於女性乳癌的復發率僅比高劑量高出1.6個百分比（4.4% 對 2.8%)，但對於患部纖維化的機率則是明顯低於接受高劑量放療的患者
        - 減少病人開銷
            - 例：最近的一項分析確定了56種口腔癌藥物，透過減少劑量或改變療程，其成本可以至少削減33%，這種方法可以為患者平均節省94,000美元
        - 減少醫療資源浪費
    - 隱憂
        - 藥物銷售額大幅下降引起經濟問題
