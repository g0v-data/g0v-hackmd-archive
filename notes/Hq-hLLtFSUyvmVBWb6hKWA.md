FB 政治人物粉專查廣告金額方式 by Ronny
=================================

前言
---
Facebook 在 2019 年 3 月推出了 AD Library ，可以查詢粉專下廣告的記錄，而在 2019 年 6 月，針對政治人物或政治議題的粉專，可讓人查詢更詳細記錄，包含下了多少預算，觸及多少人，觸及對象的性別年齡所在城市比例，出資者姓名等資訊，理想上讓政治宣傳的資訊更透明。

但 FB 針對政治的粉專資訊更透明，所看的並不是粉專的身份，而是看該粉專管理員是否有登錄為「刊登社會議題、選舉或政治相關廣告」的身份。如果下廣告的人未做登錄身份動作，那麼他的政治廣告會跟一般商業廣告一樣資訊是不透明的。目前(2019/8)台灣應該只有我(Ronny Wang，王向榮，立法委員擬參選人）一個人有登錄身份，

![](https://g0vhackmd.blob.core.windows.net/g0v-hackmd-images/upload_3ac3e16f95d4bc637d49baec7d31bed3)

如上圖，有登錄身份的廣告，就可以在 Ad Library 內看到更完整的資訊，而且廣告記錄會永久公開保存，隨時可查閱。[Ad Library 王向榮擬參選人粉專廣告資訊](https://www.facebook.com/ads/library/?active_status=all&ad_type=all&country=TW&q=%E6%96%B0%E5%8C%97%E5%B8%82%E6%9D%BF%E6%A9%8B%E5%8D%80%E7%AB%8B%E5%A7%94%E6%93%AC%E5%8F%83%E9%81%B8%E4%BA%BA%E7%8E%8B%E5%90%91%E6%A6%AE&view_all_page_id=2254769834566581)

![](https://g0vhackmd.blob.core.windows.net/g0v-hackmd-images/upload_5cd9ee15f83f58bb8a2d2c37f7bb299b)
相對起來，未登錄的候選人資訊會少很多，而且廣告結束後資訊也會消失。

而 Facebook 也有提供 Ad Library API ，可以查詢台灣全部的政治廣告（也因為有這個 API ，所以我才能確定 2019/8 為止，只有我一個人有做登錄 XD）

理想上應該是要讓所有在 Facebook 上面下廣告的政治人物或是政治議題粉專，都要去做認證登錄，這樣子可以知道各種 FB 上廣告是誰出錢的，花了多少錢，日後也可以跟政治獻金做連結。

目標
====
* 宣傳 Facebook 有這功能，並且大家可以多留意看到的政治相關廣告有哪些，收集有下廣告但是並未登錄政治粉專有哪些
* 透過網友們提供壓力，逼政治廣告都要去登錄（做法可能是直接在廣告下面留言）
* 資料量累積夠多時，可以統計各政治人物的廣告金額，以及之後可以跟政治獻金做比對，看看是否廣告金額都有誠實申報。
