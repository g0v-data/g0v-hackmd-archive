# g0.0v demo 腳本

> [點此觀看原始內容](https://g0v.hackpad.tw/yzvb2eDmeYF)


中文字是一套很難學習的符號系統，
（舉幾個形音義容易令小孩和外國人混淆的例子，使用萌典畫面）
- 我們取教育部國語辭典的 open data 延伸成遊戲，讓它們變得更好記好懂，遊戲名稱叫字宙反應爐。
- 切到 welcome page - 在浩瀚的字宙裡
- 按標題進入主畫面 \- 我們打造了一座反應爐，反應爐裡分裝了六個迷你平行字宙，
（逐一 zoom in 帶過功能）
- 藍星人字宙可以發掘其他人感興趣的字
- 萌典字宙可以找出字的定義
- 聯想字宙可以找出字的語意群組
- 既視字宙可以找出常讓人傻傻分不清楚的字
- 對撞字宙可以發掘部件的各種排列組合
- 勘輿字宙可以用街道排出字

字宙反應爐維持萌典系列的共通特性，就是鬼打牆，每一個字宙的反應結果都可以透過輸字管，push 到其他平行字宙裡面再反應一次，然後反應結果又可以 push 回來，生生不息，沒完沒了，讓你永遠都無法離開這個網頁，以達成精實的中文學習效果 XD
（一邊操作 i/o 功能）

這次因為覬覦文化部提供的獎金，我們特別增加了一項功能，就是拿文化部的 open data 來測字，把文化部的xxx連起來，經電腦自動比對後最相似的字是……（畫面：死）
