這個專案要解決什麼問題？（120 - 500 字）
------------------------------------
台灣很多人以為四年投一次票就是盡到民主義務，四年間選出來的議員做的不好就不是自己的責任了，但是我覺得四年間怎麼讓議員好好做事也應該是公民應盡的責任，只是過去想這麼做門檻太高，立院議會幾百人每天好幾軌會議在跑，而媒體又有點失靈無法善盡第四權監督責任，我不認為代議政治是需要被淘汰的，但是我們需要一個更能被大家監督的代議政治，所以想透過公民科技解決這問題。也讓議員因為知道自己在議會所做所為都是被很多人看著的，也會更認真做事而不是打混摸魚四年只變成黨的圖章而已。

你預計用什麼方式解決此問題？（120 - 500 字）
---------------------------------------
現在立院和大部份縣市議會都會上網公布議會記錄，包含委員出席、提案發言、投票等。透過爬資料及資料整理的方式，將每一個個別委員做的事情拆開來（有點像議員投票指南在做的），然後接下來幫所有立委和議員建立虛擬的臉書粉絲專頁或是 Line 帳號（必須要強調這是虛擬的或是同好會而不是本人，否則會被臉書或 Line 停權）。利用臉書或 Line API ，自動將議員們的動態張貼，像是「我昨天缺席了議會，今年已經累計缺席 n 次」「在 xxxx 委員會內，xxx 草案我投了反對票」「在質詢內政部長關於 oooo 時，我問了 xxxx」之類的程式自動產生的動態。整個立院議會做的事情很多，但是如果只有一個人做的事情應該就不會多到不是一般人無法吸收的了，因此這樣的虛擬粉專或 Line 帳號被一般民眾追蹤的可能性也提升，就此可以提升一般民眾監督議會的意願（至少可以追蹤自己選區的人）

為什麼你的方法可以解決這個問題？（120 - 500 字）
----------------------------------------
比起看立法院議會記錄或是等新聞媒體把立院發生的事包裝成新聞給你看或等鄉民整理懶人包，針對單一議員的動態做成臉書粉專讓大家追蹤，應該是更容易、無偏頗的型式。本來大家就有在追蹤很多朋友或是一些知名人物，現在對民眾來說他只要再多追蹤一兩個議員就好，對於民眾監督議會的門檻大大降低。

請列出這個計畫的目標對象，他們的需求、情境與使用動機？（120 - 500 字）
---------------------------------------------------
我的目標是所有的選民，我希望讓選民覺得監督議會應該是全民的責任，而且這件事情不應該是困難的，至少每個選民應該要監督自己選區的議員。這計劃是降低監督單一議員的門檻，讓「監督太困難」不再是個合理的理由。

請說明你如何讓目標對象知道這個服務，並完成其第一次使用，以及在他有需求時願意再來用這個服務？（120 - 500 字）
----------------------------------------------------
鼓勵目標用戶針對議員的 FB 虛擬粉專或是 Line 動態發現有值得分享的動態，例如這人缺席超過 50 次了，或是他投票的結果遵從黨意而不是民意之類，或者質詢過程的發言很實際切中要點等，都可以針對這則動態分享到自己的塗鴉牆上，這樣可以讓更多人知道 FB 或 Line 上有這樣的全部立委或議員的虛擬粉專，進而讓他們也去追蹤單一的有興趣的或是他們選區的議員。

而當用戶追蹤了之後他就不需要再主動做什麼事了，就保持他瀏覽臉書和 Line 的習慣就好。

是否已有相似的專案，本計劃的差異與不可取代性為何？
-----------------------------------------
與投票指南很相近，但投票指南的使用時機是選舉之前，本計劃的使用時間是選舉之後四年。但兩專案之間的資源可以共享。(包含整理好的議員出缺席、發言提案記錄等)

你覺得這個提案可能會面臨的困難有哪些，你又會如何解決這些困難？
--------------------------------------------------
* FB, Line 是否能由單一人開那麼多個虛擬粉專或虛擬公眾號。
  * 解決方法可能是直接與 FB, Line 台灣公關連繫說明使用目的，讓 FB, Line 明確知道後應該就不會禁止或有可能提供幫助
* 立院議會網站如果改版自動抓取機制就會死亡，以及有些議會根本沒把這些資訊放上網
    * 短期內只能用人力跟他耗，所以我把立院放前面，期中目標只有一個縣市議會而已。但理想是希望透過立院和單一縣市議會的成果，可以拿來說服其他縣市議會願意讓資料更穩定公開的提供出來
    * 資料抓取的部份可以跟投票指南專案合作共享資料

計畫結束後未來可能進一步的發展？
--------------------------
鼓勵選民可以直接在虛擬粉專內遇到不滿的議員動態時，可以直接留言回覆，而留言可以統整轉給真實的議員，讓議員能更容易得到真正的選民的壓力以及直接來虛擬粉專回應。

透過專案執行後的經驗，可以建立出議會相關記錄的資料結構格式，拿這專案的成果去遊說各議會依照資料結構產製相關記錄的開放資料，讓未來的資料連結可以更穩定，並且議會相關資料可以更容易被其他專案使用
