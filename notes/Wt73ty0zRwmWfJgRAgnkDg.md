### g0v Style Guide

Even 替社群制訂的視覺設計指引，目前尚在進行中，近日 g0v.tw、g0vis、授權中心、leve1up 等專案在討論視覺風格整合的問題時，發現 g0v 網站群可能需要類似 tabzilla 的機制，做為網站之間的橋樑。

### 國會大代誌

經過一年的努力之後，零時政府立法院網站終於有了新名字。能將公報與法條進度自動抓下放進資料庫，提供可用的 API。在網頁上顯示國會最近的行程並且推播到 twitter 供人訂閱，除了公報之外還能搜尋關鍵字，用類似程式 diff 的表達方式，顯示修法提案的前後差異。除了文字之外還整合了國會提供的影像，標示影片中每一位委員的發言時間，讓使用者可以盡快找出感興趣的片段。
![](https://g0vhackmd.blob.core.windows.net/g0v-hackmd-images/upload_17102e2ff4347e1b3a3a2205b1b0aaf5)


### 立院影城

在 [Yahoo! Hack Taiwan 2013](http://hacks.developer.yahoo.com/hack/yahoo-hack-taiwan/congress-cinema-/event_17/hack_995) 短短 30 小時內誕生、獲得最高人氣獎、以超 KUSO 丟鞋彈幕特效引起各大媒體爭相報導的立院影城，打破立法院 ivod 影音系統原本的瀏覽器限制，讓公民在家上網就可以監督國會，比出門看電影有趣萬倍。專案自 11/3 活動結束到現在[仍然不斷更新](http://ivod.ly.g0v.tw/onair)，並且和國會大代誌逐步整合。

### 動民主

動民主家族是一系列的線上民主系統，旨在利用資訊科技解決目前僵化的代議式民主制度衍生出的各種問題。自 5/16 的 Adhocracy meetup 以來，動民主系統從德國海盜黨的 liquid feedback、pirate feedback 衍生出介面重新設計的 1.0 版，在 6/30 的跨國線上會議後誕生了功能全面翻新的 2.0 版，9/30 Federico 來訪後催生了 Airesis 的中文翻譯。目前為了讓系統快速上線供公民團體使用，2.0 精簡版 [急速開發中](http://g0v.github.io/don-republic/)，另，Airesis 官方已經將社群翻譯的[中文版上線](http://www.airesis.cn/)（雖然上錯域名了囧），社群也正嘗試在 heroku 自架中文 Airesis。

### 404失蹤兒少

誕生於 hackath0n 的 404 專案，參考歐洲 notfound.org 網站，和內政部兒童局失蹤兒少資料管理中心合作，提供即時更新的失蹤兒訊息以及網頁內嵌語法，讓各網站使用自己的 404 頁面加入失蹤兒協尋。經過數個月的努力，專案網站終於成功上線，順利移交給兒福聯盟維護管理，社群代表並出席兒福聯盟舉辦的記者會協助宣傳。

### 文化部授權中心

因為整理文化部的作品感到困擾而誕生的文化部授權中心，目的是統一呈現所有文化部的授權作品，方便所有人下載利用。專案在 hackath6n 時找到強力後援，完成了初步的自動化功能。接下來需要設計音樂、視覺設計、照片、投影片等不同類別的作品頁面版型，還有收集整理各作品的 meta 資料，並且使用 github 登入功能和人力資源部整合，提供非 programmer 不經過 git 直接上傳作品的機制。

### 開源藝術示範區

誕生於 hackath6n 的開源藝術示範區，希望藉由藝術創作，吸引音樂、美術、設計等非資訊人才，同時建立藝術作品的開源模式，以便將開源之道推廣到資訊以外的領域。目前美術類作品暫時將工作檔放在 google drive，音樂類作品則在 blend.io 平台上找到便利的開源協作模式。期待文化部授權中心完成後，這些創作成果能有更好的展示方式。

### 鄉民關心你

一個企圖處理蒐集，整理，串連，行動，追蹤的去中心化群眾參與社會議題平台。
九個月來，經過無數次的 mockup 討論與修改，以及 UC Bekeley CS 學生的幫忙之下，我們終於稍稍窺見一點 kuansim 的面貌。
prototype demo: [http://dogfood.kuansim.org](http://dogfood.kuansim.org)

這個計畫最終想達成的是，對於你關心的議題，讓你蒐集及整理相關資料，找出與你立場相近的人一起採取行動，然後追蹤後續的狀況，直到達到你期望的結果。

更多資訊可以看: [blog.kuansim.org](http://blog.kuansim.org)
