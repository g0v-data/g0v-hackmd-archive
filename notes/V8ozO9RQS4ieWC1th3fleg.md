# [懶人包] 寫給對於還不熟悉 Meshtastic 的朋友

作者：@dryden（g0v slack）

source: https://www.facebook.com/groups/meshtastictw/posts/413633161045882/
寫給對於還不熟悉 Meshtastic 的朋友： 

Meshtastic 它是一個開放原始碼的專案，可以用來傳輸訊息  
有著以下特點：

-   低耗電
-   不需透過電信商基地台
-   長距離
-   單價便宜
-   不用執照等等的特點

因此我認爲，它或許會很適合做爲在臺灣，遇到大型災害時，斷電斷訊時的備援通訊方式：

大規模停電後 → 基地台備援用電用完 → 人們傳遞訊息的方式剩下無線電 → 屆時的無線電頻道應該跟上班時的日本地鐵一樣擠

\--- 
我在兩週前，被 YT 演算法推薦到一個英國大叔的 Meshtastic 影片（放在留言處）


看了以後覺得或許可以用在臺灣，因此就買了幾個來實際測試看看！

目前實測的心得： 
當我把裝置放在陽臺，我在戶外且可以看到陽臺的情況下 
距離 535 公尺是可以收到訊號的（如附圖）

而在國外的實測，傳輸距離是公里起跳，甚至可以到上百公里 
爲什麼國外的測試結果可以這麼遠？

除了有些人會用指向性天線以外， 
也是因爲這專案使用的 LoRa 技術的特性： 
只要是可視，就可以有長距離的傳輸 
但如果中間被太多建築物擋住，距離就會受限 
所以當我在都市內實測時，傳輸距離就有受限

那是不是就不適合拿來在臺灣使用了？非也 
Meshtastic 這個開源專案，就針對上述的缺點做了改良 
它讓每個使用者手上的裝置，都變成一個節點，會協助轉發訊息

因此假設目前有五個使用者，分別是 A B C D E 
今天即便 A 因爲距離限制，沒辦法傳給 E 
也可以透過轉發的方式，A → B → C → D 最後把訊息送到 E

（如附圖二）

也因此，開了這個社團，徵集大家一起來玩玩看這個小裝置， 
試驗看看是否能利用它的分散式設計， 
做爲一種臺灣的備援通訊方式

需要的裝備： 
1.Meshtastic 裝置：以臺灣目前可以快速買到的 Heltec ESP32 來說，一支只要 $760，如果是自己跟自己測試的話需要買兩支 $1,520

2.支援藍牙的 Android or iOS 手機

3.一顆行動電源，以 USB Type C 的方式供電給 Meshtastic 裝置

  
看到這裡

-   行動派的朋友，可以在線上購物買到上述的裝置
-   有興趣瞭解更多的朋友，可以點選留言的影片，瞭解更多細節 
-   有問題的朋友也可以直接發問，我目前也是才剛上手而已，可以就我知道的內容一起討論
