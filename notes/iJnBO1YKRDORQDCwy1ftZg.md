# 避難資訊快速查找應用 設計規範

說明開發一個「避難資訊快速查找應用」的規劃細節與技能需求，為有相關軟體開發技能的人士鋪路

## 技能要求<br>Skill prerequisites

* Web(PWA)/Android/iOS/其他平台 的應用軟體開發能力
* 地理位置-鄉鎮市區反查 API 操作能力

## 設計規範<br>Design specifications

* 定期查詢使用者終端裝置的所在地理位置，反查並定期下載附近鄉、鎮、區、里的避難資訊讓使用者在事件發生當下網路可能不暢通的情況下能夠快速存取避難指示（沒用到的資訊可以自動清除以節省裝置儲存空間）
* 允許使用者自行（透過地圖或是其他方式（如附近地區））下載指定的避難指示資源（此資料應獨立於前項自動獲取的項目管理，避免誤刪）
* 允許使用者透過第三方地圖（導航）應用軟體開啟避難場所的 PoI 地標
