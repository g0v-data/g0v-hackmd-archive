-----------------------------------------------------------------------------
UVA集合 A={a_1, a_2, … , a_k} 半徑Q
UGV集合G={g_1, g_2, … , g_m} 半徑R
目標集合O={o_1, o_2, … , o_n} 

O的位置L -> o_iL(t_j)  // t_j時間點o的位置

o的速度 = ⅆ(o_i L(t_j )  ,〖  o〗_i L(t_(j-1))  )/(t_j-t_(j-1) )



---

分配
g_i C(t_j ) ={o_a, …}

*完全cover
∀o_a  ∈ O
∃i o_a  ∈ g_i C(t_j ) ^
∃i o_a  ∈ a_i C(t_j )




*互斥
∀i,k 
g_i C(t_j ) ∩ g_k C(t_j ) = ∅
a_i C(t_j ) ∩ a_k C(t_j ) = ∅
g_i C(t_j ) ∩ a_i C(t_j ) =  ∅  // only 追 outlier情況= ∅


*合法
∀o_k  ∈ g_i C(t_j )
d(o_kL(tj) , g_iL(t_j) ) ≤ R
d(o_kL(t_j) , a_iL(t_j) ) ≤ Q
