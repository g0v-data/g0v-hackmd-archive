# 2-2 心法

隨著數位的發展，人們表達的管道與方法越發多元，「聲音」可以發現「」
- 學習如何從情感引領到思考、從思考化為行動？（不要純批評以及純情緒，要行動）
- 集體智慧：群眾智慧＋人工智慧
- 不要問為什麼沒有人做這個，先承認你就是那個沒有人。
- 尊重現實，尋求解方
- Release Early, Release Often. 盡早釋出，經常釋出
- Talk is cheap, show me the code. 空話少說，放碼過來
- Everyone's business, everyone helps. 眾人之事，眾人助之
