# da0新手教學

這裡寫著如何使用da0所使用的虛擬貨幣工具，歡迎參考！

## 教學

### 安裝 Metamask 錢包

請先開啟[metamask的頁面](https://metamask.io/download/)。請注意網址，網址應為：https://metamask.io/download/

:::danger
注意：錯誤的網址會下載有問題的外掛，導致您的錢包被入侵。
:::

請點選下面的 Install MetaMask。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_f763b52b4183f015835c65fcafac863c.png =400x)

點選後，畫面上方會出現小視窗，請點選「安裝」。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_cf29d78abdfd06b044fe277a793fe4b4.png =400x)

安裝好套件後，可以點選「開始使用」，以開始使用MetaMask。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_1a5b5e3079d8d18973fe5eda60daa86a.png =400x)

### 開啟自己的錢包

點選「開始使用」後，MetaMask會先問你要不要讓他們收集使用的資料，這個看個人，都可以。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_a7f6810fd0d5e83ecd90fb4645ccf259.png =400x)

再來，MetaMask錢包會問您如何開啟錢包。若您已經有註記詞，可以選左邊「匯入錢包」；若沒有，可以點選右邊的「創建錢包」。以下以「創建錢包」為例，說明如何開啟自己的第一個錢包。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_f2faca83cee6abf0ab93b32c30bfbddd.png =400x)

點選開啟錢包後，會先要求您輸入密碼。這是您在本機上使用這個錢包的密碼，當有需要的時候，會跳出來要求您輸入，以避免有人偷偷使用電腦時盜用您的數位資產。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_a295fe22c03d5ab40ebc6f6da7c63ee0.png =400x)

接下來，會有影片說明如何保護您的錢包與資產，以免遭到盜用。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_8a8447c0da4cde30301f9b70ffb4452b.png =400x)

接下來會顯示註記詞，請點選中間的灰色區塊以顯示您的註記詞。註記詞是整個錢包最重要的私鑰，請絕對不要分享給其他人。

建議您可以用1Password等軟體來儲存，或是寫在紙上收好。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_9bea3f42240406692480fd1e36666d35.png =400x)

接下來，MetaMask會確認你已經保存好註記詞了，請依照順序來點選。若您發現自己可能記錯，可以點選上方的「上一頁」回去重新看一次註記詞。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_e790f9442aa1074269bb98582634c2a5.png =400x)

確認你已經保存註記詞以後，網頁就會出現這個頁面。請點選「都完成了」完成錢包的開設。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_3c985aba14ee2817a44cc8ab3fdc5a67.png =400x)

這是您之後會看到的錢包介面。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_0da738d646b63502b6f2926583a00f9b.png =400x)

### 取得錢包地址

在錢包的介面上方，有串「0x」開頭的字樣，把滑鼠移過去，就會看到「複製到剪貼簿」的字樣。請點選他，就可以複製你的錢包地址。

:::info
以太坊等類似的區塊鏈，錢包地址會以「0x」開頭。
:::

有了錢包地址，大家就能轉錢或其他NFT給您。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_6bf6fca79ad86ccf9903e3e48636d183.png =400x)

### 新增網路

DA0的東西會放在Polygon這個鏈上，交易費用比較低。不過，MetaMask需要新增這個鏈，才會切換過去。

打開錢包後，請先點選右上角的「以太坊 主網路」，再點選「新增網路」的按鈕。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_bbc3c8a9ac1464ee2dbcb30c0bdabf16.png =400x)

若您是手機版MetaMask，請點選此處以切換網路。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_c004fa031aa9aebc42ca7c090529f868.png =400x)

接著，MetaMask會展示一些常用的網路。「Polygon Mainnet」就是我們要加入的網路。請點選右邊的「Add」。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_ae6715084bfd7ed27d60de02315e7fc7.png =400x)

接下來MetaMask會詢問你是否確定要加入，請點選「Approve」允許加入。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_92b64521deadb3a270c37ec5af4ff631.png =400x)

若您希望切過去該區塊鏈，可以選擇「Switch to Polygon Mainnet」切過去使用Polygon這個區塊鏈。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_a822cd6bb0cdbdb0797f50180f4c0683.png =400x)

MetaMask會跳出頁面告訴你已經切過去了。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_640669e755791407868bde57e37c4787.png =400x)

### 自行添加 Polygon Mumbai測試網

由於DA0的許多機制還在測試中，因此暫時會先使用測試網站來進行相關的測試。這邊說明如何加入Polygon Mumbai這個測試用的Polygon區塊鏈。

請先點選右上角的網路分頁，再點選「新增網路」。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_f15faf99f8dd743587d469f27e237c6f.png =400x)

在新增網路的介面中，請點選下面的「Add a network manually」。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_663c933d79c82df81009b9b9d27f58ae.png =400x)

在出現的頁面中，依序輸入以下資訊：

:::info
* Network Name: `Polygon Mumbai Testnet`
* New RPC URL: `https://rpc-mumbai.maticvigil.com`
* Chain ID: `80001`
* Currency Symbol: `MATIC`
* Block Explorer URL: `https://mumbai.polygonscan.com/`
:::

輸入完成後，請點選「儲存」。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_e7c1f5b538f4ac8806009258a6cee737.png =400x)

手機版介面如下：

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_a5e33515c7702ccc9bbf149f3d97764c.jpg =400x)

加入後，手機版MetaMask會跳出通知：

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_b6567e979be46780418ea72259ccc170.jpg =400x)

之後，在上面的網路分頁，就能切換到「Polygon Mumbai Testnet」了！

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_59f893da79db6e91823119bc9234b946.png =400x)

最後，你會看到右下角有小綠框，告訴您已經加入「Polygon Mumbai Testnet」了。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_cbf9d97c09980cb36ac33c60ad57e2fc.png =400x)

### 加入DA01測試幣

目前DA0正在測試網上發測試幣，測試整個機制。不過，這個代幣（Token）需要自行加入，才能看到。

首先，請先點選網路，確認自己已經切換到「Polygon Mumbai Testnet」。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_59f893da79db6e91823119bc9234b946.png =400x)

接著請點選下面的「Import Tokens」。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_e1bbc00e74703145657128a5f03d7892.png =400x)

在出來的頁面中，新增以下資訊：

:::info
* 代幣合同位址： `0x5D231b3E2e1Fcd7fda9169fC71a2e93CDC038180`
* 代幣代號：`DA01`（會自動帶入，其實不用自行輸入）
* 小數點精度：`0`
:::

新增以上資訊後，請點選下方的「Add custom token」

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_b228f90413bd7549674a5f172b862c96.png =400x)

MetaMask會請你確定是否要加入代幣，請點選「Import tokens」加入。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_dbb4d3c0b3c88e2f1ca8c0af6c481989.png =400x)

之後，就可以在MetaMask介面上看到DA01，也可以操作了。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_829185a6b5d22ea610af3337ffcf3a0d.png =400x)

### 取得測試用Matic代幣

區塊鏈上的操作都需要支付該鏈的代幣，以作為手續費（gas）使用。Polygon上的手續費需要使用Matic代幣。

所幸，我們使用的是測試網，因此我們可以索取一些Matic代幣來使用。

請先在錢包介面上，點選自己Account下面的地址，複製你的錢包地址。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_baee282134638fc7bb5ea2a8cefe935b.png =400x)

接著，打開這個發送測試幣的網址：

https://faucet.polygon.technology/

確認Network在「Mumbai」，Token是「MATIC Token」。

接著請在Wallet Address中輸入剛剛複製下來的錢包地址，再點選「Submit」。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_9a205052126c0a9ac1c265faac616031.png =400x)

這邊會請你確認資訊是否正確，確認後，請點「Confirm」。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_6521294de65a044a7fc39598a8b3e1d5.png =400x)

發送成功後，會出現這個視窗，代表您已經索取完畢了。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_5db9724b6fb670db9e2530da09393717.png =400x)

若您看到的是這個視窗，代表您已經有Matic了，不需要再索取測試用的Matic。

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_716a12e671d5e89ed47f32a1d40ec040.png =400x)

稍等一兩分鐘，您就會看到錢包中出現一些Matic了！

![](https://s3-ap-northeast-1.amazonaws.com/g0v-hackmd-images/uploads/upload_b54d8e57f4b47ed836cbee24f2b84356.png =400x)
