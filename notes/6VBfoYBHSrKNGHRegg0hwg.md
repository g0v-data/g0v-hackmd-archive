#### polis 
- 目前的報告：https://polis.tw/report/r3janntdhm9ynaeydxjhk
- 意見徵集的連結：https://polis.tw/9xeamzkbjp 
- raw data csv：

| A 組                     | B 組          |
| ----------------------- | ------------ |
| 對新增平台義務較保留              | 平台應負更多責任     |
| 對政府介入保持警覺               | 政府應建立制度      |
| 對 Safety by Design 支持度低 | 高度支持預設安全     |
| 不支持平台事前措施               | 支持平台與制度共同治理  |
| 對教育措施也未表達支持             | 教育、平台、政府共同合作 |


#### mentimeter
- 對兒少開放的 AI 服務，應依不同年齡設計不同功能權限，而不是讓所有使用者接觸完全相同的生成能力。年齡驗證
- 政府與平台不應把兒少 AI 安全責任完全推給家長與老師，而應提供清楚的通報流程、教學資源與技術工具。
- 如同菸、酒、色情內容的問題，兒少保護不能只靠年齡限制，而是要靠家庭、學校、社會系統的真實陪伴
