# 克難臨時激動轉播點設立法

> [點此觀看原始內容](https://g0v.hackpad.tw/e2uv6HB47HT)


### 摘要

全球移動  WiMax AP -> USB -> Laptop + HD WebCam -> Google Hangout

### 設備

- 全球移動大力丸關掉Wifi, 只使用USB連到Laptop
- HD WebCam

### 步驟


1.  關掉大力丸Wifi
2.  裝Google Hangout
3.  發起Google Hangout 直撥
- [https://www.google.com/+/learnmore/hangouts/onair.html](https://www.google.com/+/learnmore/hangouts/onair.html)
- 直播之
4.  裝上Webcam
5.  改變錄影  Source
6.  發布連結。
