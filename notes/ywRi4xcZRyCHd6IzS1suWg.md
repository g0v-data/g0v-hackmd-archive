# IssueTaiwan 台灣好問題

> [點此觀看原始內容](https://g0v.hackpad.tw/OCelk22n02v)


- **要解決的問題：**
1.  『我背後有民意』，可是民意，在哪裡？-->台灣政治議題決策與人民意見的脫鉤
2.  議題風向讓「有心人」操作,無法說服大眾,導致台灣民間及網路環境充滿酸民及負面說法。
3.  失落的公民教育
4.  女性對公共議題的關注與發言聲量低於男性
5.  熱血的人很多，但是工具散落在各處。


- **建議解決方案：**
### _一個屬於全民的中立“議題投票”民主平台_

1.  篩選出台灣十大議題，並將正反兩方的意見都放在下方( [Johnson Liang](https://g0v.hackpad.tw/ep/profile/z4JarLrJBea) 將網路新聞連結跟分類）
2.  資訊來源/新聞來源 必須為”
3.  降低門檻，加速並幫助民眾快速了解政策。透過視覺化與畫重點等開放編輯的方式，邀請使用者加入共同完成。
4.  邀請全民上網來看完相關資料，慎重的投票，透過對議題的表態，凝聚最大民意共識，從而影響政府施政方向與民意代表。
5.  民眾上網需以「Facebook Connect 」，因此可以進行性別/地區做一個民意基礎的意向問卷結果
6.  在特定選區的民眾得以用“提案”方式，讓同選區的民眾也上網去投票，所呈現出來的民意得以讓候選人重視（將個人的力量聚合，成為真正不可被忽略的民意）

- **目的 --\> 功能：**
1.  提供全民參與議題的平台，讓總統/立委候選人重視到真正的民意導向
2.  補全失落的公民教育，增加公共議題的有趣性與親民程度。
3.  整合g0v之前的一些方案，整合為一個完整的公民擁有的民主平台，透過行銷並串連網路與傳統媒體，導入使用者流量，發揮真正的影響力。
