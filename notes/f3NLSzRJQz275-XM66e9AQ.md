# 個人化口罩需求計算機

### 初始想法
「個人化口罩需求計算機」透過問卷自動產生建議每個人在什麼時候需要用口罩、一週大概會需要用幾個、需要重複使用幾次。不過需要有防疫背景的人來確認數字會是正確的、還有研究一下算出來會減少口罩恐慌還是會增加。

#### note
* 目前政府限制每人每週可以買到兩片口罩，但其實每個人需求從0~n不等，是否有可能通過計算讓大家評估購買口罩的數量讓需求總量再下降？
* 是否有可能提倡家庭人口資源共享的概念？e.g 三口之家一個人生病的話那總共有六片口罩可以用，可以讓生病的人戴

## 問卷草稿
* 最近是否有上呼吸道症狀或是發燒？（感冒生病中） (true/false)
* 一週出門幾天？ (0~7)
* 是否搭乘大眾交通工具？ (true/false + 一週幾天搭乘大眾交通工具)
* 工作環境是否有多人且密閉？ (true/false + 幾天？)
* 工作環境是否會與許多人近距離接觸？ (true/false)
* 是否固定拜訪醫院？（幾次？） (0~7次)
* 是否常與免疫力低下人群接觸？（e.g. 老人、幼兒）
* 是否有慢性病？ (true/false)
* 是否從疫區回來？(true/false)

* 需要建立每個問題之間的關聯性
    * e.g. 呼吸道症狀 + 每週搭乘大眾交通運輸 = 一週需要七片口罩
    * e.g. 呼吸道症狀 + 每週出門一天 = 一週需要一片口罩

## 口罩用途
* 保護「免疫力低下人群」
* 減少「已患病者」病菌傳播
* 保護「高風險工作族群」如醫療人士與旅遊服務產業（待補充修正）
* 懶得化妝的去帶棉口罩就好了 :P
