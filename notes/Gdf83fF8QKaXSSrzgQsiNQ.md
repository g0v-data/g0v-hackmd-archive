# 爬蟲工具
## PyRailgun
 ### 簡介
>支持抓取javascript所渲染的頁面
>python網頁爬蟲模塊
>支持python2.7及python3.X
>授權協議:MIT
>作業系統:跨平台 Windows Linux OS X
>特點:簡潔、輕量、高效的網頁抓取框架
>https://github.com/princehaku/pyrailgun
>
