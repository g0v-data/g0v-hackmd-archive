# 2025-09-23 <br>花蓮馬太鞍溪堰塞湖溢流-災後清理工作 <br>
:::info
1. 避免個人前往，請與有組織的單位報名。(號召中的組織包括壯闊台灣、慈濟、卓溪新女力、楊華美議員辦公室、Amis中會等等)。
2. 審慎評估自行能力與專業幫助程度再前往，災區需基本的自我照顧能力，照顧好自己才能照顧別人。
3. 災區道路受損狀況嚴重，災後大型機具、救災車輛湧入，**目前進光復的車輛已經擁塞，如欲前往者，盡可能搭乘火車前往**。
4. 保護自身安全，尊重當地的需求，垃圾自己帶走。
:::


### 個人裝備：
* 個人防護
    * 防護手套（厚布或橡膠材質）
    * 防塵口罩
    * 防滑雨鞋或工作靴
    * 安全護目鏡（視現場情況而定）
    * 頭燈、備用電池(至少三組)
* 基本穿著
    * 雨衣、雨褲、或防水衣物
    * 快乾材質非棉麻換洗衣物(不要帶太多衣物行李，基地沒地方放）
* 其他
    * 個人餐具碗筷（減少在災區使用一次性餐具）
    * 個人藥物&醫藥防護包：優碘、酒精、OK繃、外用抗生素（軟膏或粉狀）、個人抗過敏藥物


### 清理工具：
* 鐵鍬 / 鋤頭 / 鏟子（主要用於清理泥沙和瓦礫）
* 撬棍
* 鋸子（手鋸或鏈鋸）
* 鐵鎚（用於拆除工作）
* 大型垃圾袋 / 集裝袋
* 大型垃圾桶（臨時堆置廢棄物）
* 清潔劑 / 消毒水
* 拖把、掃帚、抹布（視需求補充）

### 大型機具支援（視實際狀況調派）：
    * 小型挖土機 / 推土機（用於處理大規模瓦礫或道路障礙）
    * 發電機（若當地電力尚未恢復）
