# Django 實作練習

View the book with "<i class="fa fa-book fa-fw"></i> Book Mode".

安裝Django
---
- **首先在終端機上安裝pip套件**
:::info
  sudo apt-get install python-pip
:::
- **並利用pip套件來安裝Django**
:::info
  pip install django
:::

創建Django骨架(project)
---
- **先在終端機上把路徑指到您 Django 專案想要放的位置**
- **接著輸入以下指令，系統就會自動建立您第一個專案(專案名為 Demo)**
:::info
  django-admin startproject Demo
:::
