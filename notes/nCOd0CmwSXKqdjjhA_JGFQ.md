# 2024 夏日線上源力增能營結業式 x 專案實作工作坊

:::warning
**零時小學校 2024 夏日線上源力增能營**
- [營隊首頁](https://g0v.hackmd.io/@jothon/Sch001SSR2024/)
- [營隊報名連結](https://g0v-jothon.kktix.cc/events/sch001ssr2024)
- [零時小學校官方 Line ](https://lin.ee/mffftkf )
:::


有問題想解決，但不知道從何開始？
有專案想做，但不知道如何找人一起做？

零時小學校將於 8/24 舉辦半日實體「提案工作坊」，活動由零時小學校將引導學員從問題思辨與定義、解方發想，到團隊溝通與協作。

此外，活動也將邀請零時小學校專案孵化競賽過往獲獎團隊分享專案協作經驗，零時小學校也將分享專案共通性問題與解方分享，包含專案社群經營等。

:::info
:star:【專案實作工作坊 活動資訊】
- 時間：8/24（六） 13:00 - 16:00
- 場地：NPO HUB TAIPEI（台北市中正區重慶南路三段2號）
- 報名連結：https://forms.gle/1nwj7spEFkfNQFHG6
- 對象：任何想要了解黑客松提案流程，或想知道提案是怎麼一回事的夥伴！
:::

|15:40 - 16:00|源力增能營結業式x工作坊回饋|
