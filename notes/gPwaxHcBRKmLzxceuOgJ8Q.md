:::spoiler 抽油煙機、瓦斯爐
1. 抽油煙機筆記
   - 衡量標準：
     - 清潔難易度，因為油煙會卡在通過的各種地方
     - 吸氣效率（排風量 m^3/min）、寬度（愈寬可吸越多範圍）、深度
     - 噪音
     - 用電量
   - [這篇](https://go-ezbuy.com/range-hood/)有完整解釋
   - 各類型抽油煙機比較：
     - 傳統中式：便宜、吸氣效率高，但會撞到頭，而且醜醜
     - 近吸/側吸式：吸氣效率高，漂亮，不會撞到頭，但比較大台，離爐台近，熱度會被吸走
   - 選手們：
     - 櫻花 R7600 / R7602 ，近吸，18m^3/min 網路定價 15k
3. 瓦斯爐筆記
   - 內焰省能源，但火力太集中，不適合中華炒鍋
   - 檯面式的，要防滲漏，因為筒身是合板
   - 鄉民都說^[[ptt](https://www.ptt.cc/bbs/E-appliance/M.1479956538.A.724.html)]林內品質比較好，櫻花 G2522S、林內 RB-202GH
   - 選手們：
     - 櫻花 G2522S、林內 RB-202GH，二級能效，都是 8k-9k
:::

:::spoiler 冷知識
1. 一個 210cm 的一字流理臺，大概多少錢？包含水槽、龍頭
   1. 檯面以耐用程度而言：髮絲紋路不鏽鋼(30??/cm) > 人造石(45/cm) > 石英石(90/cm) ^[[社團討論：廚房枱面材質(先紀錄各時期討論) - 小院](https://www.courcasa.com/p/r70R)]
   2. 水槽大點比較好用，至少內徑 50cm
   3. 只有冷水的龍頭，會比較便宜嗎 XD
      > 沒這種東西，記得用三通把冷熱水都接上進水口，轉到熱水時，才不會沒水或漏水
   5. 各區域佔比：瓦斯爐 80cm ，水槽 80-90cm ，所以工作區只有 40-50cm
      - 檯面深度 60cm

### 08-17 天然氣公司估價

無論要怎麼做流理臺，都可能會動到天然氣位子。按照規定，更改天然氣管線，需要由瓦斯公司負責，否則後續有糾紛（例如漏氣）時，會很麻煩。

因為事小，而且公司反應超快，所以由 ddio 獨自進行。

:::spoiler 活動紀錄

雖然事情很小，但這是目前最順利的場勘 XD

週一下午申請估價，當天馬上約隔日早上，看完之後，一小時報價就出惹...

天然氣公司表示：

1. 軟管法規最長可到 1.8m ，要的話可請廚具、水電直接拉，就不用找公司（只是會多一條東西懸在空中）
2. 若要在室內裝 90 度的轉角開關，減少軟管迴旋面積，因為室內已經從六分管，縮小為四分管，所以有可能無法處理，要只能找水電來私接
3. 找公司做的話，建議的方案：
   1. 只改廚房管線，講好新的出口，瓦斯公司可負責改管 + 洗洞，4,995
   2. 因為管線老舊，建議整組管線直接更新，包含前項，延長使用年限，否則有可能會漏氣，11,070
4. 施工補充：
   1. 兩個工期都是一天，工資約 2k ，所以直上第二項，會省 2k
   2. 除了原本的廚房、熱水器外，屋主有私接管線，到時後要封掉，不然公司不會承接
   3. 開關都可沿用，就是換管線
   4. 因為從外面接近陽台的地方，沒有被封死（例如已經埋在外推的陽台牆面裡），所以可以直接在陽台施工，不然就要搭鷹架了
5. 公司施工流程：
   1. 打電話約現場丈量（免費）
   2. 電話報價，確定的話，再約施工時間
   3. 現場施工
   4. 因為是外包工班，所以須轉帳或現場付現，無法記在瓦斯帳單內


:::

#### 廚房

目前的流理台的問題：
1. 太矮
2. 表面老舊
3. 水槽太小
4. 收納空間不足
5. 抽油煙機位置不佳，須放在窗框上
6. 排水管位置離牆太遠，就算重作櫥櫃，內部也無法裝抽屜

解法1.做很深的一字，全部拆除
拆掉現在的流理台，裝新的一字型流理臺，增加收納空間，安裝抽油煙機在牆上。

考量：

1. 水管管線會擋到收納櫃，可能無法安裝。要確認水管的管線走向。
2. 預算較貴，B&Q師傅說十萬起跳。
3. 天然氣要用軟的管線拉到瓦斯爐。

解法2.拆水槽，只做一點新櫥櫃
為了不動神秘的水管空間，又要增加收納，讓部分空間可以符合工作習慣，可拆掉水槽部分，安裝特規裝流理臺，保留的原本流理台用木作架高，增加收納空間，瓦斯爐放在原本的流理台上，抽油煙機在窗上。

考量：

1. 可避開水管管線增加部分功能。
2. 預算較貴，B&Q師傅說十萬起跳。
3. 天然氣要用軟的管線拉到瓦斯爐。
4. 水槽靠外側，可避免水管問題。
5. 抽油煙機的高度較低


解法3.（B&Q師傅建議）保留原流理台，重新貼磁磚即可。

考量：

1. 預算較低。
2. 收納空間另外處理，可訂做櫥櫃。
3. 流理臺考慮用木板架高。
4. 抽油煙機的高度較低
5. 卡到既有天然氣管線。


後續：
只要拆流理台，就會需要泥作跟重新訂製新的流理台，從B&Q的師父回覆，預算似乎都會比預期高。需要再請系統櫥櫃廠商與水電師傅再評估水管的問題會不會影響那麼大以及估預算。


#### 電表
說明：目前僅有30安培，要增加到50安培使用需要重新拉線。
解法：

1. 相對於上一個水電師傅要另外拉一條線，重建一個電錶箱，B&Q的師傅建議用原本的電線管路增加電線。
2. 原本的電箱拉出來後可以新增一個延伸的電箱就好。
3. 延伸的電箱可以專注在廚房電器，原本的電箱則主要是冷氣以及其他加電器。

後續：
再請另一個水電師傅就兩個做法確認可行性。

#### 地板

2. 請工班敲掉水泥結構，重鋪水泥與磁磚，避免後續再發生一樣的情況。

後續：
1.請問yutin地板拆除DIY可行性
2.若拆除風險有疑慮，又或拆除地板後，覺得需拆除結構，則未來請工班拆除水泥結構。
