# 衛福部-食品營養成分資料集，資料圖像化

> [點此觀看原始內容](https://g0v.hackpad.tw/j2d8Uur2EJO)

- Why? ：
            近年來食安問題逐漸被重視，選擇自己下廚的人很多，但食品成分的詳細資料難以取得，
            政府雖然製作了食品營養成分資料集，但是呈現方式使一般人無法使用，所以產生了將該
            資料集製作成web app並圖像化的想法，使一般民眾可以上去查詢、過濾想要查詢的食物，
            並獲得該食物的詳細成分圖表(詳細至胺基酸、礦物質種類等等)
- How? ：
            引用政府開放資料集，整理匯入SQL資料庫，製作成互動式網站。
            授權方式 : 政府資料開放授權條款-第1版

- Need?：
            政府開放資料:[https://data.gov.tw/dataset/8543](https://data.gov.tw/dataset/8543)
            ASP.NET
            SQL
            Javascript
            HTML/CSS
