# Good Firtst Task

多人協作翻譯工具

主要功能：

1. 需求者：上傳需要翻譯的文章
2. 翻譯者：翻譯與驗收翻譯內容


Story：


- 上傳原文

需求者使用 Google OAuth 登入後台 -> 點擊「上傳原文」 -> 將「原文文章內容」整篇貼上 -> 輸入「原文標題」-> 選擇「原文語言」 -> 複選「需要翻譯的語言」 -> 點擊「確認上傳」完成上傳

- 進行翻譯

翻譯者點擊「文章」-> 選擇要查看的語言 -> 查看所有行數已翻譯的狀況 -> 點擊「要翻譯的行數」進入翻譯介面 -> 於翻譯介面上輸入翻譯內容 -> 點擊「翻譯完成」將翻譯結果上傳

- 進行驗收

翻譯者點擊「文章」-> 選擇要查看的語言 -> 點擊「已完成的翻譯」 -> 點擊「確認驗收」完成驗收

- 隨機翻譯
- 隨機驗收

- 提供翻譯建議
