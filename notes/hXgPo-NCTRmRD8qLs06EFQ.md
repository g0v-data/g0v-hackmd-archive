# 公民行動方案：南港小基站錯誤時區案例
## 🎯 問題背景

南港某處 4G/5G 微型基地台設定錯誤，導致手機經過時自動跳到 UTC+7 時區。
此狀況可能影響手機時間同步，進而造成 金融交易、通訊、定位、醫療等服務異常。
需要透過 公民檢測 → 蒐證 → 通報 → 追蹤 的方式，促使管理單位修正。

---

## 📝 行動步驟

現場檢測：手機 App / SDR 測量 → 確認問題。
精準定位：必要時用 SDR + 定向天線比對。
證據收集：表格填寫 + 照片 + 截圖。
對外通報：電信公司客服 + NCC 1915。
後續追蹤：定期複測，更新結果，若未改善 → 媒體/民代關注。

---

```

    A[現場檢測<br>手機 App / SDR] --> B{是否發現時區異常?}
    B -- 否 --> Z[結束<br>無異常]

    C --> D[證據收集<br>表格 + 照片 + 截圖]
    D --> E[對外通報<br>電信公司客服 + NCC 1915]
    E --> F{業者是否修正?}
    F -- 是 --> G[後續追蹤<br>再次檢測確認]
    F -- 否 --> H[擴大行動<br>媒體/民代關注]
    G --> I[完成案例<br>建立公民觀測紀錄]
    H --> I

```

## ✅ 預期成果

鎖定問題基地台位置並蒐證完整紀錄。
向電信公司與 NCC 提出正式通報。
問題獲得修正，減少民眾因錯誤時區產生的困擾。
累積經驗 → 發展為 「公民電信觀測行動手冊」。
