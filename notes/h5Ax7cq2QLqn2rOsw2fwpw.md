# Part 1：符號與基本概念（看懂講義的第一步）

## 1.2 向量（Vector）— 從零開始

### 什麼是向量？

向量就是「一列數字」。比如你去便利商店買了 3 瓶水和 2 包餅乾，可以用向量表示：

```
x = [3]    ← 這是一個 ℝ² 的向量（2個數字排在一起）
    [2]
```

更正式地說，ℝⁿ 中的向量就是 n 個實數排成一列：

```
x = [x₁]
    [x₂]
    [ ⋮ ]
    [xₙ]
```

### 向量的基本運算

**加法**（對應位置相加）：

```
[1]   [4]   [1+4]   [5]
[2] + [5] = [2+5] = [7]
[3]   [6]   [3+6]   [9]
```

**純量乘法**（每個元素乘以一個數字）：

```
    [1]   [3]
3 × [2] = [6]
    [3]   [9]
```

### 💡 為什麼要學向量？

因為優化問題的「決策變數」通常不只一個。比如你要同時決定「廣告預算」和「人力成本」來最小化總花費，這兩個變數就可以包成一個向量 x = [x₁, x₂]ᵀ。

## 1.3 矩陣（Matrix）— 向量的延伸

### 什麼是矩陣？

矩陣就是「數字排成的矩形表格」。一個 m×n 的矩陣有 m 列（row）、n 行（column）：

```
A = [a₁₁  a₁₂  a₁₃]     ← 這是 2×3 的矩陣 (ℝ²ˣ³)
    [a₂₁  a₂₂  a₂₃]
```

### 矩陣乘法（最重要的運算！）

A 是 m×n 矩陣，x 是 n×1 向量，則 Ax 是 m×1 向量。

**規則**：第 i 列的結果 = A 的第 i 列跟 x 做「內積」（對應相乘再加總）。

```
[2  3] × [4] = [2×4 + 3×5] = [23]
[1  0]   [5]   [1×4 + 0×5]   [4 ]
```

### 轉置（Transpose）

把矩陣「翻轉」——列變行，行變列。記作 Aᵀ（讀作 "A transpose"）。

```
A = [1  2  3]        Aᵀ = [1  4]
    [4  5  6]              [2  5]
                           [3  6]
```

對向量來說：如果 x 是行向量（column vector），xᵀ 就是列向量（row vector）。

### 共軛轉置（Hermitian Conjugate）†

如果矩陣裡有複數，除了轉置還要取「共軛」（把 i 變 -i）。

```
M = [1+i    2  ]        M† = [1-i   3+i]
    [3-i   4+2i]              [2    4-2i]
```

**如果矩陣全是實數，M† 就等於 Mᵀ。** 所以你看到 † 的時候，對實數矩陣直接當轉置就好。

---

## 1.4 特殊矩陣（考試超愛考！）

### 對稱矩陣（Symmetric Matrix）

A = Aᵀ。也就是沿著主對角線鏡像對稱。

```
A = [1  2  3]
    [2  5  6]    ← 注意 (1,2)位=2，(2,1)位也=2
    [3  6  9]
```

**記號**：Sⁿ 表示所有 n×n 對稱矩陣的集合。

### Hermitian 矩陣

M = M†。是對稱矩陣在複數世界的延伸。

**重要性質**：Hermitian 矩陣的特徵值一定是實數！

### 單位矩陣（Identity Matrix）I

對角線全是 1，其他全是 0。任何矩陣乘以 I 都不變（就像數字乘以 1）。

```
I = [1  0  0]
    [0  1  0]        AI = IA = A
    [0  0  1]
```

### Unitary 矩陣

U†U = I = UU†。可以想成「旋轉」——長度和角度都不會改變。

對實數矩陣，unitary 就是 orthogonal（正交矩陣）：QᵀQ = I。

# Part 2：特徵值分解（Eigenvalue Decomposition）

## 2.1 什麼是特徵值和特徵向量？

### 直覺

想像你對一個向量 v 做矩陣乘法 Mv。通常結果會改變方向和長度。但如果某個特殊的 v 乘完 M 之後**方向不變**，只是伸縮了，那這個 v 就是**特徵向量（eigenvector）**，伸縮的倍率 λ 就是**特徵值（eigenvalue）**。

```
Mv = λv
```

- M：n×n 矩陣
- v：特徵向量（非零向量）
- λ：特徵值（一個數字）

### 例子

```
M = [2  1]     v = [1]     Mv = [2×1+1×1] = [3] = 3 × [1]
    [1  2]         [1]          [1×1+2×1]   [3]       [1]
```

所以 v = [1, 1]ᵀ 是特徵向量，λ = 3 是特徵值。

## 2.2 譜分解（Spectral Decomposition）

### 定理（講義 p.14）

如果 M 是 **normal matrix**（即 MM† = M†M），那麼 M 可以分解為：

```
M = UΛU†
```

- U：unitary 矩陣（行就是特徵向量）
- Λ = diag(λ₁, ..., λₙ)：對角矩陣，裡面放特徵值

### 為什麼重要？

因為一旦分解成這個形式，很多運算變得超簡單：

```
M² = UΛ²U†         （特徵值平方就好）
M⁻¹ = UΛ⁻¹U†       （特徵值取倒數就好，前提是都不為0）
M^(1/2) = UΛ^(1/2)U†  （特徵值開根號就好）
```

### 考試重點

**Hermitian 矩陣一定是 normal**（因為 M = M†，所以 MM† = M² = M†M），因此一定可以做譜分解，而且特徵值一定是**實數**。

## 2.3 正半定矩陣（Positive Semidefinite, PSD）⭐⭐⭐

### 這是整門課最重要的概念之一！

**定義**（講義 p.16）：一個 Hermitian 矩陣 M ∈ ℂⁿˣⁿ 是 positive semidefinite（PSD），如果對**任意**向量 x：

```
x†Mx ≥ 0
```

記號：M ≥ 0 或 M ⪰ 0。

**等價條件**：M 是 PSD ⟺ M 的所有特徵值 ≥ 0。

### 直覺理解

x†Mx 可以想成一個「能量」的概念。PSD 代表不管你輸入什麼向量，算出來的「能量」都不會是負的。

### Positive Definite（PD）

更嚴格：x†Mx > 0 對所有 **非零** x 成立。

等價於：所有特徵值 > 0。

記號：M > 0 或 M ≻ 0。

### 例子

```
M = [2  1]    特徵值為 3 和 1，都 > 0 → M 是 positive definite
    [1  2]

驗證：x†Mx = [x₁ x₂] [2  1] [x₁] = 2x₁² + 2x₁x₂ + 2x₂²
                       [1  2] [x₂]
    = x₁² + (x₁+x₂)² + x₂² ≥ 0  ✓
```

### 記號整理（講義 p.17）

| 記號 | 意思 |
|------|------|
| Sⁿ | n×n 對稱/Hermitian 矩陣的集合 |
| Sⁿ₊ | Sⁿ 中的 PSD 矩陣集合 |
| Sⁿ₊₊ | Sⁿ 中的 PD 矩陣集合 |

### PSD 矩陣的各種「次方」

如果 M ∈ Sⁿ₊，M = UΛU†，Λ = diag(λ₁, ..., λₙ)，λⱼ ≥ 0，那麼：

| 運算 | 公式 | 條件 |
|------|------|------|
| M^(1/2) | UΛ^(1/2)U† | λⱼ ≥ 0 即可 |
| M⁻¹ | UΛ⁻¹U† | 需要所有 λⱼ > 0（即 M ≻ 0）|
| M⁻^(1/2) | UΛ⁻^(1/2)U† | 需要所有 λⱼ > 0 |
| M̃（pseudoinverse）| UΛ̃U† | Λ̃ 對非零特徵值取倒數，零保持零 |

# Part 3：Norm（範數）— 衡量「大小」的工具

## 3.1 什麼是 Norm？

Norm 就是衡量向量「有多大」的一種方式。就像你量一把尺的長度一樣。

### 四個必要條件（定義，講義 p.27）

一個函數 f: ℝⁿ → ℝ 是 norm，記作 ‖x‖，需要滿足：

1. **非負性**：‖x‖ ≥ 0
2. **正定性**：‖x‖ = 0 ⟺ x = 0（只有零向量的 norm 是 0）
3. **齊次性**：‖tx‖ = |t| · ‖x‖（拉伸 t 倍，norm 也變 t 倍）
4. **三角不等式**：‖x+y‖ ≤ ‖x‖ + ‖y‖（繞路不比直走短）

## 3.2 ℓp-Norm（最常見的一族）

對 v = [v₁, v₂, ..., vₙ]ᵀ，p ≥ 1：

```
‖v‖_p = (|v₁|^p + |v₂|^p + ... + |vₙ|^p)^(1/p)
```

### 三個最重要的特例

**ℓ₂-norm（Euclidean norm，歐幾里得距離）**：

```
‖x‖₂ = √(x₁² + x₂² + ... + xₙ²)
```

就是你最熟悉的「距離公式」！比如 x = [3, 4]ᵀ，‖x‖₂ = √(9+16) = 5。

**ℓ₁-norm（Manhattan distance，曼哈頓距離）**：

```
‖x‖₁ = |x₁| + |x₂| + ... + |xₙ|
```

走「田字格」的距離。比如 x = [3, -4]ᵀ，‖x‖₁ = 3 + 4 = 7。

**ℓ∞-norm（Infinity norm）**：

```
‖x‖∞ = max{|x₁|, |x₂|, ..., |xₙ|}
```

取最大的那個分量。比如 x = [3, -4]ᵀ，‖x‖∞ = 4。

**重要引理**（講義 p.29）：

```
lim_{p→∞} ‖x‖_p = max_i |xᵢ| = ‖x‖∞
```

### Norm Ball

以 x 為圓心、r 為半徑的 norm ball：

```
B(x, r) = {z ∈ ℝⁿ : ‖z - x‖ ≤ r}
```

不同 norm 的 ball 長得不一樣：ℓ₂ 是圓形，ℓ₁ 是菱形，ℓ∞ 是正方形。

### 🌟 關鍵性質（講義 p.32）

**Every norm is convex.** （每個 norm 都是凸函數）

這個性質後面會很常用到！

## 3.3 Quadratic Norm（二次範數）

如果 P ∈ Sⁿ₊₊（正定矩陣），可以定義：

```
‖x‖_P = √(xᵀPx) = ‖P^(1/2)x‖₂
```

這是 ℓ₂-norm 的推廣——用 P 來「扭曲」空間中各方向的距離衡量。

# Part 4：內積（Inner Product）

## 4.1 定義

內積 ⟨·,·⟩ 是把兩個向量變成一個數字的運算，需要滿足：

1. **第一個引數線性**：⟨αu + βv, w⟩ = α⟨u, w⟩ + β⟨v, w⟩
2. **共軛對稱**：⟨v, w⟩ = ⟨w, v⟩* （*是取共軛；實數情況下就是 ⟨v,w⟩ = ⟨w,v⟩）
3. **正定性**：⟨v, v⟩ ≥ 0，等號⟺ v = 0

### 最常見的內積

對 x, y ∈ ℝⁿ：

```
⟨x, y⟩ = xᵀy = x₁y₁ + x₂y₂ + ... + xₙyₙ
```

對複數向量：⟨x, y⟩ = x†y = Σ xᵢ* yᵢ

### 由內積產生的 Norm

```
‖x‖ = √⟨x, x⟩
```

比如標準內積產生的就是 ℓ₂-norm。

## 4.2 Cauchy-Schwarz 不等式 ⭐

```
|⟨x, y⟩| ≤ ‖x‖ · ‖y‖
```

等號成立 ⟺ x = cy（x 和 y 平行）。

**直覺**：兩個向量的「相關性」不會超過它們各自的「大小」。

## 4.3 Frobenius 內積（矩陣的內積）

對 X, Y ∈ ℂᵐˣⁿ：

```
⟨X, Y⟩_F = tr(Y†X)
```

其中 tr 是 **trace**（跡）= 主對角線元素之和。

### Trace 的重要性質

1. tr(AB) = tr(BA) （交換律——注意只有在 trace 裡面才成立！）
2. tr(cA + B) = c·tr(A) + tr(B) （線性）

# Part 5：矩陣範數（Matrix Norms）

## 5.1 Frobenius Norm

```
‖M‖_F = √(tr(MᵀM)) = √(Σᵢ Σⱼ |Mᵢⱼ|²)
```

就是把矩陣所有元素平方加起來再開根號——跟向量的 ℓ₂-norm 概念一樣。

## 5.2 Operator Norm（Induced Norm）⭐

```
‖M‖_{a,b} = sup{‖Mu‖_a : ‖u‖_b ≤ 1}
```

**直覺**：矩陣 M 最多能把一個「單位大小」的向量拉伸多少倍。

### 最重要的特例：‖M‖_{2,2}

```
‖M‖_{2,2} = √(λ_max(MᵀM))
```

其中 λ_max 是 MᵀM 的最大特徵值。

**如果 M 是 Hermitian**：

```
‖M‖_{2,2} = λ_max(M)   （直接是 M 的最大特徵值）
```

### ℓ∞ Operator Norm

```
‖M‖_∞ = max_{i} Σⱼ |Mᵢⱼ|
```

每一列的絕對值之和，取最大的那一列。

### Hermitian 矩陣的特徵值界（講義 p.37，HW#1）⭐

如果 A 是 Hermitian，特徵值為 λ₁, ..., λₙ，那麼：

```
λ_min · xᵀx ≤ xᵀAx ≤ λ_max · xᵀx
```

對任何非零 x 成立。這個不等式非常常用！

# Part 6：Optimization Problem（優化問題）

## 6.1 Standard Form（標準形式）⭐⭐⭐

```
minimize    f₀(x)           ← 目標函數（要最小化的東西）
subject to  fᵢ(x) ≤ 0,  i = 1,...,m    ← 不等式約束
            hⱼ(x) = 0,  j = 1,...,k    ← 等式約束
```

### 各部分解讀

| 名稱 | 符號 | 意思 |
|------|------|------|
| 優化變數 | x ∈ ℝⁿ | 你要找的「最佳答案」|
| 目標函數 | f₀(x) | 你想讓它越小越好的東西 |
| 不等式約束 | fᵢ(x) ≤ 0 | 答案必須滿足的「限制條件」|
| 等式約束 | hⱼ(x) = 0 | 答案必須在這個「面」上 |
| 定義域 | D = ∩ dom fᵢ ∩ dom hⱼ | 所有函數都有定義的範圍 |

### 可行解（Feasible Point）

滿足所有約束的 x 叫做 feasible。如果存在 feasible x，問題就是 feasible；否則 infeasible。

### 最優值（Optimal Value）

```
p* = inf{f₀(x) : x is feasible}
```

- 如果找得到 x* 使得 f₀(x*) = p*，則 x* 是**最優解（optimal point）**。
- p* 可能是 -∞（問題 unbounded below）。
- p* 可能是 +∞（問題 infeasible）。

## 6.2 Infimum 和 Supremum（講義 p.7）

### inf（最大下界）

對非空集合 S ⊆ ℝ，inf(S) = ω 如果：

1. ω ≤ x，∀x ∈ S （ω 是下界）
2. 如果 c 也是下界，那 ω ≥ c （ω 是最大的下界）

**跟 min 的差別**：min 要求最小值真的在集合裡面，inf 不需要。

例子：S = (0, 1] = {x : 0 < x ≤ 1}
- min(S) 不存在（0 不在 S 裡面，但沒有其他數是「最小的」）
- inf(S) = 0（0 是最大的下界）

### sup（最小上界）

對偶的概念。sup(S) 是最小的上界。

### 💡 考試常見

講義 p.8 的例子：

- f(x) = eˣ，x ∈ ℝ：inf eˣ = 0（eˣ 永遠不會等於 0，但可以無限接近）
- f(x) = 1/x，x > 0：inf 1/x = 0

## 6.3 例題解析

### 例題 1（講義 p.9）

```
minimize    2x² + 3x + 1
subject to  x ≥ 0
```

**步驟 1**：配方。

```
2x² + 3x + 1 = 2(x + 3/4)² - 1/8
```

無約束最小值在 x = -3/4，此時 f = -1/8。

**步驟 2**：檢查約束。

x = -3/4 < 0，不滿足 x ≥ 0！所以 x = -3/4 不 feasible。

**步驟 3**：在可行域內找最小。

f(x) = 2x² + 3x + 1 在 x ≥ 0 範圍內是遞增的（因為 f'(x) = 4x + 3 > 0 when x ≥ 0）。

所以 p* = f(0) = 1，x* = 0。

### 例題 2：最短距離（講義 p.9）

```
minimize    x² + y²              ← 到原點的距離平方
subject to  x²/3 - 4xy + y² = 1  ← 在這條雙曲線上
```

這是一個有等式約束的優化問題。目標函數是凸的（二次函數），但約束不是仿射的，所以不是標準凸優化問題。

# Part 7：Convex Set（凸集）⭐⭐⭐

## 定義（講義 p.19）

集合 C ⊆ ℝⁿ 是**凸集**，如果對 C 中任意兩點 x, y 和任意 θ ∈ [0, 1]：

```
θx + (1-θ)y ∈ C
```

### 直覺

在 C 中任取兩點，連線上的所有點也都在 C 裡面。

### 判斷技巧

- ✅ 凸集：實心圓、三角形、長方形、半平面、整個 ℝⁿ
- ❌ 非凸集：甜甜圈、L形、星形、離散點的集合

### 凸組合（Convex Combination）

θx + (1-θ)y 叫做 x 和 y 的凸組合。可以推廣到多個點：

```
θ₁x₁ + θ₂x₂ + ... + θₖxₖ，其中 θᵢ ≥ 0 且 Σθᵢ = 1
```

# Part 8：Convex Function（凸函數）⭐⭐⭐

## 定義（講義 p.20）

f: D → ℝ 是凸函數，如果：

1. dom f = D 是凸集
2. 對所有 x, y ∈ D，θ ∈ [0, 1]：

```
f(θx + (1-θ)y) ≤ θf(x) + (1-θ)f(y)
```

### 直覺（超重要！）

函數圖形上任取兩點，連線段一定在函數圖形的**上方**（或剛好在上面）。

「碗形」= 凸函數。「山丘形」= 凹函數（concave）。

### 常見的凸函數

| 函數 | 定義域 | 凸/凹 |
|------|--------|--------|
| ax² + bx + c (a>0) | ℝ | 凸 |
| eˣ | ℝ | 凸 |
| \|x\| | ℝ | 凸 |
| -ln(x) | x > 0 | 凸 |
| ln(x) | x > 0 | 凹 |
| ‖x‖ (任何 norm) | ℝⁿ | 凸 |

### Concave Function（凹函數）

f 是凹函數 ⟺ -f 是凸函數。

# Part 9：Convex Optimization Problem（凸優化問題）⭐⭐⭐

## 定義（講義 p.21）

```
minimize    f₀(x)           ← 凸函數
subject to  fᵢ(x) ≤ 0      ← 凸函數
            hⱼ(x) = 0       ← 仿射函數（hⱼ(x) = aⱼᵀx + bⱼ）
```

### 為什麼凸優化這麼重要？

**局部最優 = 全域最優！** 這是凸優化最大的優勢。一般的優化問題可能有很多局部最小值，你找到一個不知道是不是全域最好的。但凸優化問題不會有這個煩惱。

# Part 10：Linear Program (LP)（線性規劃）

## Standard Form（講義 p.22）

```
minimize    cᵀx + d
subject to  Gx ≤ h     ← 向量不等式（每個分量都 ≤）
            Ax = b
```

### 特點

- 目標函數和約束都是**仿射（affine）**的
- 可行域是一個**多面體（polyhedron）**
- 可以用 **simplex method** 求解

### 例題（講義 p.22）

```
maximize    x + y
subject to  3x + y ≤ 10
            x + 3y ≤ 10
            x ≥ 0, y ≥ 0
```

先畫出可行域（四條線圍出的區域），然後把目標函數 x + y = c 的等高線往右上推，最後碰到可行域邊界的點就是最優解。

**注意**：maximize 要改成 minimize -(x+y) 才符合標準形式。

# Part 11：Least Squares（最小二乘法）

## 定義（講義 p.23）

```
minimize    ‖Ax - b‖₂²
```

### 展開

```
f(x) = (Ax - b)ᵀ(Ax - b) = xᵀAᵀAx - 2bᵀAx + bᵀb
```

這是一個**二次函數**，可以證明是凸的（因為 AᵀA 一定是 PSD）。

### 解

對 f 取梯度令其為零：∇f = 2AᵀAx - 2Aᵀb = 0，得到**正規方程（normal equation）**：

```
AᵀAx = Aᵀb
```

# Part 12：Chebyshev 近似與等價問題轉換（講義 p.24）

## 原始問題

```
minimize    max_{i=1,...,k} |aᵢᵀx - bᵢ|
```

這個 max 讓它看起來不是標準形式。

## 等價轉換

引入新變數 t：

```
minimize    t
subject to  |aᵢᵀx - bᵢ| ≤ t,   ∀i
```

進一步拆開絕對值：

```
minimize    t
subject to  aᵢᵀx - bᵢ ≤ t,   ∀i
           -(aᵢᵀx - bᵢ) ≤ t,  ∀i
```

這就是一個 **LP**！

### 💡 考試技巧

遇到 min-max 或絕對值優化問題，經常可以引入輔助變數 t 轉換成 LP。

# Part 13：Generalized Inequality 與 SDP（講義 p.11）

## 推廣的不等式

| 比較對象 | a ≥ b 的定義 |
|----------|-------------|
| 實數 | a - b ≥ 0 |
| 向量 | aᵢ ≥ bᵢ，∀i（分量逐一比較）|
| 矩陣 | A - B ⪰ 0（A-B 是 PSD）|

## Semidefinite Program (SDP)

當約束涉及矩陣的 PSD 條件時，就是 SDP。這是比 LP 更廣泛的凸優化問題。

# Part 14：Barrier Method 簡介（講義 p.25）

## 概念

把不等式約束轉化為目標函數中的「懲罰項」，用 log barrier：

```
minimize    f₀(x) - (1/t) Σ log(-fᵢ(x))
```

當 t → ∞ 時，解逼近原問題的最優解。

用 Newton's method 求解每個 t 值下的無約束問題，然後逐步增大 t。

# 📋 考試重點速查 Cheat Sheet

## 必背定義

1. **Convex Set**: θx + (1-θ)y ∈ C, ∀x,y ∈ C, θ ∈ [0,1]
2. **Convex Function**: f(θx + (1-θ)y) ≤ θf(x) + (1-θ)f(y)
3. **PSD**: x†Mx ≥ 0, ∀x ⟺ 所有特徵值 ≥ 0
4. **Norm 的四個條件**: 非負、正定、齊次、三角不等式
5. **Spectral Decomposition**: M = UΛU† (M normal)

## 必記公式

| 公式 | 用途 |
|------|------|
| ‖x‖_p = (Σ\|xᵢ\|^p)^(1/p) | ℓp-norm |
| ‖x‖∞ = maxᵢ \|xᵢ\| | Infinity norm |
| ‖M‖_{2,2} = √λ_max(MᵀM) | Operator norm |
| λ_min·xᵀx ≤ xᵀAx ≤ λ_max·xᵀx | Hermitian eigenvalue bound |
| ⟨X,Y⟩_F = tr(Y†X) | Frobenius inner product |
| AᵀAx = Aᵀb | Least squares normal equation |

## 必會題型

1. **判斷集合是否凸**：用定義或反例
2. **判斷函數是否凸**：用定義、二階條件（Hessian ⪰ 0）
3. **把問題寫成標準形式**：特別是 min-max → LP 的轉換
4. **PSD 判斷**：計算特徵值或用定義驗證
5. **Norm 計算**：ℓ₁, ℓ₂, ℓ∞ 的計算
6. **矩陣的譜分解**和利用它計算 M^(1/2), M⁻¹ 等

## 常見陷阱

- maximize 要轉成 minimize -f₀(x) 才是標準形式
- 等式約束 hⱼ(x) = 0 必須是**仿射（affine）** 才能是凸優化問題
- inf ≠ min（inf 可能取不到）
- PSD 只對 Hermitian/symmetric 矩陣討論
- ‖·‖_P 需要 P 是 positive definite（不只是 PSD！）
