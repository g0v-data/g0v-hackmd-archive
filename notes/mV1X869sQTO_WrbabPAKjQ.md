# 響應 Open Data Day ，收集 open data sets

> [點此觀看原始內容](https://g0v.hackpad.tw/TudWZiYx0D3)

網站：[http://opendataday.org/](http://opendataday.org/)

3/4的第23次g0v黑客松響應Open Data Day的活動，我們想寫一篇介紹Open Data Day與今年四大主題的文章，希望藉由介紹台灣相關的專案與資源，讓大家更認識活動之外，也有更好的切入點著手貢獻。

今年倡議四大主題：
1.  開放研究資料
2.  公共金流追蹤
3.  開放環境資料
4.  開放人權資料

誠摯邀請大家響應Open Data Day：
1.  收集整理台灣相關的 Open Data 連結，貼到下方整理。
2.  再整理到零時政府資料中心  [http://data.g0v.tw/](http://data.g0v.tw/)

*Open Data Day介紹文草稿：[https://hackpad.com/Open-Data-Day--pBoILBg7pYU](https://hackpad.com/Open-Data-Day--pBoILBg7pYU)

- 國外開放人權資料
1.  Civics在open government partnership 2016 所提出的公民社會打壓情況監督資料庫:[https://monitor.civicus.org/](https://monitor.civicus.org/)
2.  Transparency Toolkit: [https://transparencytoolkit.org/](https://transparencytoolkit.org/) 開發免費軟體去收集和分析開放資料，觀察監查、監控資料的各機關、企業的作為以維護人權。底下有兩個專案與此有關：
ICWATCH: [https://icwatch.wikileaks.org/](https://icwatch.wikileaks.org/)，從linkedin去抓可能實際在從事或協助國家監控的人，並列出名單。
surveillance industry index: [https://sii.transparencytoolkit.org/search?action=index&bibtex\_type\_facet=Company&controller=docs&page=1](https://sii.transparencytoolkit.org/search?action=index&bibtex_type_facet=Company&controller=docs&page=1)，將與監控工業有關的資訊（生產、使用、消費等）釋出
