# 2021-03-25 實地看屋練習

奉 Finn 的推坑，與其卡在準備工作，不如先開始看一些覺得成案機率低的房子，當作練習，所以在[擬好各種找屋待做](https://g0v.hackmd.io/@ddio/corent/https%3A%2F%2Fg0v.hackmd.io%2F7g8m5QJ9S_aaU0q35J429g)後，直接約了幾間房子來看，確認幾個自己的假設，以及重新訓練看屋 SOP 。

這次看屋練習想要達到的目標是：

1. 確認「可以裝修的房子」，應該有哪些必備，以及必定不能有的條件
2. 訓練如何從 591 網站判斷上述條件
3. 增加多元物件經驗，多看些自己不熟的地區（三重、萬華、板橋）與房型（頂加、非公寓）
4. 練習與房東談租屋條件，包含裝修、租期
5. 驗證原本規劃的[看屋步驟](https://g0v.hackmd.io/@ddio/corent/https%3A%2F%2Fg0v.hackmd.io%2F7g8m5QJ9S_aaU0q35J429g)，以及[空間設計模式](https://g0v.hackmd.io/@ddio/corent/https%3A%2F%2Fg0v.hackmd.io%2F7g8m5QJ9S_aaU0q35J429g)是否實用

在這兩週裡， ddio 用以下條件，總共找了 6 間房子，最後實際成功看到其中 4 間：

1. 地區：萬華、三重、板橋、中永和
2. 租金：17,000 以下（因為覺得若承擔裝修成本，租金可能有議價空間）
3. 坪數：15 坪以上
4. 其他：可炊，低度或無裝潢（用照片辨識）

## 591 概況

1. 萬華數量少，有少數一兩間租不出去物件，但剛好這陣子刊登過期，所以只能跳過
2. 新北的物件數，板橋 > 中永和 = 三重
   - 三重雖然數量也多，但大多是裝潢好的，或是小坪數一房一廳
   - 板橋與中永和，大多為房東親自刊登，頂加佔三成左右
   - 廚房普遍都是水泥砌的，貼白磁磚，加上會凸一格瓦斯筒的家，非常醜 XD 但好像可以來想怎麼改比較好
3. 這個價位在新北也能找到屋況不錯的公寓， 17,000 的空屋，通常都牆面與天花板整理得不錯，15,000 的則通常有些小缺點
4. 最近看到的都是公寓，沒有透天與木造建築

   5. 廚房為陽台改裝，太過狹長，無室外曬衣空間
   6. 衛浴牆面太多水管，洗衣機出水口在淋浴處，乾溼分離與清潔難處理
   7. 同棟維護尚可，頂樓完全加蓋
   8. 交通普通

   1. 太隨性的房東也不行，要找在意維護物件品質的房東 orz
      1. 房間、廚房流理臺貼滿不同風格的壁紙
      2. 電箱更新過但徹底不管走線，牆壁上到處都是電線，但因為更新過，房東不願意接受整理

      > 原本以為很挑房客的房東（約看房時就先身家調查），會很在意物件維護，結果其實不一定。也比較可以體會為何搬走後要回復原狀...
      > 
   2. 裝修可能性評估
      1. 強調「裝修前，會先徵得房東同意」，可以增加房東放心程度，不管後續要不要復原
      2. 細節可以第二次再談，不然會出現各種假設，影響面試順暢度
   3. 20 萬裝修預算，無法處理的東西：
      1. 新增牆面，或刪除水泥磚牆
      2. 調整水管，例如洗衣機放客廳，排水直接沖浴室這種，整個無解
   4. 發現無法接受的廚房格局：
      1. 縱身太深又太窄的一字形廚房，因為動線無調整可能，特別是廚房被塞了熱水器時，有種被干擾的厭惡感
         > 小又淺的，可以結合廚房外空間
         > 和客廳空用空間的，可作吧台或中島區隔

## 小結與接續規劃

1. 看一間房子的成本（含搜尋、交通、事後檢討）
   1. 不熟練時，一間含搜尋、交通、看房，到事後筆記，要 2.5hr
   2. 熟練後，如果多間排好連續看，也要 1.5hr
1. 新增設計模式：
   1. 水泥磚造流理臺改造
   2. 無隔間方型客廳廚房改造，加上兩個工作空間、一個聚會空間
   3. L 型客廳併飯廳改造，加上兩個工作空間、一個聚會空間
2. 需要避開的物件：
   1. 狹長邊間廚房，因為動線無法改變
   2. 不在陽台的洗衣機，因為排水、風乾、氣味都很可能有問題
   3. 在客廳的瓦斯熱水器，因為一氧化碳中毒風險更高
   4. 只有單面採光的房子，例如背面無防火巷的公寓，因為碰到排氣與鄰居油煙問題機率高
4. 看屋步驟
   1. 建物週邊環境、聲音，以及樓梯間整潔程度
   2. 確認陽台逃生
   3. 初步確認公寓空間規劃，準備和屋主討論
      1. 可雙人工作
      2. 有各自的臥室，隔音堪用
      3. 廚房空間足夠，流理臺外，至少可擺 120cm * 45cm * 3 = 4.5 * 0.5 步長的閒置牆壁
      4. 可曬衣洗衣
      5. 有客廳等可休息的公共空間
   4. 詳細確認各項規格
      1. 用電，各房間插座數量與位置，電箱[總安培數](https://www.courcasa.com/p/gXq8) > 70A
      2. 確認設計模式可行
      3. 確認防水、通風、西晒
      4. 確認水電繳費、與頂樓分攤方式（如果有頂加）
   5. 確認裝修可能
      1. 提出想法，告知會與屋主協調
      2. 確認大致意願，以及復原的需求
5. 頂加隔熱機制
