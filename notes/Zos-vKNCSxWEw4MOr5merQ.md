# Task Board

零激盪會議大家提到一個 Job board 讓大家可以找到方式參與很重要，因此做了這樣的一個 Task board。

Task board 的使用方式很簡單，大家可以跳到自己有興趣坑裡的 board 上看看自己有沒有什麼能做的，接著在 Board 裡做 Comment，tag 坑主，他們會指引各位的下一步。

如果還是困惑，可以隨時問我 @n0ah
