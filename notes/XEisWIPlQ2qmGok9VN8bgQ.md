# Google Earth Engine

Google Earth Engine (GEE) 是由 Google 建置的線上 GIS 分析及遙測資訊視覺化平台，目前可免費(非營利用途)提供遙測資料、地形資料…等，讓學術、商業以及政府單位等使用。 該平台亦提供了許多 API 跟遙測資料處理工具讓使用者可彈性呼叫 (參考 Developer’s Guides 與 Code Editor)，並建置客製化平台。
