# #53 認識地理資料、找到地理資料、提問地理資料

:::warning
**g0v 零時小學校數位公民課程**
[🏫 首頁 Homepage](https://g0v.hackmd.io/@jothon/B1NIQnmXs)
:::

### ▪️ 課程簡介

為學習者提供接觸「地理資料」的機會，協助學習者使用「[政府資料開放平臺](https://data.gov.tw/)」網站，查找自己感興趣的地理資料主題，像是美食餐廳，並且進一步了解，這些資料是如何在空間上表達出具體的位置，介紹地理座標、地址定位等方法與工具。

引導學習者使用「共筆文件」，用共筆文件描述自己感興趣的地理資料的來源網址與欄位內容，並且在「[g0v slack #gis 社群頻道](https://g0v.hackmd.io/@chewei/gis/)」分享資料特點，也鼓勵提問資料使用過程中所遇到的困難，培養學習者查找主題(地理)資料集、活用社群頻道發問與共學的資料學習模式。

### ▪️ 課程認證問題（給需要營隊認證證書者）
:::info
**#53-1**
講師在影片中，介紹了哪幾個常見的地理資料呈現工具？（擇一回答即可）
:::
:::info
**#53-2**
請問講師第一個介紹的案例是哪個縣市的建築物年齡地圖呢？
:::
:::info
**#53-3**
請問講師第一個介紹的資料平台為？（請打網站名稱，共八個中文字）
:::
:::info
**#53-4**
請問講者最後一個介紹的線上協作地圖工具為？（請打英文，中間請記得空格）
:::
:::info
**#53-5**
講者以哪個資料匯入 google my map 圖層作為示範案例？（五個字，＿？＿足跡）
:::
:::info
**#53-6**
講者分享 uMap 是ＸＸ原始碼的服務，請問ＸＸ兩個字為何？
:::
:::info
**#53-7**
AwesomeTable 是以何者作為它的資料集？（提示：英文） 
:::
:::info
**#53-8**
如果想要透過「OpenStreetMap 資料庫標註」的方式來協作，講者推薦哪個工具呢？
:::
:::info
**#53-9**
講者建議 QGIS 功能還很多，若有興趣，可以到哪個單位的 QGIS 資源網找資料？ （三個字）
:::

:::success
請於零時小學校官方 Line（ https://lin.ee/mffftkf ） 回答答案，回答正確會獲得認證勳章，最終我們將以認證勳章作為營隊參與證書的認證依據，並提供營隊參與證明。若不方便使用官方 Line 做認證，請自行提供認證答案，並寄信至 jothon-organizers@g0v.tw。
:::
